Консервативность и монотонность разностных схем

Рассматривается одномерный закон сохранения

\[ \frac{\partial u}{\partial t} + \frac{\partial f(u)}{\partial x} = 0, \tag{8.1} \]

для которого строится разностная аппроксимация на равномерной пространственно-временной сетке.

Консервативность

Определение 1. Разностная схема для уравнения (8.1) называется консервативной, если она может быть записана в виде \[ u_i^{n+1} = u_i^n - \frac{\Delta t}{\Delta x}\left(F_{i+1/2}^n - F_{i-1/2}^n\right), \tag{8.2} \] где \(F_{i+1/2}^n\) — численный поток, одинаково входящий в уравнения для соседних ячеек.
Замечание. Форма (8.2) обеспечивает дискретный закон баланса: изменение величины в ячейке равно разности потоков через её границы. При суммировании по всем ячейкам внутренние потоки взаимно сокращаются, что гарантирует сохранение интегральной величины с точностью до граничных потоков.
Замечание. Для разрывных решений консервативность схемы является необходимым условием сходимости к физически корректному (энтропийному) решению. Неконсервативные схемы могут давать неверные скорости распространения разрывов.

Монотонность

Определение 2. Разностная схема называется монотонной, если значения решения на новом временном слое являются монотонными функциями значений предыдущего слоя, то есть \[ \frac{\partial u_i^{n+1}}{\partial u_k^n} \ge 0 \quad \text{для всех } i, k. \tag{8.3} \]
Определение 3 (по Фридрихсу). Разностная схема монотонна, если из условия \(u_i^n \ge v_i^n\) для всех \(i\) следует \(u_i^{n+1} \ge v_i^{n+1}\) для всех \(i\).

Дискретный принцип максимума

Определение 4. Разностная схема удовлетворяет дискретному принципу максимума, если \[ \min_k u_k^n \le u_i^{n+1} \le \max_k u_k^n \tag{8.4} \] для всех \(i\). Это эквивалентно тому, что \(u_i^{n+1}\) является выпуклой комбинацией значений предыдущего слоя: \[ u_i^{n+1} = \sum_k a_k u_k^n, \qquad a_k \ge 0, \quad \sum_k a_k = 1. \tag{8.5} \]

Полная вариация и TVD

Определение 5. Полной вариацией сеточной функции \(\{u_i^n\}\) называется величина \[ \mathrm{TV}(u^n) = \sum_i |u_{i+1}^n - u_i^n|. \]
Определение 6. Разностная схема называется TVD (total variation diminishing), если для всех \(n\) выполняется \(\mathrm{TV}(u^{n+1}) \le \mathrm{TV}(u^n)\).
Замечание. Свойство TVD ограничивает рост суммарной вариации решения, однако не гарантирует строгого отсутствия новых локальных максимумов или минимумов. Новые экстремумы возможны, если они не приводят к увеличению полной вариации.

Связь между определениями

Теорема 1 (Годунов–Фридрихс–Хартерн). Для линейной разностной схемы следующие утверждения эквивалентны:
  1. аналитическая монотонность в смысле (8.3);
  2. неотрицательность коэффициентов и выполнение условий (8.5);
  3. выполнение дискретного принципа максимума.
Кроме того, каждое из этих условий влечёт выполнение TVD-свойства, однако обратное утверждение, в общем случае, неверно.

Пример. Противопоточная схема первого порядка для уравнения переноса является монотонной, удовлетворяет принципу максимума и является TVD. Центральная схема второго порядка не является монотонной и нарушает принцип максимума, что проявляется в появлении осцилляций вблизи разрывов.
