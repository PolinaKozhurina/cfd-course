Лекция 8 · Метод конечных элементов

Некоторые свойства конечно-элементных аппроксимаций

Принципы классической конечно-элементной техники

Исходной для метода конечных элементов (МКЭ) является идея о кусочно-полиномиальной аппроксимации неизвестной функции на области \(\Omega_h\), разбитой на подобласти — ячейки, или конечные элементы \(e_m\). Для отдельного конечного элемента \(e_m\) задача приближения функции \(\theta\) из некоторого функционального пространства \(V\) состоит в нахождении аппроксимирующей функции \(\theta_h\), удовлетворяющей трём требованиям:

  1. \(\theta_h\) на \(e_m\) является полиномом, как правило, невысокой степени;
  2. \(\theta_h\) имеет разложение по базису из функций форм \(N_j(x)\): \[ \theta_h = \sum_j N_j(x)\,T_j, \] причём \(\theta_h\) совпадает со значениями \(T_k\) в узлах \(x_k\) конечного элемента \(e_m\), т.е. \(\theta_h(x_k) = T_k\);
  3. коэффициенты \(T_j\) из разложения определяются из решения системы линейных алгебраических уравнений (СЛАУ), получаемой, например, методом Галёркина, вариационным или иным методом.

Решение \(\theta_h\) приближает функцию \(\theta\) из функционального пространства \(V\). Для конечных элементов, построенных с помощью отображения \(x = x(\xi)\) некоторой канонической фигуры \(\Omega^{em}_\xi\) (треугольник, квадрат, тетраэдр, куб) на элемент \(\Omega^{em}_x\), функции формы \(N_j\) в выражении для \(\theta_h\) выражаются зависимостями от канонических переменных \(\xi\).

Лагранжевы и эрмитовы элементы. В курсах вычислительной математики задача приближения функции по её значениям в наборе точек называется задачей лагранжевой интерполяции. По этой причине большинство практически используемых конечных элементов называются лагранжевыми. Более сложные аппроксимации, использующие как значения самой функции, так и значения её производных, называются эрмитовыми и порождают эрмитовы конечные элементы.

Эрмитовы конечные элементы применяются для решения задач о деформациях упругих балок, пластин и оболочек, описываемых дифференциальными уравнениями четвёртого порядка и выше.

Дальнейшее рассмотрение проводится на примере задачи для уравнения теплопроводности, описывающего поле температуры \(T\) внутри области \(\Omega\):

\[ \frac{\partial(\rho c\,T)}{\partial t} + \operatorname{div}\mathbf q = W, \qquad \mathbf q = -k\,\nabla T\cdot A, \tag{8.1} \]

где \(\rho = \rho(x)\) — плотность, \(\rho(x)\ge\rho_0>0\); \(c>0\) — удельная теплоёмкость; \(W\) — объёмная плотность мощности источников тепла (интенсивность источников); \(\mathbf q\) — поток тепла; \(k\ge k_0>0\) — коэффициент теплопроводности; \(A\) — площадь сечения стержня, вдоль которого происходит теплопередача.

Кусочно-линейная конечно-элементная аппроксимация \(\theta_h(x)\) функции \(\theta(x)\) на области \(\Omega=[a,b]\), разбитой на элементы \(e_1,\dots,e_5\). В узлах \(x_k\) выполняется условие интерполяции \(\theta_h(x_k)=T_k\); внутри каждого элемента \(\theta_h\) — полином малой степени.

Одномерный линейный лагранжев конечный элемент

Пусть одномерный конечный элемент \(e_m = \Omega^{em}_x\) с номером \(m\) занимает область \(x\in e_m = [x_i, x_j]\), \(x_i < x_j\). (Используется произвольная нумерация узлов \(x_i, x_j\), применяемая для более сложных, особенно двумерных и трёхмерных задач.) Данный отрезок может быть отображён на канонический отрезок \(\xi\in[-1,1]\) линейной функцией

\[ x = \frac{h_m}{2}\,\xi + x_{0m}, \tag{8.2} \]

где \(h_m = x_j - x_i\) — длина исходного отрезка; \(x_{0m} = (x_i + x_j)/2\) — его средняя точка. Зависимость (8.2) можно записать в виде параметрического отображения

\[ x = \mathbf N^{em}_x(\xi)\cdot\mathbf X^{em}, \tag{8.3} \]

где \(\mathbf N^{em}_x(\xi) = \big[N_1(\xi),\,N_2(\xi)\big]\) — вектор-строка линейных функций формы

\[ N_1(\xi) = \frac{1-\xi}{2},\qquad N_2(\xi) = \frac{1+\xi}{2}, \tag{8.4} \]

а \(\mathbf X^{em} = \{x_i,\,x_j\}\) — вектор-столбец координат узловых точек. Функции формы можно представить в единой записи

\[ N_s(\xi) = \frac{1 + \xi_s\,\xi}{2},\qquad \xi_1 = -1,\ \xi_2 = 1. \tag{8.5} \]

Отображение (8.2) переводит взаимно однозначно канонический отрезок \(\Omega^{em}_\xi=[-1,1]\) в элемент \(\Omega^{em}_x=[x_i,x_j]\): точка \(\xi=-1\) переходит в левую точку \(x_i\), точка \(\xi=1\) — в правую \(x_j\), а середина \(\xi=0\) — в среднюю точку \(x_{0m}\). Обратное отображение также линейно:

\[ \xi = \frac{2}{h_m}\,(x - x_{0m}). \tag{8.6} \]

Поскольку в слабую формулировку стационарной задачи теплопроводности входят интегралы с производными первого порядка, минимальной степенью полинома для аппроксимирующей функции \(\theta_h(x)=\theta_h(x(\xi))\) должна быть первая. Простейшая полиномиальная аппроксимация на каноническом отрезке — линейное приближение

\[ \theta^{em}_h(\xi) = \mathbf N^{em*}_\theta(\xi)\cdot\mathbf T^{em}, \tag{8.7} \]

где \(\mathbf N^{em*}_\theta = \big[N_1(\xi),\,N_2(\xi)\big]\), \(\mathbf T^{em}=\{T_i,\,T_j\}\). Представление (8.7) определяет линейную функцию от \(\xi\):

\[ \theta^{em}_h(\xi) = a_m + b_m\,\xi,\qquad a_m = \frac{T_i + T_j}{2},\quad b_m = \frac{T_i - T_j}{2}. \tag{8.8} \]

В совокупности с (8.6) представление (8.7) даёт линейную аппроксимацию по \(x\) на элементе \(e_m=[x_i,x_j]\):

\[ \theta_h(x) = \mathbf N^{em*}_\theta(\xi(x))\cdot\mathbf T^{em},\qquad \mathbf N^{em*}_\theta(\xi(x)) = \big[N_{g1}(x),\,N_{g2}(x)\big], \tag{8.9} \]
\[ N_{g1}(x) = \frac{x_j - x}{h_m},\qquad N_{g2}(x) = \frac{x - x_i}{h_m}. \tag{8.10} \]

Функции \(N_{g1}(x), N_{g2}(x)\) — линейные функции от \(x\), принимающие значение «1» в одном узле и «0» в другом. Таким образом, приближение (8.9), (8.10) есть полиномиальная конечно-элементная аппроксимация на \(e_m\) и определяет линейный по \(x\) конечный элемент с двумя узлами. Константы \(T_i, T_j\) суть значения аппроксимирующей функции в узлах:

\[ \theta_h(x_i)=T_i,\quad \theta_h(x_j)=T_j;\qquad \theta^{em}_h(-1)=T_i,\quad \theta^{em}_h(1)=T_j. \]
Линейный лагранжев элемент (Рис. 2.1 источника). Вверху: «шапочные» функции формы в физических \(x\) и канонических \(\xi\) координатах — каждая равна 1 в своём узле и 0 в другом. Внизу: линейная аппроксимация \(\theta_h\) проходит через узловые значения \(T_i, T_j\). Отображение \(x=x(\xi)\) переводит \([-1,1]\) в \([x_i,x_j]\).
Идеология суперпозиции двух представлений. Выражения (8.9)–(8.10) проще, чем эквивалентные им (8.7) совместно с (8.6). Тем не менее для более сложных элементов удобнее идеология суперпозиции: (а) параметрического отображения канонической фигуры на исходный элемент и (б) аппроксимации полевой функции по локальным переменным на канонической фигуре. Эта идеология становится центральной в двумерных и трёхмерных задачах.

Квадратичный элемент: лагранжевы элементы высших порядков

Линейный элемент даёт кусочно-линейную аппроксимацию поля температур. Если решение недостаточно точно, можно перейти к более мелкой сетке. Можно показать (см. ниже), что при использовании элементов с функциями формы порядка \(p\) для одномерной стационарной задачи теплопроводности справедлива оценка

\[ \|\theta - \theta_h\|_V = O(h^p),\qquad h = \max_m \frac{h_m}{\operatorname{diam}\Omega}, \tag{8.11} \]

где \(V\) — подходящее функциональное пространство, например \(V=\{\eta\in W^1_2(\Omega):\ \eta=0,\ x\in\Gamma_\theta\}\). Для кусочно-линейных элементов (\(p=1\)) имеем \(\|\theta-\theta_h\|_V = O(h)\) — линейную сходимость по размеру элемента.

Кроме медленной сходимости, у линейных элементов есть вторая проблема: для многих приложений важны производные решения (тепловые потоки, деформации, напряжения). При линейных элементах производная аппроксимирующей функции уже кусочно-постоянна и разрывна в узлах. Один путь повышения точности — увеличение числа элементов при том же \(p\); но скорость сходимости остаётся \(O(h^p)\). Альтернатива — выбор полиномиальных аппроксимаций более высокого порядка: при \(h\ll1\) имеем \(h^{p_1}>h^{p_2}\) при \(p_1

Для квадратичной лагранжевой интерполяции на \(e_m=[x_i,x_j]\) нужен дополнительный узел внутри элемента. Логично взять серединную точку \(x_l = x_{0m} = (x_i + x_j)/2\); отображение (8.2) ставит ей в соответствие \(\xi=0\). На каноническом отрезке выделяем три узла \(\xi_1=-1,\ \xi_2=0,\ \xi_3=1\). Квадратичные функции формы, равные 1 в одном узле и 0 в двух других, единственны:

\[ N_1(\xi) = \frac{\xi(\xi-1)}{2},\quad N_2(\xi) = 1-\xi^2,\quad N_3(\xi) = \frac{\xi(\xi+1)}{2}. \tag{8.12} \]

Их можно записать в общей форме (\(\xi_1=-1,\ \xi_2=0,\ \xi_3=1\)):

\[ N_s(\xi) = \frac{3\xi_s^2 - 2}{2}\Big[\xi_s^2(1+\xi_s\xi) - (1-\xi^2)\Big],\quad s=1,2,3. \tag{8.13} \]

Квадратичное представление функции на каноническом отрезке:

\[ \theta^{em}_h(\xi) = \mathbf N^{em*}_\theta(\xi)\cdot\mathbf T^{em},\qquad \mathbf N^{em*}_\theta = \big[N_1,N_2,N_3\big],\ \ \mathbf T^{em}=\{T_i,T_l,T_j\}. \tag{8.14} \]

С учётом обратного отображения (8.6) выражение (8.14) даёт квадратичную зависимость \(\theta_x(x)\) от физической переменной. Элементная матрица теплопроводности для квадратичного элемента имеет традиционную для МКЭ форму

\[ K^{em}_{kpq} = \frac{2}{h_m}\int_{-1}^{1} kA\,N'_p(\xi)\,N'_q(\xi)\,d\xi,\qquad p,q=1,2,3. \tag{8.15} \]
Вырожденность матрицы. Как можно проверить, в случае второй краевой задачи для квадратичной аппроксимации элементная матрица теплопроводности вырождена: \(\det K^{em}_k = 0\). Неотрицательная определённость \(K^{em}_k\) следует из отсутствия главных граничных условий при слабой постановке стационарной задачи теплопроводности.
Квадратичный лагранжев элемент с тремя узлами (Рис. 2.2 источника). Три параболические функции формы \(N_1, N_2, N_3\) на каноническом отрезке: каждая равна 1 в своём узле \(\{-1,0,1\}\) и 0 в двух остальных. Центральная \(N_2=1-\xi^2\) симметрична; концевые \(N_{1,3}\) дают небольшое «провисание» внутри элемента.

Кубический элемент и общая формула Лагранжа

Аналогично строится кубический элемент. Примем четыре равноотстоящих узла на каноническом отрезке: \(\xi_1=-1,\ \xi_2=-1/3,\ \xi_3=1/3,\ \xi_4=1\). Функции формы получаются из общего вида интерполяционного многочлена Лагранжа степени \(n\) для функции \(f\) по узлам \(\xi_i\), \(i=1,\dots,n+1\):

\[ L_n(\xi) = \sum_{i=1}^{n+1} f(\xi_i)\!\!\prod_{\substack{j\ne i\\ j=1,\dots,n+1}}\!\!\frac{\xi-\xi_j}{\xi_i-\xi_j}. \tag{8.16} \]

Значение \(L_n\) в узле совпадает со значением функции: \(L_n(\xi_s)=f(\xi_s)\). Если искать базисную функцию из условия \(N_s(\xi_k)=\delta_{ks}\), то

\[ L_{ns}(\xi) = \prod_{\substack{j\ne s\\ j=1,\dots,n+1}}\frac{\xi-\xi_j}{\xi_s-\xi_j}. \tag{8.17} \]

Отождествляя \(N_s(\xi)=L_{ns}(\xi)\), при \(n=3\) с принятыми узлами получаем:

\[ \begin{aligned} N_1(\xi) &= -\tfrac{9}{16}\big(\xi+\tfrac13\big)\big(\xi-\tfrac13\big)(\xi-1),\\ N_2(\xi) &= \tfrac{27}{16}(\xi+1)\big(\xi-\tfrac13\big)(\xi-1),\\ N_3(\xi) &= -\tfrac{27}{16}(\xi+1)\big(\xi+\tfrac13\big)(\xi-1),\\ N_4(\xi) &= \tfrac{9}{16}\big(\xi+\tfrac13\big)\big(\xi-\tfrac13\big)(\xi+1). \end{aligned} \tag{8.18} \]

В единой записи (\(\xi_1=-1,\ \xi_2=-1/3,\ \xi_3=1/3,\ \xi_4=1\)):

\[ N_s(\xi) = \frac{(\xi_s+\xi)\big(9\xi_s^2\xi^2 - 1\big)}{4\xi_s^3\big(9\xi_s^2 - 5\big)},\quad s=1,2,3,4. \tag{8.19} \]

Кубическое представление функции: \(\theta^{em}_h(\xi) = \mathbf N^{em*}_\theta(\xi)\cdot\mathbf T^{em}\), где \(\mathbf N^{em*}_\theta=[N_1,N_2,N_3,N_4]\), \(\mathbf T^{em}=\{T_i,T_k,T_l,T_j\}\). Узлы \(x_k, x_l\) располагаются равномерно внутри \([x_i,x_j]\) на расстоянии \(h_m/3\) от концов: \(x_k=(2x_i+x_j)/3\), \(x_l=(x_i+2x_j)/3\). Формулы (8.18)–(8.19) общие для всех одномерных кубических лагранжевых элементов в форме прямолинейных отрезков с одной полевой функцией, безотносительно к типу задачи.

Кубический лагранжев элемент с четырьмя узлами (Рис. 2.3 источника). Четыре кубических функции формы по узлам \(\{-1,-\tfrac13,\tfrac13,1\}\); каждая равна 1 в своём узле и 0 в трёх остальных. Видны характерные «выбросы» (overshoot) полиномов высокого порядка между узлами.

Применение элементов высокого порядка ведёт к более громоздким вычислениям, особенно для 2D/3D, поэтому необходим компромисс между простотой элемента и точностью. Среди элементов классического лагранжева типа наибольшее распространение получили элементы с линейной, квадратичной и кубической аппроксимацией по каждой пространственной координате.

Эрмитовы элементы

Лагранжевы элементы обеспечивают непрерывность \(\theta_h\) (класс \(C^0\)), но производные \(\theta_h'\) в узлах, вообще говоря, разрывны. Для уравнений четвёртого порядка (изгиб балок, пластин, оболочек) слабая формулировка содержит интегралы со вторыми производными, и требуется непрерывность не только функции, но и её первой производной (класс \(C^1\)). Этого нельзя достичь, интерполируя только значения \(T_k\): нужны дополнительные степени свободы — узловые значения производных.

Эрмитова аппроксимация. На элементе \([x_i,x_j]\) задаются в каждом узле как значение функции \(T_k\), так и значение производной \(T'_k = (d\theta_h/dx)_k\). Соответствующие функции формы — кубические эрмитовы полиномы \(H_{0k}, H_{1k}\), удовлетворяющие \[ H_{0k}(x_l)=\delta_{kl},\quad H_{0k}'(x_l)=0,\qquad H_{1k}(x_l)=0,\quad H_{1k}'(x_l)=\delta_{kl}. \]

На каноническом отрезке \([-1,1]\) с узлами \(\xi=\pm1\) эрмитовы функции формы имеют вид

\[ \begin{aligned} H_{0,1}(\xi) &= \tfrac14(1-\xi)^2(2+\xi), & H_{1,1}(\xi) &= \tfrac{h_m}{8}(1-\xi)^2(1+\xi),\\ H_{0,2}(\xi) &= \tfrac14(1+\xi)^2(2-\xi), & H_{1,2}(\xi) &= -\tfrac{h_m}{8}(1+\xi)^2(1-\xi), \end{aligned} \tag{8.20} \]

а аппроксимация записывается через 4 степени свободы элемента:

\[ \theta^{em}_h(\xi) = H_{0,1}T_i + H_{1,1}T'_i + H_{0,2}T_j + H_{1,2}T'_j. \tag{8.21} \]
Эрмитов элемент с 4 степенями свободы. Сплошные кривые \(H_{0,1}, H_{0,2}\) отвечают узловым значениям \(T_i, T_j\) (равны 1 в своём узле, имеют нулевой наклон в обоих узлах). Штриховые \(H_{1,1}, H_{1,2}\) отвечают узловым производным \(T'_i, T'_j\) (нулевое значение, единичный наклон в своём узле). Такой элемент обеспечивает непрерывность класса \(C^1\), необходимую для уравнений 4-го порядка.
Степени свободы лагранжева и эрмитова элементов. У лагранжева элемента в каждом узле задаётся лишь значение функции \(T\) (точки) — обеспечивается непрерывность \(C^0\). У эрмитова элемента дополнительно задаётся узловая производная \(T'\) (стрелки-наклоны) — это даёт гладкую стыковку \(C^1\) на границах элементов.

Интерактив: лагранжев (C⁰) против эрмитова (C¹) элемента

Ниже одна и та же гладкая функция интерполируется на трёх элементах. В режиме Лагранж используются только узловые значения (кусочно-кубический лагранжев интерполянт без согласования наклонов), в режиме Эрмит — узловые значения и производные (кубические эрмитовы полиномы \(H_{0k}, H_{1k}\)). Сравниваются сама интерполяция и её производная: смотрите на стыки элементов.

производная f′(x) и интерполянта
Что видно в интерактиве. На левом графике обе интерполяции проходят через узловые значения и близки к точной функции. Различие проявляется на правом графике производной: у лагранжева элемента (C⁰) на стыках элементов (штриховые линии) производная разрывна — кривая делает скачок, а сама интерполяция имеет излом. У эрмитова элемента (C¹) узловые наклоны согласованы, поэтому производная непрерывна, а стыковка гладкая (поле статуса показывает максимальный скачок производной ≈ 0). Именно это свойство \(C^1\) необходимо для уравнений 4-го порядка.

Погрешности КЭ-аппроксимаций. Оценки сходимости

Одними из важнейших вопросов при применении приближённых методов являются сходимость \(\theta_h\to\theta\) и оценка её скорости, т.е. оценка нормы погрешности \(\|\theta(x)-\theta_h(x)\|_V\) (или эквивалентной энергетической нормы \(\|\cdot\|_a\)). Если задача допускает вариационную формулировку

\[ A(u)=f,\quad A>\varepsilon E,\ \varepsilon>0,\qquad u = \arg\min\Phi(v),\ \ \Phi(v) = (Av,v) - 2(v,f), \tag{8.22} \]

то применима теорема о том, что приближение Ритца (минимизирующее функционал) даёт наилучшие аппроксимационные свойства КЭ-решения. Если вместо \(\theta_h\) взять кусочно-полиномиальный интерполянт \(\theta_I(x)\) точного решения (т.е. \(\theta_I(x_j)=\theta(x_j)\)), то

\[ \|\theta-\theta_h\|_a \le \|\theta-\theta_I\|_a. \tag{8.23} \]

Поэтому достаточно оценить погрешность интерполяции \(\|\theta-\theta_I\|_a\), а затем воспользоваться (8.23). Оценка опирается на следующую лемму.

Лемма 2.1. Если \(\psi(x)\in C^2[a,b]\) и \(M_2 = \max_{[a,b]}|\psi''(x)|\), то \[ |\psi(x)| \le \max\{|\psi(a)|,|\psi(b)|\} + \tfrac18 M_2 (b-a)^2, \tag{8.24} \] \[ |\psi'(x)| \le \frac{|\psi(b)-\psi(a)|}{b-a} + M_2(b-a). \tag{8.25} \]
Пусть \(L_1(x) = \psi(a)\dfrac{b-x}{b-a} + \psi(b)\dfrac{x-a}{b-a}\) — линейный интерполянт Лагранжа функции \(\psi\) на \([a,b]\). Тогда \(\psi(x) = L_1(x) + \omega(x)\), где остаточный член \[ \omega(x) = \frac{\psi''(\eta)}{2}(x-a)(x-b),\quad \eta\in(a,b). \] Поскольку \(L_1\) линейна, \(|L_1(x)|\le\max\{|\psi(a)|,|\psi(b)|\}\). Так как \(\max_{[a,b]}|(x-a)(x-b)| = (b-a)^2/4\) (в точке \(x=(a+b)/2\)), получаем \(|\omega(x)|\le M_2(b-a)^2/8\). Складывая, приходим к (8.24).
Для \(\omega(x)\) имеем \(\omega(a)=\omega(b)=0\) и \(|\omega''|\le M_2\). По теореме Ролля \(\exists\,\xi_1\in(a,b):\ \omega'(\xi_1)=0\). По формуле конечных приращений \(\omega'(x)=\omega'(\xi_1)+\omega''(\xi_2)(x-\xi_1)\), откуда \(|\omega'(x)|\le M_2(b-a)\). Тогда, дифференцируя \(\psi=L_1+\omega\), получаем (8.25).

Перейдём к оценке нормы погрешности для линейных элементов. Возьмём в качестве \(\theta_I(x)\) линейный интерполянт точного решения и обозначим \(\psi(x)=\theta(x)-\theta_I(x)\). Так как \(\theta_I\) интерполирует \(\theta\) по узлам, \(\psi(x_l)=0\) во всех узлах. Энергетическая норма погрешности:

\[ \|\psi\|_a^2 = \|\theta-\theta_h\|_a^2 = \int_a^b kA\left(\frac{\partial\psi}{\partial x}\right)^2 dx = \sum_m \int_{e_m} kA\,(\psi')^2\,dx. \tag{8.26} \]

Если \(\theta\in C^2[a,b]\) и \(M_2=\max_{[a,b]}|\theta''|\), то, поскольку \(\theta_I''=0\) на каждом элементе, \(\psi''=\theta''-\theta_I''=\theta''\), и \(|\psi''|\le M_2\). На каждом элементе \(e_m\) длины \(h_m\), поскольку \(\psi\) обращается в ноль в узлах, оценка (8.25) даёт

\[ |\psi'| \le M_2\,h_m,\qquad x\in e_m. \]

Подставляя в (8.26), записываем цепочку оценок:

\[ \|\psi\|_a^2 \le \sum_m M_2^2\,h_m^2\!\int_{e_m}\!\! kA\,dx \le M_2^2\,\max_{[a,b]}(kA)\sum_m h_m^3 \le M_2^2\,\max_{[a,b]}(kA)\,(b-a)\,h^2, \]

где \(h=\max_m h_m\). В результате

\[ \|\psi\|_a = \|\theta-\theta_I\|_a \le M_2\,C_a\,h,\qquad C_a^2 = \max_{[a,b]}(kA)\,(b-a). \]

Поскольку нормы \(\|\cdot\|_a\) и \(\|\cdot\|_V\) эквивалентны, \(\|\theta-\theta_I\|_V \le M_2 C_1 h\). С учётом теоремы о приближении Ритца (8.23) это доказывает следующую теорему.

Теорема 2.1 (линейные элементы). Пусть \(\theta\in V\) — решение одномерной стационарной задачи теплопроводности с ограниченной второй производной, \(M_2=\max_{[a,b]}|\theta''|\), и \(\theta_h\) — приближённое решение по МКЭ с линейными элементами. Тогда \[ \|\theta-\theta_h\|_V \le M_2\,C_1\,h, \tag{8.27} \] где \(C_1\) не зависит от \(h\), а \(h=\max_m h_m\).
Теорема 2.2 (элементы порядка \(p\)). Пусть \(\theta\in V\) имеет ограниченную производную порядка \(p+1\), \(M_{p+1}=\max_{[a,b]}|\theta^{(p+1)}|\), и \(\theta_h\) получено по МКЭ с лагранжевыми элементами порядка \(p\), \(h=\max_m h_m\). Тогда \[ \|\theta-\theta_h\|_V \le M_{p+1}\,C_p\,h^p, \tag{8.28} \] где \(C_p\) не зависит от \(h\).

Теорема 2.2 показывает, что при использовании элементов порядка \(p\) имеет место сходимость порядка \(O(h^p)\). Заметим, что приведённая выше оценка по энергетической норме (\(W^1_2\)) даёт для гладкого решения порядок \(p\) для производной и \(p+1\) для самой функции в норме \(L_2\) — именно последний порядок (\(\approx2\) для линейных, \(\approx3\) для квадратичных) измеряется в интерактиве ниже.

Интерактив: КЭ-решение против точного — влияние числа и порядка элементов

Здесь снова решается задача \(-u''=f\) на \([0,1]\) с \(u(0)=u(1)=0\) методом конечных элементов, но теперь акцент — на самом решении: переключатель порядка элементов \(P_1/P_2/P_3\) (лагранжевы линейные, квадратичные, кубические) и ползунок числа элементов. Слева рисуется точное решение и восстановленный внутри элементов КЭ-полином (плюс узлы); справа — поточечная ошибка \(u_h(x)-u(x)\). Видно, как точность растёт с порядком \(p\) и с измельчением сетки.

точное uₑ vs КЭ uₕ
поточечная ошибка uₕ − uₑ
Что видно в интерактиве. При фиксированной сетке переключение \(P_1\to P_2\to P_3\) резко уменьшает поточечную ошибку (правый график) на 1–2 порядка — это p-сходимость. Измельчение сетки ползунком при том же порядке тоже снижает ошибку, но медленнее — это h-сходимость. Линейный элемент даёт ломаную (полином 1-й степени на каждом элементе) с заметной ошибкой внутри элементов; кубический почти сливается с точным решением. Для осциллирующего \(\sin(3\pi x)\) нужно больше элементов, чтобы «поймать» все полуволны. Обратите внимание: в узлах ошибка часто близка к нулю (суперсходимость в узлах), а максимум достигается внутри элементов.

h-сходимость и p-сходимость

Из оценки (8.28) следуют два пути повышения точности:

  • h-сходимость. Фиксируем порядок \(p\) и измельчаем сетку, уменьшая \(h\). Погрешность убывает как \(O(h^p)\).
  • p-сходимость. Фиксируем сетку и повышаем порядок элементов \(p\). При \(h\ll1\) каждое увеличение \(p\) на единицу резко уменьшает погрешность.

Какая из них быстрее? Сравнение логично проводить при одинаковом числе узлов, так как число узлов определяет порядок СЛАУ и трудоёмкость её решения. Как показывают многочисленные расчёты, p-сходимость выше h-сходимости. Например, два кубических элемента дают более точное решение, чем три квадратичных; три квадратичных — точнее шести линейных. При этом во всех случаях в сетке 7 узлов, т.е. размерности СЛАУ одинаковы.

Компромисс. Трудоёмкость формирования элементных объектов растёт с порядком \(p\). Для неоднородных сред и сложных границ требуются элементы малой длины. Поэтому на практике необходим разумный компромисс между порядком элементов и густотой сетки; наиболее популярны элементы 1-й, 2-й и 3-й степеней.
Качественный график сходимости в координатах log–log: норма ошибки в \(L_2\) против размера элемента \(h\). Линейные элементы дают прямую с наклоном \(\approx2\), квадратичные — с наклоном \(\approx3\). Чем круче наклон, тем быстрее сходимость при измельчении сетки.

Интерактив: измерение h-сходимости

Ниже численно решается одномерная краевая задача \(-u'' = f\) на \([0,1]\) с \(u(0)=u(1)=0\) методом конечных элементов — линейными и квадратичными элементами. Источник \(f(x)\) подобран так, чтобы точное решение было известно: \(u(x)=\sin(\pi x)\) даёт \(f(x)=\pi^2\sin(\pi x)\). Для серии сеток измеряется \(L_2\)-норма ошибки и строится log–log график; угловые коэффициенты прямых дают наблюдаемые порядки сходимости.

Решение: точное vs КЭ
Сходимость L₂(h) — log-log
Что видно в интерактиве. На графике решения линейные элементы дают ломаную, проходящую через узловые значения, а квадратичные — гладкую кривую, почти неотличимую от точной уже при немногих элементах. На log–log графике сходимости измеренный наклон для \(p=1\) близок к \(2\), а для \(p=2\) — к \(3\), что согласуется с теорией: \(L_2\)-ошибка ведёт себя как \(O(h^{p+1})\). Для более осциллирующего решения \(\sin(3\pi x)\) асимптотический порядок выходит на теоретический лишь после достаточного измельчения сетки (пред-асимптотическая область).
