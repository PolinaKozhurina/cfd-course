Формула Тейлора с остаточным членом

На прошлом занятии мы выделили три источника погрешности и подробно разобрали ошибку округления. Сегодня — главный инструмент для второго источника, ошибки усечения. Почти каждый численный метод (разностные производные, квадратуры, методы интегрирования ОДУ) строится на замене функции отрезком её ряда Тейлора, а оценка отброшенного «хвоста» и есть оценка ошибки метода.

Теорема (формула Тейлора с остатком в форме Лагранжа). Пусть \(f\) имеет \(n+1\) непрерывную производную на отрезке между \(a\) и \(x\). Тогда \[ f(x) = \sum_{k=0}^{n} \frac{f^{(k)}(a)}{k!}\,(x-a)^k \;+\; R_n(x), \] где остаточный член представим в виде \[ R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}\,(x-a)^{n+1},\qquad \xi \text{ между } a \text{ и } x. \] Многочлен \(P_n(x)=\sum_{k=0}^{n} f^{(k)}(a)(x-a)^k/k!\) называется многочленом Тейлора степени \(n\) в точке \(a\).

Точка \(\xi\) существует, но неизвестна; на практике остаток оценивают сверху, заменяя \(f^{(n+1)}(\xi)\) на его максимум \(M_{n+1}=\max|f^{(n+1)}|\):

\[ |R_n(x)| \;\le\; \frac{M_{n+1}}{(n+1)!}\,|x-a|^{n+1}. \tag{2.1} \]
Определение. При \(a=0\) формула Тейлора называется формулой Маклорена: \[ f(x) = \sum_{k=0}^{n}\frac{f^{(k)}(0)}{k!}\,x^k + R_n(x). \] Если \(R_n(x)\to 0\) при \(n\to\infty\) на некотором множестве, частичные суммы сходятся к \(f\), и говорят, что \(f\) разлагается в ряд на этом множестве.

Классические разложения

Пять разложений в точке \(a=0\) встречаются постоянно; их полезно знать наизусть вместе с областью сходимости.

\[ e^{x} = \sum_{k=0}^{\infty}\frac{x^k}{k!} = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \dots, \qquad x\in\mathbb{R}. \tag{2.2} \]
\[ \sin x = \sum_{k=0}^{\infty}\frac{(-1)^k x^{2k+1}}{(2k+1)!}, \qquad \cos x = \sum_{k=0}^{\infty}\frac{(-1)^k x^{2k}}{(2k)!}, \qquad x\in\mathbb{R}. \tag{2.3} \]

Для \(e^x,\ \sin x,\ \cos x\) ряд сходится на всей прямой: факториал в знаменателе бьёт любую степень \(|x|^{n+1}\), поэтому остаток (2.1) стремится к нулю при каждом фиксированном \(x\). Следующие три ряда сходятся лишь в круге радиуса \(1\):

\[ \ln(1+x) = \sum_{k=1}^{\infty}\frac{(-1)^{k+1}x^k}{k} = x - \frac{x^2}{2} + \frac{x^3}{3} - \dots, \qquad -1 < x \le 1. \tag{2.4} \]
\[ \frac{1}{1-x} = \sum_{k=0}^{\infty} x^k = 1 + x + x^2 + \dots, \qquad |x| < 1. \tag{2.5} \]
\[ (1+x)^{\alpha} = \sum_{k=0}^{\infty}\binom{\alpha}{k}x^k, \qquad \binom{\alpha}{k}=\frac{\alpha(\alpha-1)\cdots(\alpha-k+1)}{k!}, \qquad |x| < 1. \tag{2.6} \]
Радиус сходимости. Геометрический ряд (2.5) расходится при \(|x|\ge 1\) — это видно прямо: при \(|x|\ge1\) члены \(x^k\) не стремятся к нулю. У \(\ln(1+x)\) ближайшая особенность функции — в точке \(x=-1\), и именно расстояние до неё, равное \(1\), задаёт радиус. У \(e^x\) особенностей в конечной части плоскости нет, поэтому радиус бесконечен.

Усечение ряда — это ошибка метода

Бесконечную сумму компьютер вычислить не может: ряд обрывают на конечном числе членов. Отброшенный хвост — это и есть ошибка усечения (truncation error), погрешность метода в чистом виде, присутствующая даже при идеальной арифметике.

Порядок малости при \(x\to 0\). Если оборвать ряд Маклорена на члене степени \(n\), то по оценке (2.1) \[ \bigl|f(x) - P_n(x)\bigr| = |R_n(x)| \le \frac{M_{n+1}}{(n+1)!}\,|x|^{n+1} = O\!\left(x^{\,n+1}\right),\qquad x\to 0. \] То есть ошибка ведёт себя как первый отброшенный член. Каждый дополнительный член повышает порядок точности на единицу.

Например, приближение \(\sin x \approx x\) (обрыв на \(n=1\)) имеет ошибку \(O(x^3)\): при \(x=0.1\) она порядка \(10^{-3}\)/6, а при \(x=0.01\) — уже \(10^{-6}\)/6. Это правило «порядок \(=\) первый брошенный член» лежит в основе оценки точности разностных схем на следующих занятиях.

Рис. 1. Многочлены Тейлора \(P_1,P_3,P_5\) синуса в нуле. Вблизи \(x=0\) все совпадают с \(\sin x\); чем выше степень, тем дальше от нуля приближение «держится» за кривую.

Когда ряд встречается с округлением: \(e^{-x}\)

Сходимость ряда — свойство математическое; на конечной арифметике добавляется второй эффект. Рассмотрим прямое суммирование ряда (2.2) для отрицательного аргумента:

\[ e^{-x} = \sum_{k=0}^{\infty}\frac{(-x)^k}{k!} = 1 - x + \frac{x^2}{2} - \frac{x^3}{6} + \dots \tag{2.7} \]

При \(x=20\) слагаемые сначала растут по модулю до огромных значений (наибольший член \(\approx 20^{20}/20! \approx 4\times10^7\)), затем убывают, а сумма всех их равна крошечному \(e^{-20}\approx 2\times10^{-9}\). Чтобы получить такой малый результат как разность чисел порядка \(10^7\), нужно, чтобы старшие разряды этих чисел взаимно уничтожились — это та самая катастрофическая потеря значимости с занятия 1, но порождённая знакопеременным рядом. Лекарство то же: переписать вычисление, например \(e^{-20}=1/e^{20}\), где все слагаемые положительны. Проверим это руками в практикуме.

Достаточно стандартной библиотеки, NumPy и Matplotlib. Цель — увидеть ошибку усечения как функцию числа членов и столкнуться с знакопеременной неустойчивостью. Справочник по инструментам — в памятке Python/NumPy/SciPy.

Практикум 1. Частичные суммы и ошибка усечения

Считаем частичные суммы рядов для \(\exp\) и \(\sin\) устойчиво, домножая предыдущий член на отношение соседних членов (без вычисления факториалов и степеней по отдельности), и сравниваем с эталоном math.exp/math.sin.

partial_sums.py▶ python
import math
import numpy as np
import matplotlib.pyplot as plt

def exp_partial(x, n):
    """Сумма первых n членов ряда exp: term_k = x^k / k!."""
    s, term = 0.0, 1.0          # член k=0 равен 1
    for k in range(n):
        s += term
        term *= x / (k + 1)     # переход k -> k+1
    return s

def sin_partial(x, n):
    """Сумма первых n членов ряда sin: знакопеременный, нечётные степени."""
    s, term = 0.0, x            # член k=0 равен x
    for k in range(n):
        s += term
        term *= -x * x / ((2*k + 2) * (2*k + 3))
    return s

x = 1.5
ns = range(1, 13)
err_exp = [abs(exp_partial(x, n) - math.exp(x)) for n in ns]
err_sin = [abs(sin_partial(x, n) - math.sin(x)) for n in ns]

plt.semilogy(list(ns), err_exp, 'o-', label='exp')
plt.semilogy(list(ns), err_sin, 's-', label='sin')
plt.xlabel('число членов n'); plt.ylabel('|ошибка усечения|')
plt.legend(); plt.grid(True, which='both'); plt.show()
n= 2 exp: ошибка 1.18e+00 sin: ошибка 5.62e-01 n= 4 exp: ошибка 4.71e-02 sin: ошибка 8.95e-03 n= 6 exp: ошибка 5.95e-04 sin: ошибка 3.84e-05 n= 8 exp: ошибка 3.30e-06 sin: ошибка 7.97e-08 n=10 exp: ошибка 1.10e-08 sin: ошибка 9.92e-11 n=12 exp: ошибка 2.50e-11 sin: ошибка 7.16e-14

На полулогарифмическом графике ошибка падает почти прямой линией — это сверхбыстрая (факториальная) сходимость. Каждый член добавляет несколько верных цифр, пока ошибка усечения не упрётся в уровень округления \(\sim u\), после чего улучшать нечего.

Практикум 2. Знакопеременный ряд и устойчивое \(e^{-20}\)

exp_neg.py▶ python
import math

def exp_series(x, n=200):
    """Прямое суммирование ряда exp(x) первыми n членами."""
    s, term = 0.0, 1.0
    for k in range(n):
        s += term
        term *= x / (k + 1)
    return s

x = 20.0
naive  = exp_series(-x)          # знакопеременный ряд -> потеря значимости
stable = 1.0 / exp_series(x)     # все члены положительны -> устойчиво
true   = math.exp(-x)

print(f"наивно  e^-20 = {naive: .6e}")
print(f"1/e^20  e^-20 = {stable: .6e}")
print(f"math.exp e^-20 = {true: .6e}")
print(f"относит. ошибка наивной = {abs(naive-true)/true: .2e}")
print(f"относит. ошибка 1/e^20  = {abs(stable-true)/true: .2e}")
наивно e^-20 = 5.621884e-09 1/e^20 e^-20 = 2.061154e-09 math.exp e^-20 = 2.061154e-09 относит. ошибка наивной = 1.73e+00 относит. ошибка 1/e^20 = 2.71e-16

Наивное суммирование ошибается на 170 %: ни одной верной цифры. Промежуточные слагаемые достигают \(\sim4\times10^7\), и при их сокращении до \(2\times10^{-9}\) теряются все 16 значащих цифр. Устойчивая формула \(1/e^{20}\) даёт ошибку порядка \(u\) — машинную точность. Мораль занятия 1 в новой обёртке: избегайте вычитания близких больших величин, даже если оно спрятано внутри знакопеременного ряда.

Задача 2.1Сколько членов нужно? Напишите функцию, которая для заданных \(x\) и точности \(\tau\) возвращает наименьшее число членов ряда \(\sin x\), при котором первый отброшенный член по модулю меньше \(\tau\) (для знакопеременного ряда это гарантирует \(|R_n|<\tau\)). Проверьте для \(x=1\), \(\tau=10^{-10}\) и сравните фактическую ошибку с \(\tau\).
Подсказка Для ряда Лейбница (знакопеременного, с убывающими по модулю членами) остаток не превосходит первого отброшенного члена: \(|R_n|\le |t_{n}|\). Наращивайте член \(t_k\) тем же рекуррентным домножением, что в практикуме, и остановитесь, как только \(|t_k|<\tau\). Для \(x=1,\ \tau=10^{-10}\) хватает \(n=7\) членов.
Задача 2.2Радиус сходимости численно. Для \(\ln(1+x)\) и \(1/(1-x)\) постройте \(|f(x)-P_n(x)|\) при \(n=5,10,20,40\) на сетке \(x\in[-0.99,0.99]\) и при \(x=1.1,1.5\). Покажите на графике (semilogy), что при \(|x|<1\) рост \(n\) уменьшает ошибку, а при \(|x|>1\) — увеличивает: ряд расходится.
Подсказка Частичную сумму \(1/(1-x)\) считайте как \(P_n=\sum_{k=0}^{n-1}x^k\) (член \(t_{k+1}=t_k\cdot x\)); для \(\ln(1+x)\) член \(t_{k+1}=t_k\cdot(-x)\cdot k/(k+1)\), старт \(t_1=x\). При \(x=1.5\) добавление членов увеличивает \(|f-P_n|\): расходимость видна как растущие вверх кривые.

Интерактив: частичные суммы Тейлора

Слева — выбранная функция \(f\) (чёрная) и её многочлен Тейлора \(P_N\) (фиолетовый) на отрезке; справа — ошибка усечения \(|f-P_N|\) в полулогарифмическом масштабе. Двигайте слайдер \(N\). Для \(\sin x\) и \(e^x\) рост \(N\) расширяет зону совпадения на всю прямую. Для \(\ln(1+x)\) и \(1/(1-x)\) вне круга \(|x|<1\) (отмечен красными границами) добавление членов уводит \(P_N\) от \(f\) — ряд расходится.

Задача приближения функций

Ряд Тейлора даёт локальное приближение: многочлен \(P_n\) очень точен у точки \(a\), но вдали от неё ошибка \(|x-a|^{n+1}\) растёт, а для \(\ln\) и \(1/(1-x)\) приближение и вовсе разваливается за пределами круга сходимости. На многих задачах нужно иное: один многочлен, хорошо приближающий функцию на всём отрезке сразу. Это общая постановка теории приближений.

Задача наилучшего приближения. Даны: функция \(f\in C[a,b]\), класс приближающих функций (например, многочлены степени \(\le n\), обозначим \(\mathcal{P}_n\)) и норма \(\|\cdot\|\), измеряющая величину ошибки. Требуется найти \(p^\star\in\mathcal{P}_n\), минимизирующий уклонение: \[ \|f-p^\star\| \;=\; \min_{p\in\mathcal{P}_n}\,\|f-p\|. \] Такой \(p^\star\) называют элементом наилучшего приближения. Ответ зависит и от класса, и от выбранной нормы.

Нормы функций на отрезке

«Насколько одна функция близка к другой» можно мерить по-разному. Три нормы на пространстве \(C[a,b]\) встречаются чаще всего; каждая порождает свою задачу наилучшего приближения.

\[ \|g\|_{\infty} = \max_{a\le x\le b}|g(x)| \quad\text{(равномерная, чебышёвская)}, \tag{2.8} \]
\[ \|g\|_{2} = \left(\int_a^b g(x)^2\,dx\right)^{1/2} \quad\text{(среднеквадратичная)},\qquad \|g\|_{1} = \int_a^b |g(x)|\,dx. \tag{2.9} \]

Различие принципиально. Равномерная норма \(\|f-p\|_\infty\) контролирует наибольшую ошибку: «нигде на отрезке не ошибиться больше чем на \(\varepsilon\)». Норма \(\|f-p\|_2\) штрафует средний квадрат уклонения и прощает редкие узкие выбросы. Минимизация по \(\|\cdot\|_\infty\) ведёт к задаче минимакса, минимизация по \(\|\cdot\|_2\) — к методу наименьших квадратов.

Рис. 2. Две меры уклонения. Равномерная норма следит за единственным наибольшим зазором; среднеквадратичная — за интегралом квадрата разности по всему отрезку.

Теорема Вейерштрасса

Прежде чем искать наилучший многочлен, естественно спросить: а можно ли вообще приблизить любую непрерывную функцию многочленом сколь угодно точно? Ответ — да, и это фундамент всей теории.

Теорема Вейерштрасса (об аппроксимации, без доказательства). Для любой непрерывной функции \(f\in C[a,b]\) и любого \(\varepsilon>0\) найдётся многочлен \(p\) такой, что \[ \|f-p\|_{\infty} = \max_{a\le x\le b}|f(x)-p(x)| < \varepsilon. \] Иными словами, множество многочленов плотно в \(C[a,b]\) по равномерной норме: непрерывную функцию можно приблизить многочленом равномерно с любой наперёд заданной точностью.
Что теорема не обещает. Она гарантирует существование хорошего многочлена, но не говорит, какой степени он будет и как его построить. Более того, наивный путь — повышать степень интерполяционного многочлена на равномерной сетке — может приводить к расходимости (феномен Рунге, занятие 3). Поэтому нужны конструктивные методы: чебышёвские узлы, минимакс, наименьшие квадраты.

Минимакс и наименьшие квадраты: анонс

Выбор нормы определяет, какой именно «наилучший» многочлен мы получим, и каким методом он строится. Этим посвящены ближайшие занятия.

  • Минимакс (равномерное приближение, \(\|\cdot\|_\infty\)). Минимизируем наибольшую ошибку. Решение обладает свойством чебышёвского альтернанса: ошибка \(f-p^\star\) достигает максимума по модулю с чередованием знака не менее чем в \(n+2\) точках. Сюда же примыкают полиномы Чебышёва и интерполяция в их узлах (занятия 3–4).
  • Метод наименьших квадратов (\(\|\cdot\|_2\)). Минимизируем интеграл (или сумму) квадратов уклонения. Задача линейная: решение даётся системой нормальных уравнений или ортогональными многочленами (Лежандра, Чебышёва). Этому посвящено занятие 5.
Тейлор против глобального приближения. Остаток Тейлора \(R_n\) минимизирует ошибку в одной точке \(a\) и её окрестности — все производные \(f\) и \(p\) там совпадают до порядка \(n\). Минимакс и МНК, наоборот, «размазывают» ошибку по всему отрезку, жертвуя точностью в центре ради контроля на краях. Поэтому для приближения на отрезке многочлен Тейлора почти всегда хуже наилучшего: например, \(\|e^x-P_n\|_\infty\) на \([-1,1]\) в разы больше минимаксного уклонения той же степени.

Итоги занятия

Что нужно унести с занятия 2:
  1. Формула Тейлора \(f(x)=\sum_{k=0}^n f^{(k)}(a)(x-a)^k/k! + R_n\) с остатком Лагранжа \(R_n=f^{(n+1)}(\xi)(x-a)^{n+1}/(n+1)!\) — основной инструмент локального приближения.
  2. Знать наизусть разложения \(e^x,\sin x,\cos x\) (сходятся всюду) и \(\ln(1+x),\,1/(1-x),\,(1+x)^\alpha\) (радиус сходимости \(1\)).
  3. Ошибка усечения = отброшенный хвост ряда. Обрыв на степени \(n\) даёт \(O(x^{n+1})\) при \(x\to0\): порядок точности равен первому брошенному члену.
  4. Знакопеременный ряд (как для \(e^{-x}\) при больших \(x\)) на конечной арифметике страдает катастрофической потерей значимости — связь с занятием 1. Лекарство: переписать (\(e^{-x}=1/e^{x}\)).
  5. Задача наилучшего приближения: класс (многочлены \(\le n\)) + норма (\(\|\cdot\|_\infty,\|\cdot\|_2,\|\cdot\|_1\)). По Вейерштрассу многочлены плотны в \(C[a,b]\). Минимакс (\(\|\cdot\|_\infty\)) и МНК (\(\|\cdot\|_2\)) дают глобальное приближение — в отличие от локального Тейлора.
Классная работа. Доделать задачи 2.1–2.3. Дополнительно: численно сравнить на \([-1,1]\) ошибку \(\|e^x-P_5\|_\infty\) многочлена Тейлора степени 5 с уклонением «среднего» многочлена, полученного смещением свободного члена \(P_5\) на половину размаха ошибки. Убедиться, что простое смещение уже уменьшает равномерную ошибку — первый шаг к минимаксу.
