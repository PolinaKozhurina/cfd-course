Точный решатель задачи Римана для одномерных законов сохранения

Назначение точного решателя задачи Римана

В предыдущем параграфе была описана структура точного решения задачи Римана для одномерных гиперболических систем и, в частности, для уравнений Эйлера. Однако для использования этой теории в численных методах необходимо не только качественное описание волн, но и алгоритм, позволяющий по заданным левому и правому состояниям \((\rho_L, v_L, p_L)\), \((\rho_R, v_R, p_R)\) вычислить параметры промежуточного состояния и определить значения решения в заданной точке диаграммы \(x\)–\(t\). Такой алгоритм называют точным решателем задачи Римана.

Точный решатель используется, в частности, в методе Годунова для вычисления численных потоков через границы ячеек разностной сетки.

Неизвестные промежуточного состояния

Структура решения задачи Римана для уравнений Эйлера предполагает существование промежуточного состояния, разделённого контактным разрывом, в котором давление и скорость постоянны. Следовательно, основными неизвестными задачи являются \(p_*\), \(v_*\). После их определения плотности \(\rho_{*L}\) и \(\rho_{*R}\) вычисляются отдельно по формулам, соответствующим типу левой и правой волн.

Таким образом, центральной задачей точного решателя является определение величин \(p_*\) и \(v_*\).

Функции левой и правой волны

Для построения алгоритма удобно ввести функции, описывающие связь между давлением и скоростью по каждую сторону контактного разрыва. Для заданного давления \(p\) определим функции \(f_L(p)\) и \(f_R(p)\) так, что

\[ v_* = v_L - f_L(p_*), \qquad v_* = v_R + f_R(p_*). \]

Функциональный вид этих зависимостей определяется типом соответствующей волны.

Левая волна

Волна разрежения (\(p \le p_L\)): левая волна является волной разрежения, и функция \(f_L(p)\) имеет вид

\[ f_L(p) = \frac{2a_L}{\gamma - 1}\left[\left(\frac{p}{p_L}\right)^{(\gamma-1)/(2\gamma)} - 1\right]. \tag{17.1} \]

Ударная волна (\(p > p_L\)): функция \(f_L(p)\) записывается как

\[ f_L(p) = (p - p_L)\sqrt{\frac{2}{(\gamma+1)\rho_L}\cdot\frac{1}{p + \frac{\gamma-1}{\gamma+1}p_L}}. \tag{17.2} \]
Физический смысл. Функция \(f_L(p)\) описывает, насколько уменьшается скорость при прохождении левой волны. Для разрежения (17.1) зависимость гладкая (степенная), для удара (17.2) — содержит корень из давлений. В обоих случаях \(f_L\) монотонно возрастает по \(p\).

Правая волна

Волна разрежения (\(p \le p_R\)):

\[ f_R(p) = \frac{2a_R}{\gamma - 1}\left[\left(\frac{p}{p_R}\right)^{(\gamma-1)/(2\gamma)} - 1\right]. \tag{17.3} \]

Ударная волна (\(p > p_R\)):

\[ f_R(p) = (p - p_R)\sqrt{\frac{2}{(\gamma+1)\rho_R}\cdot\frac{1}{p + \frac{\gamma-1}{\gamma+1}p_R}}. \tag{17.4} \]

Уравнение на давление в промежуточном состоянии

Из условий равенства скорости по обе стороны контактного разрыва следует нелинейное уравнение относительно \(p_*\):

\[ f_L(p_*) + f_R(p_*) + v_R - v_L = 0. \tag{17.5} \]

Это уравнение является скалярным, монотонным по \(p_*\), и потому может быть эффективно решено итерационными методами.

Практика. Для решения уравнения (17.5) используется метод Ньютона или его модификации, причём в качестве начального приближения берётся либо акустическая аппроксимация (линеаризация вблизи среднего состояния), либо простое среднее \(p_0 = (p_L + p_R)/2\). Сходимость метода Ньютона здесь быстрая благодаря монотонности и выпуклости \(f_L + f_R\).

Вычисление скорости

После нахождения \(p_*\) скорость \(v_*\) вычисляется по формуле

\[ v_* = \frac{1}{2}\bigl(v_L + v_R + f_R(p_*) - f_L(p_*)\bigr). \tag{17.6} \]

Вычисление плотностей в промежуточных состояниях

После определения \(p_*\) и \(v_*\) плотности \(\rho_{*L}\) и \(\rho_{*R}\) находятся в зависимости от типа волн.

Для волны разрежения используется адиабатическое соотношение:

\[ \rho_* = \rho\left(\frac{p_*}{p}\right)^{1/\gamma}. \tag{17.7} \]

Для ударной волны плотность определяется из соотношений Ранкина–Гюгонио:

\[ \rho_* = \rho\,\frac{p_*/p + (\gamma-1)/(\gamma+1)}{(\gamma-1)/(\gamma+1)\cdot p_*/p + 1}. \tag{17.8} \]
Замечание. Формулы (17.7) и (17.8) применяются отдельно для левой стороны (\(\rho = \rho_L, p = p_L\)) и правой стороны (\(\rho = \rho_R, p = p_R\)). В результате получаются два различных значения плотности \(\rho_{*L} \ne \rho_{*R}\), разделённых контактным разрывом.

Процедура выборки решения

Для практического применения решателя необходимо определить значение решения в заданной точке диаграммы \(x\)–\(t\), обычно в точке \(x = 0\) при \(t > 0\). Это эквивалентно определению знака автомодельной переменной \(\xi = x/t\).

Процедура выборки состоит в сравнении \(\xi\) со скоростями волн: \(S_L\), \(S_C = v_*\), \(S_R\), а в случае разрежений — с границами веера \(S_H\) и \(S_T\). В зависимости от положения точки \(\xi\) она попадает либо в область исходного состояния, либо в область промежуточного состояния, либо внутрь веера разрежения, где состояние определяется по автомодельным формулам.

Алгоритм выборки (для \(\xi = 0\)):
  1. Если \(\xi < S_L\): состояние = \(\mathbf{u}_L\)
  2. Если \(S_L \le \xi < v_*\): состояние = \(\mathbf{u}_{*L}\) (или внутри веера левого разрежения)
  3. Если \(\xi = v_*\): контакт — скорость и давление одинаковы, плотность скачкообразна
  4. Если \(v_* < \xi \le S_R\): состояние = \(\mathbf{u}_{*R}\) (или внутри веера правого разрежения)
  5. Если \(\xi > S_R\): состояние = \(\mathbf{u}_R\)

Использование точного решателя в методе Годунова

В методе Годунова точный решатель задачи Римана применяется для вычисления численного потока через границу ячейки:

\[ \mathbf{F}_{i+1/2} = \mathbf{f}\bigl(\mathbf{u}(0, t_n + 0)\bigr). \]

Таким образом, решатель задачи Римана является центральным элементом схемы, обеспечивающим корректный учёт волновой структуры решения и физический баланс консервативных величин.

Замечание. Несмотря на высокую вычислительную стоимость (итерационное решение уравнения (17.5) на каждой грани), точный решатель задачи Римана служит эталоном, на основе которого строятся приближённые решатели, рассматриваемые в §18–§22. Именно сравнение с точным решением задачи Римана позволяет оценить качество приближённых методов.
