Сеточные функции и расчётная сетка

Для начала введём понятие сеточных функций. Рассмотрим линейное гиперболическое дифференциальное уравнение в частных производных вида

\[ \frac{\partial u}{\partial t} + a\,\frac{\partial u}{\partial x} = 0, \qquad -\infty < x < \infty, \tag{1.1} \]

где \(x\) — пространственная координата, \(t\) — время, \(a = \mathrm{const}\). Дальше по курсу будем называть (1.1) уравнением переноса.

Предположим, что нам надо решить уравнение (1.1). Для обеспечения единственности его решения мы должны задать функцию \(u(x,t)\) в некоторый момент \(t = t_0\). Обычно выбирается значение \(t_0 = 0\). Итак, предположим, что при \(t=0\) задана функция

\[ u(x, 0) = u_0(x), \qquad -\infty < x < \infty. \tag{1.2} \]

Математическая задача (1.1)–(1.2) называется задачей с начальными данными, или задачей Коши.

Конечно-разностная схема

Теперь рассмотрим вопрос о построении конечно-разностной схемы для численного решения задачи Коши. В случае аналитического решения уравнения в частных производных его решение находится в области непрерывного изменения переменных \(x\) и \(t\). В отличие от аналитического случая, приближённое решение уравнения в частных производных по методу конечных разностей определяется только в некоторых дискретных точках плоскости \((x, t)\).

Введём на оси \(x\) бесконечное множество точек

\[ \{0,\;\pm h,\;\pm 2h,\;\ldots\}, \tag{1.3} \]

где \(h\) — некоторая заданная положительная величина. Это множество точек называется расчётной сеткой на оси \(x\), или пространственной сеткой (потому что \(x\) — пространственная переменная). Точки \(x_i = ih\), \(i = 0, \pm 1, \pm 2, \ldots\) из множества (1.3) называются узлами пространственной сетки. Величина \(h\) называется шагом пространственной сетки.

Предположим, что численное решение уравнения (1.1) должно быть определено в конечном временном промежутке \(0 \le t \le T\), где \(T\) — заданная величина, \(0 < T < \infty\). Аналогично множеству (1.3) мы можем определить сетку на оси \(t\):

\[ 0 = t_0 < t_1 < t_2 < \ldots < t_{N-1} < t_N = T. \tag{1.4} \]

Равномерная сетка

Сетки (1.3)–(1.4) называются равномерными, если выполняются следующие условия:

\[ x_i = ih, \quad i = 0, \pm 1, \pm 2, \ldots\,; \qquad t_n = n\tau, \quad n = 0, \ldots, N. \tag{1.5} \]

Величина \(\tau = T/N\) называется временным шагом.

Введём теперь равномерную сетку \(G_h\) в плоскости \((x, t)\) как множество точек пересечения прямых линий. Функция \(u_i^n = u(x_i, t_n)\), которая определена в узлах сетки \(G_h\), называется сеточной функцией.

Равномерная расчётная сетка \(G_h\) с шагом \(h\) по пространству и \(\tau\) по времени.

Типы расчётных сеток

Назовём расчётную сетку структурированной (или регулярной), если она взаимно однозначно отображается на прямоугольную сетку в единичном кубе с сохранением порядка узлов сетки. Структурированные сетки могут быть равномерными (постоянный шаг сетки) и неравномерными. Также структурированные сетки можно разделить на декартовы, то есть прямоугольные, и криволинейные. Все остальные сетки назовём неструктурированными.

Сеточные нормы

Так как далее будут введены различные операторы, то необходимо определиться с понятием нормы в заданном нами дискретном пространстве. Рассмотрим некоторые варианты сеточных норм.

1. Сеточный аналог равномерной нормы

\[ \|u\|_C = \max_{x \in G} |u(x)|, \tag{1.6} \]

где \(G\) — замкнутая ограниченная область, имеет вид

\[ \|u_h\| = \max_{x \in G_h} |u_h(x)|. \tag{1.7} \]

2. Сеточные аналоги нормы \(L_2\)

\[ \|u\|_{L_2} = \left(\int_0^1 u^2(x)\,dx\right)^{1/2} \tag{1.8} \]

в пространстве \(L_2\) определяются с помощью интегральных сумм и имеют вид:

\[ \|u_h\| = \sqrt{\sum_{i=1}^{N} u_h^2(x)\,h\,}, \qquad \|u_h\| = \sqrt{\sum_{i=0}^{N-1} u_h^2(x)\,h\,}. \tag{1.9} \]

Основные определения

Завершим введение и первую тему последними определениями, которые понадобятся в будущем.

Определение 1. Разностная схема — это совокупность разностных уравнений для определения значений сеточной функции внутри расчётной области, дополненная соответствующими начальными и граничными условиями для этой сеточной функции.
Определение 2. Шаблон схемы представляет из себя конфигурацию расчётных узлов в области интегрирования, используемых на каждом элементарном шаге вычислений.
Примеры шаблонов: явный трёхточечный (уголок), неявный трёхточечный и «крест» (три временных слоя).
