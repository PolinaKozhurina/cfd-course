§17 · Семестр 1

Точный решатель задачи Римана для одномерных законов сохранения

Назначение точного решателя

В предыдущем параграфе была описана структура точного решения задачи Римана для одномерных гиперболических систем и, в частности, для уравнений Эйлера. Однако для использования этой теории в численных методах необходимо не только качественное описание волн, но и алгоритм, позволяющий по заданным левому и правому состояниям \((\rho_L, v_L, p_L)\), \((\rho_R, v_R, p_R)\) вычислить параметры промежуточного состояния и определить значения решения в заданной точке диаграммы \(x\)–\(t\). Такой алгоритм называют точным решателем задачи Римана.

Точный решатель используется, в частности, в методе Годунова для вычисления численных потоков через границы ячеек разностной сетки.

Неизвестные промежуточного состояния

Структура решения задачи Римана для уравнений Эйлера предполагает существование промежуточного состояния, разделённого контактным разрывом, в котором давление и скорость постоянны. Следовательно, основными неизвестными задачи являются \(p_*\), \(v_*\). После их определения плотности \(\rho_{*L}\) и \(\rho_{*R}\) вычисляются отдельно по формулам, соответствующим типу левой и правой волн.

Функции левой и правой волны

Для построения алгоритма удобно ввести функции, описывающие связь между давлением и скоростью по каждую сторону контактного разрыва. Для заданного давления \(p\) определим функции \(f_L(p)\) и \(f_R(p)\) так, что

\[ v_* = v_L - f_L(p_*), \qquad v_* = v_R + f_R(p_*). \]

Функциональный вид этих зависимостей определяется типом соответствующей волны. Если \(p \le p_L\), то левая волна является волной разрежения, и функция \(f_L(p)\) имеет вид

\[ f_L(p) = \frac{2a_L}{\gamma - 1}\left[\left(\frac{p}{p_L}\right)^{(\gamma-1)/(2\gamma)} - 1\right]. \]

Если же \(p > p_L\), то левая волна является ударной, и

\[ f_L(p) = (p - p_L)\sqrt{\frac{A_L}{p + B_L}}, \qquad A_L = \frac{2}{(\gamma+1)\rho_L}, \quad B_L = \frac{\gamma-1}{\gamma+1}p_L. \]

Аналогичные формулы справедливы для правой волны \(f_R(p)\).

Итерационная процедура определения давления

Приравнивая два выражения для \(v_*\), получаем нелинейное уравнение относительно \(p_*\):

\[ f_L(p_*) + f_R(p_*) + v_R - v_L = 0. \]

Это уравнение решается итерационным методом (например, методом Ньютона). После определения \(p_*\) вычисляется \(v_*\):

\[ v_* = \frac{1}{2}(v_L + v_R) + \frac{1}{2}\bigl(f_R(p_*) - f_L(p_*)\bigr). \]

Процедура выборки решения

Для практического применения решателя необходимо определить значение решения в заданной точке диаграммы \(x\)–\(t\), обычно в точке \(x = 0\) при \(t > 0\). Это эквивалентно определению знака автомодельной переменной \(\xi = x/t\).

Процедура выборки состоит в сравнении \(\xi\) со скоростями волн: \(S_L\), \(S_C = v_*\), \(S_R\), а в случае разрежений — с границами веера \(S_H\) и \(S_T\). В зависимости от положения точки \(\xi\) она попадает либо в область исходного состояния, либо в область промежуточного состояния, либо внутрь веера разрежения, где состояние определяется по автомодельным формулам.

Использование в методе Годунова

В методе Годунова точный решатель задачи Римана применяется для вычисления численного потока через границу ячейки:

\[ \mathbf{F}_{i+1/2} = \mathbf{f}\bigl(\mathbf{u}(0, t_n + 0)\bigr). \]

Таким образом, решатель задачи Римана является центральным элементом схемы, обеспечивающим корректный учёт волновой структуры решения и физический баланс консервативных величин.

Замечание. Несмотря на высокую вычислительную стоимость, точный решатель задачи Римана служит эталоном, на основе которого строятся приближённые решатели, рассматриваемые в последующих параграфах.
