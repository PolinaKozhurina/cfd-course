Одномерные уравнения газовой динамики в переменных Лагранжа

В этом разделе выводится лагранжева форма одномерных уравнений газовой динамики из уравнений Эйлера. Основная идея состоит в переходе от независимых переменных \((x, t)\) к массовой (материальной) координате, что приводит к существенно иной интерпретации сетки и потоков.

Уравнения газовой динамики в форме Эйлера

Рассмотрим одномерные уравнения Эйлера в консервативной форме:

\[ \frac{\partial}{\partial t}\begin{pmatrix}\rho \\ \rho u \\ E\end{pmatrix} + \frac{\partial}{\partial x}\begin{pmatrix}\rho u \\ \rho u^2 + p \\ u(E+p)\end{pmatrix} = 0, \tag{12.1} \]

где \(\rho = \rho(x,t)\) — плотность, \(u = u(x,t)\) — скорость, \(p = p(\rho, e)\) — давление, \(E = \rho e + \tfrac{1}{2}\rho u^2\) — полная энергия на единицу объёма, \(e\) — внутренняя энергия на единицу массы.

Из (12.1) следуют уравнения:

\[ \rho_t + (\rho u)_x = 0, \tag{12.2} \]
\[ (\rho u)_t + (\rho u^2 + p)_x = 0, \tag{12.3} \]
\[ E_t + \bigl(u(E+p)\bigr)_x = 0. \tag{12.4} \]

Материальная производная и массовая координата

Рис. 1. Эйлерова сетка фиксирована в пространстве, вещество течёт сквозь ячейки. В лагранжевом описании сетка сама движется вместе с частицами — контактные разрывы «сидят» на узлах.
Определение 1. Материальная (субстанциональная) производная определяется как \[ \frac{D}{Dt} = \frac{\partial}{\partial t} + u\frac{\partial}{\partial x}. \] Она описывает скорость изменения величины вдоль траектории частицы газа.
Определение 2. Введём массовую (лагранжеву) координату \(m\) как массу газа слева от точки \(x\): \[ m(x,t) = \int_{x_0}^{x} \rho(\xi, t)\,d\xi, \tag{12.5} \] где \(x_0\) — фиксированная точка (например, левая граница области).

Из определения (12.5) немедленно получаем

\[ \frac{\partial m}{\partial x} = \rho, \qquad \frac{\partial x}{\partial m} = \frac{1}{\rho}. \tag{12.6} \]
Определение 3. Удельный объём (специфический объём) обозначим \(\tau = 1/\rho\). Тогда из (12.6) следует \(\partial x / \partial m = \tau\). \[ \tag{12.7} \]

Вывод лагранжевой формы из уравнения неразрывности

Начнём с уравнения неразрывности (12.2): \(\rho_t + (\rho u)_x = 0\). Раскроем производную:

\[ \rho_t + u\rho_x + \rho u_x = 0, \quad\text{то есть}\quad \frac{D\rho}{Dt} + \rho\, u_x = 0. \tag{12.8} \]

Перейдём к удельному объёму \(\tau = 1/\rho\): \(D\tau/Dt = -(1/\rho^2)(D\rho/Dt)\). Подставляя (12.8), получаем

\[ \frac{D\tau}{Dt} = -\frac{1}{\rho^2}(-\rho u_x) = \frac{u_x}{\rho} = \tau u_x. \]

Используя связь производных: \(u_x = \rho\, u_m\), получаем \(D\tau/Dt = u_m\). В лагранжевом описании \(D/Dt = \partial/\partial t|_m\), следовательно:

\[ \tau_t - u_m = 0. \tag{12.10} \]

Вывод уравнения движения

Из уравнения импульса (12.3) в неконсервативной форме:

\[ \frac{Du}{Dt} + \frac{1}{\rho}p_x = 0. \tag{12.11} \]

Переходим к массовой координате: \((1/\rho)p_x = p_m\), \(Du/Dt = u_t\) при фиксированном \(m\). Следовательно:

\[ u_t + p_m = 0. \tag{12.12} \]

Вывод энергетического уравнения

Стандартное преобразование (вычитание \(u\)-кратного уравнения импульса из уравнения энергии) даёт:

\[ \frac{De}{Dt} + \frac{p}{\rho}u_x = 0. \tag{12.13} \]

Используя \(u_x = \rho u_m\) и \(De/Dt = e_t\) (при фиксированном \(m\)):

\[ e_t + p\, u_m = 0, \quad\text{или}\quad e_t + p\,\tau_t = 0. \tag{12.14} \]

Итоговая лагранжева система

Собирая (12.10), (12.12), (12.14), получаем одномерные уравнения газовой динамики в переменных Лагранжа:

\[ \begin{cases} \tau_t - u_m = 0, \\[4pt] u_t + p_m = 0, \\[4pt] e_t + p\, u_m = 0, \end{cases} \qquad p = p(\tau, e). \tag{12.15} \]
Замечание. Уравнения (12.15) часто дополняют кинематическим соотношением \(x_m = \tau\), которое описывает эволюцию положения частиц (узлов лагранжевой сетки).

Преимущества лагранжевого подхода

Определение 4. Лагранжевый подход означает, что сетка «движется вместе с веществом»: фиксированные значения \(m\) соответствуют фиксированным массам вещества.

Основные преимущества лагранжевого описания:

  1. Точное следование разрывам, связанным с веществом. Контактные разрывы и материальные поверхности естественно «сидят» на сетке и не размазываются чисто конвективно.
  2. Удобство для задач с большими деформациями и движущимися границами. Границы, движущиеся вместе с веществом (свободные поверхности), описываются проще.
  3. Простая структура потоков. В уравнении импульса (12.12) появляется производная \(p_m\) без конвективного переноса, что облегчает построение некоторых схем.

Акустическое приближение

Линеаризуя систему (12.15) около постоянного состояния \((\tau_0, u_0=0, e_0)\), получим волновое уравнение. Пусть \(\tau = \tau_0 + \tau'\), \(u = u'\), \(p = p_0 + p'\), причём \(p' = -(\partial p/\partial\tau)_s\,\tau' = c_L^2\,\tau'\) в адиабатическом пределе, где \(c_L^2 = -(\partial p/\partial\tau)_s\) — лагранжева скорость звука (в переменных массы). Дифференцируя первое уравнение по \(t\), второе по \(m\), получаем

\[ \tau'_{tt} - u'_{mt} = 0, \qquad u'_{tm} + p'_{mm} = 0 \ \Rightarrow\ \tau'_{tt} + p'_{mm} = 0. \]

Подставляя \(p' = c_L^2\,\tau'\), приходим к волновому уравнению в лагранжевых координатах:

\[ \tau'_{tt} + c_L^2\,\tau'_{mm} = 0. \tag{12.16} \]

(Внимание на знак: в переменных Лагранжа \(-\partial p/\partial\tau > 0\) для обычных сред, и уравнение имеет правильный волновой вид с характеристиками \(dm/dt = \pm c_L\).) Переход к эйлеровой скорости звука \(c_E\) даёт связь \(c_L = \rho_0\, c_E\).

Интерактив: движение лагранжевых частиц

Ниже — анимированная визуализация движения лагранжевых частиц в задаче Римана. Показаны три представления одного и того же течения:

  • Вверху: мировые линии частиц в плоскости \((x, t)\) — траектории «жизни» каждой частицы газа. Цвет линии отражает текущую локальную плотность.
  • В центре: ряд частиц на физической оси \(x\) в текущий момент. Размер шарика пропорционален удельному объёму \(\tau = 1/\rho\) (большой шарик = разрежённый газ, маленький = сжатый). Пружинные отрезки между соседями показывают растяжение/сжатие ячеек лагранжевой сетки.
  • Внизу: профиль плотности \(\rho(x, t)\) в том же моменте с позициями частиц для контекста.

Что видно в интерактиве:
  • Sod: маркеры слева от контактного разрыва проходят через волну разрежения и растягиваются в плотности (сетка разрежается). Маркеры справа от контактного разрыва сжимаются ударной волной (сетка сжимается, плотность растёт).
  • Поршень: наблюдается волна сжатия, движущаяся вправо. Маркеры собираются к её фронту — в лагранжевой сетке это означает сильное уменьшение объёма ячейки.
  • Волна разрежения: маркеры расходятся — лагранжева сетка сильно разрежается. В реальном коде это может привести к падению качества сетки.
Именно поэтому в длительных расчётах лагранжевой сетки нужен remap (перенос) — регуляризация сетки периодически восстанавливает равномерность. Этот подход реализован в ALE-схемах (Arbitrary Lagrangian–Eulerian).

Ограничения лагранжевого подхода

  1. Сгущение/разрежение сетки. При сильных сжатиях ячейки могут вырождаться, а при разрежениях — чрезмерно растягиваться, что ухудшает точность и устойчивость.
  2. Трудности многомерного обобщения. В многомерных задачах лагранжева сетка может запутываться, возникают проблемы перекоса ячеек и потери качества сетки.
  3. Необходимость перемешивания (remap) в ALE-подходах. Для длительных расчётов часто требуется регуляризация сетки и перенос решения на «улучшенную» сетку.
Упражнение. Получить из системы (12.15) волновое уравнение в акустическом приближении (линеаризация около постоянного состояния) и выразить скорость звука через производные уравнения состояния \(p(\tau, e)\).
