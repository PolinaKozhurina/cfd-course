Спектральный признак устойчивости

В прошлом параграфе было введено понятие устойчивости. Простыми словами, разностная схема устойчива по правой части, если малые возмущения сеточной функции \(u_i^n\) приводят к малому изменению решения. Близость решений оценивается с использованием сеточных норм.

Спектральный признак устойчивости разностных схем является наиболее распространённым в силу своей простоты и универсальности. Его можно применять как к явным, так и к неявным схемам, к системам уравнений, к задачам со многими переменными. Однако стоит обратить внимание, что мы не всегда можем им воспользоваться.

Замечание. Метод гармоник допускает использование только для однородных уравнений с постоянными коэффициентами без учёта краевых условий.

Суть метода заключается в том, что свойства разностных схем оцениваются по поведению во времени решений специального вида (гармонических функций, или гармоник), являющихся собственными функциями оператора перехода.

Гармоники и оператор перехода

Определение 1. Оператор перехода — оператор, обеспечивающий переход решения с предыдущего временного слоя на следующий.

Для функций непрерывных аргументов \(x, t\) гармоникой называют выражение \(u = e^{i\omega t + ikx}\), где \(\omega\) и \(k\) — постоянные, причём \(k\) — вещественное. Для дискретных функций, определённых на сетке \(x_m = mh\), \(t_n = n\tau\), гармоника имеет вид

\[ u_m^n = \exp(\omega\tau n + ikhm) = \lambda^n \exp(i\alpha m), \tag{5.1} \]

где \(\lambda = \exp(\omega\tau)\), \(\alpha = kh\). Величина \(\alpha\) меняется от 0 (постоянная в направлении \(x\) функция) до \(2\pi\) (функция с наибольшей частотой изменения \(x\) по сетке). Заметим, что в некоторых руководствах верхнее значение \(\alpha\) ограничивают значением \(\pi\), что соответствует одной полуволне периодической функции на сеточном интервале.

Множитель перехода

Подстановка \(u_m^n = \lambda^n \exp(i\alpha m)\) в разностную схему даёт соотношение, связывающее между собой \(\lambda\) и \(\alpha\): так называемое характеристическое (дисперсионное) уравнение \(\lambda = \lambda(\alpha)\). Величина \(\lambda\) — в общем случае комплексная — именуется множителем (коэффициентом) перехода. Смысл последнего названия ясен из равенств

\[ u_m^n = \lambda^n \exp(i\alpha m), \qquad u_m^{n+1} = \lambda^{n+1} \exp(i\alpha m), \]

откуда \(u_m^{n+1} = \lambda u_m^n\). Сеточная функция \(u_m^0\) в форме (5.1) имеет вид \(\exp(i\alpha m)\), и её можно рассматривать как начальное условие задачи. Из (5.1) следует, что \(u_m^n = \lambda^n u_m^0\), откуда

\[ \max_m |u_m^n| = |\lambda|^n \max_m |u_m^0|. \tag{5.2} \]

Необходимое условие устойчивости (условие Неймана)

Воспользуемся вторым определением устойчивости

\[ \|u^{(h)}\|_{U_h} \le C\|f^{(h)}\|_{F_h}, \qquad C \ne C(h) > 0 \tag{5.3} \]

и введём нормы следующим образом

\[ \|u^{(h)}\|_{U_h} = \max_{m,n}\|u_m^n\|, \qquad \|f^{(h)}\|_{F_h} = \max_m \|u_m^0\|. \tag{5.4} \]

Тогда, как следует из соотношений (5.2)–(5.4), для устойчивости схемы во всяком случае необходимо, чтобы при всех \(n\) и для всех гармоник (\(0 \le \alpha \le 2\pi\)) имело место условие \(|\lambda|^n \le C\), \(n = 0, 1, \ldots, N\). Замечая, что \(n \sim 1/\tau\), имеем

\[ |\lambda(\alpha)| \le 1 + C\tau. \tag{5.5} \]

Это ограничение есть необходимое спектральное условие устойчивости (условие Неймана), выражающее ограниченность гармоник по модулю при \(\tau \to 0\), \(n \to \infty\). Геометрическое истолкование условия (5.5) сводится к тому, что в комплексной плоскости \(\mathrm{Re}\,\lambda + i\,\mathrm{Im}\,\lambda\) все значения \(\lambda(\alpha)\) при \(0 \le \alpha \le 2\pi\) не должны выходить из круга радиуса \(1 + C\tau\). На практике обычно заменяют неравенство (5.5) условием

\[ |\lambda| \le 1. \]
Условие Неймана: множитель перехода \(\lambda(\alpha)\) для схемы «уголок» при \(r=0.7\) — окружность радиуса \(r\) с центром в \(1-r\), лежащая внутри единичного круга.

Пример: схема «уголок»

Рассмотрим применение спектрального метода для исследования устойчивости. Исходное дифференциальное уравнение \(u_t + au_x = 0\), \(a = \mathrm{const} > 0\).

Явная схема с несимметричной аппроксимацией \(u_x\) разностью назад:

\[ \frac{u_m^{n+1} - u_m^n}{\tau} + a\frac{u_m^n - u_{m-1}^n}{h} = 0, \]
\[ u_m^{n+1} = (1 - r)u_m^n + r\,u_{m-1}^n, \qquad r = \frac{a\tau}{h}. \]

Характеристическое уравнение \(\lambda = 1 - r + r\exp(-i\alpha)\) — уравнение окружности радиуса \(r\) с центром в точке \(1-r\). Условие Неймана выполняется при \(r \le 1\), что совпадает с критерием Куранта.

Упражнение

Упражнение. Найдите характеристическое уравнение и определите область устойчивости для схемы \[ \frac{u_m^{n+1} - u_m^n}{\tau} + a\frac{u_{m+1}^n - u_m^n}{h} = 0. \]
