Организация

Формат работы

  • Группы по 2–3 человека
  • Код общий на группу (одна папка group_NN/)
  • Каждый участник должен уметь объяснить любой фрагмент
  • Маркеры // CHECK: TAG_NAME в коде — обязательны

Структура папки группы

group_NN/
├── src/ — исходный код C++
├── results/ — графики и числа
└── README.md — кто что делал

Как сдавать

  • Зарегистрироваться на сайте (кнопка «Войти»)
  • Ввести номер группы
  • Заполнить контрольные значения из расчётов
  • Система автоматически сравнит с эталоном

Что проверяется

  • Код с маркерами CHECK: (Ctrl+F)
  • Контрольные числа vs эталон (≤5% ошибка)
  • Графики: ρ, P, u, линии тока, Cp
  • Каждый участник объясняет свою часть
