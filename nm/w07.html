Численное интегрирование I. Формулы Ньютона–Котеса

Лекция A · ≤ 30 мин

Идея квадратурной формулы

Определённый интеграл \(\int_a^b f(x)\,dx\) почти никогда не выражается в элементарных функциях: уже \(\int e^{-x^2}dx\) или \(\int \frac{\sin x}{x}dx\) не берутся аналитически. Зато интеграл всегда можно вычислить приближённо — как взвешенную сумму значений подынтегральной функции в нескольких точках. Такая формула называется квадратурной:

\[ \int_a^b f(x)\,dx \;\approx\; \sum_{i=0}^{n} w_i\, f(x_i), \tag{7.1} \]

где \(x_i\) — узлы, а \(w_i\) — веса. Вся теория сводится к тому, как выбрать узлы и веса, чтобы при минимальном числе вычислений \(f\) получить максимальную точность.

Базовая идея. Заменим подынтегральную функцию \(f\) её интерполяционным полиномом \(P_n\) по узлам \(x_0,\dots,x_n\) и проинтегрируем точно полином: \[ \int_a^b f\,dx \approx \int_a^b P_n(x)\,dx = \sum_{i=0}^{n} \Big(\underbrace{\int_a^b \ell_i(x)\,dx}_{w_i}\Big) f(x_i), \] где \(\ell_i\) — базисные полиномы Лагранжа. Веса \(w_i\) суть интегралы от \(\ell_i\) и не зависят от \(f\) — их считают один раз.

Если узлы выбраны равноотстоящими, \(x_i = a + i h\), \(h = (b-a)/n\), получаются классические формулы Ньютона–Котеса. Качество квадратуры измеряют её алгебраической точностью.

Алгебраическая точность. Квадратурная формула имеет алгебраическую точность \(m\), если она точна для всех многочленов степени \(\le m\) и неточна хотя бы для одного многочлена степени \(m+1\). Чем выше точность, тем быстрее формула сходится для гладких функций.

Простейшие формулы Ньютона–Котеса

Рассмотрим три базовые формулы на отрезке \([a,b]\), отвечающие полиномам степени \(0\) (точнее — \(1\) по симметрии), \(1\) и \(2\).

Прямоугольники и трапеции

Формула средних прямоугольников (midpoint). Заменяем \(f\) её значением в середине отрезка: \[ \int_a^b f\,dx \approx (b-a)\, f\!\Big(\frac{a+b}{2}\Big),\qquad R_{\text{mid}} = \frac{(b-a)^3}{24}\, f''(\xi). \] Хотя формула использует один узел (полином степени 0), за счёт симметрии она интегрирует линейные функции точно — алгебраическая точность \(1\).
Формула трапеций. Заменяем \(f\) хордой через концевые точки: \[ \int_a^b f\,dx \approx \frac{b-a}{2}\big(f(a)+f(b)\big),\qquad R_{\text{trap}} = -\frac{(b-a)^3}{12}\, f''(\xi). \] Алгебраическая точность \(1\): формула точна для линейных \(f\), а ошибка пропорциональна \(f''\) — кривизне.

Интересно, что прямоугольники вдвое точнее трапеций (коэффициент \(1/24\) против \(1/12\)) и противоположны по знаку: трапеция переоценивает интеграл выпуклой функции, средняя точка — недооценивает. Это наблюдение — зародыш идеи Симпсона.

Рис. 1. Три базовые квадратуры. Прямоугольник заменяет функцию константой по средней точке, трапеция — хордой, Симпсон — параболой по трём узлам. Парабола «ловит» кубические члены бесплатно.

Формула Симпсона

Возьмём три равноотстоящих узла: концы \(a,b\) и середину \(m=(a+b)/2\). Через них проведём параболу и проинтегрируем её. Получается формула Симпсона:

Формула Симпсона (парабола по 3 узлам). \[ \int_a^b f\,dx \approx \frac{b-a}{6}\Big(f(a)+4f\big(\tfrac{a+b}{2}\big)+f(b)\Big),\qquad R_{\text{Simp}} = -\frac{(b-a)^5}{2880}\, f^{(4)}(\xi). \] Алгебраическая точность 3 (!): несмотря на то, что мы интерполировали лишь параболой (степень 2), формула точна и для кубических функций.
Почему точность не \(2\), а \(3\)? Ошибка интерполяции для кубического члена \(f(x)=x^3\) нечётна относительно середины \(m\), а формула Симпсона симметрична относительно \(m\). При интегрировании по симметричному отрезку нечётная добавка сокращается, и кубический член интегрируется точно. Первый «непокрытый» член — четвёртой степени, поэтому ошибка пропорциональна \(f^{(4)}\). Этот «бесплатный» лишний порядок — главная причина популярности Симпсона.

Составные формулы

Одна парабола на всём \([a,b]\) хороша лишь для коротких отрезков: множитель \((b-a)^5\) в остатке губителен при большом \(b-a\). Стандартное лекарство — разбить отрезок на \(n\) равных частей шага \(h=(b-a)/n\), применить простую формулу на каждом подотрезке и сложить. Получаются составные (кусочные) квадратуры.

Составная формула трапеций. Узлы \(x_i=a+ih\), значения \(f_i=f(x_i)\): \[ \int_a^b f\,dx \approx h\Big(\tfrac12 f_0 + f_1 + f_2 + \dots + f_{n-1} + \tfrac12 f_n\Big),\qquad R = -\frac{(b-a)\,h^2}{12}\, f''(\xi) = O(h^2). \] Внутренние узлы входят с весом \(h\), концевые — с весом \(h/2\).
Составная формула Симпсона (требует чётного \(n\)): \[ \int_a^b f\,dx \approx \frac{h}{3}\Big(f_0 + 4f_1 + 2f_2 + 4f_3 + \dots + 2f_{n-2} + 4f_{n-1} + f_n\Big),\qquad R = -\frac{(b-a)\,h^4}{180}\, f^{(4)}(\xi) = O(h^4). \] Чередующиеся веса \(4,2,4,2,\dots\) на внутренних узлах — следствие склейки парабол по парам подотрезков.

Ключевое различие: при измельчении сетки ошибка трапеций убывает как \(h^2\), а Симпсона — как \(h^4\). Каждое удвоение \(n\) уменьшает ошибку трапеций в \(4\) раза, а Симпсона — в \(16\). Это мы проверим численно прямо сейчас.

Практикум 1. Составные трапеция и Симпсон

quadrature.py▶ python
import numpy as np

def trapezoid(f, a, b, n):
    """Составная формула трапеций, n подотрезков."""
    x = np.linspace(a, b, n + 1)
    y = f(x)
    h = (b - a) / n
    return h * (0.5 * y[0] + y[1:-1].sum() + 0.5 * y[-1])

def simpson(f, a, b, n):
    """Составная формула Симпсона, n чётное."""
    if n % 2:
        raise ValueError("n должно быть чётным")
    x = np.linspace(a, b, n + 1)
    y = f(x)
    h = (b - a) / n
    return h / 3 * (y[0] + y[-1]
                    + 4 * y[1:-1:2].sum()
                    + 2 * y[2:-2:2].sum())

# Тест: интеграл sin на [0, pi] равен ровно 2
f = np.sin
I_exact = 2.0
for n in (4, 8, 16, 32):
    print(f"n={n:3d}  trap={trapezoid(f,0,np.pi,n):.10f}"
          f"  simp={simpson(f,0,np.pi,n):.10f}")
print("точное :", I_exact)

Практикум 2. Порядок сходимости и таблица Ромберга

order_check.py▶ python
import numpy as np

f = np.sin; a, b, I = 0.0, np.pi, 2.0
ns = 2 ** np.arange(2, 12)          # 4, 8, ..., 2048

err_t = np.array([abs(trapezoid(f,a,b,n) - I) for n in ns])
err_s = np.array([abs(simpson(f,a,b,n)   - I) for n in ns])

# наклон log(err) по log(n): должен быть -2 и -4
p_t = np.polyfit(np.log(ns), np.log(err_t), 1)[0]
p_s = np.polyfit(np.log(ns), np.log(err_s), 1)[0]
print(f"наклон трапеции : {p_t:.3f}  (ожидаем -2)")
print(f"наклон Симпсона : {p_s:.3f}  (ожидаем -4)")
наклон трапеции : -2.000 (ожидаем -2) наклон Симпсона : -4.000 (ожидаем -4)

Наклоны подтверждают теорию: \(O(h^2)\) и \(O(h^4)\). Теперь — экстраполяция Ромберга. Она применяет экстраполяцию Ричардсона к составной трапеции и почти даром получает высокий порядок.

romberg.py▶ python
import numpy as np

def romberg(f, a, b, levels=6):
    """Таблица Ромберга T[k][j], T[k][0] = трапеция с 2**k интервалами."""
    T = np.zeros((levels, levels))
    for k in range(levels):
        T[k, 0] = trapezoid(f, a, b, 2 ** k)
        for j in range(1, k + 1):
            # T[k,j] = (4**j * T[k,j-1] - T[k-1,j-1]) / (4**j - 1)
            T[k, j] = (4**j * T[k, j-1] - T[k-1, j-1]) / (4**j - 1)
    return T

T = romberg(np.sin, 0, np.pi, levels=6)
for k in range(6):
    print("  ".join(f"{T[k,j]:.10f}" for j in range(k + 1)))
print("ошибка лучшего :", abs(T[5, 5] - 2.0))

import numpy as np
from scipy import integrate

f = np.sin
x = np.linspace(0, np.pi, 17)        # 17 узлов = 16 интервалов
y = f(x)

# по таблице значений на равномерной сетке:
print("trapezoid :", integrate.trapezoid(y, x))
print("simpson   :", integrate.simpson(y, x))

# адаптивная квадратура по функции (на основе QUADPACK):
val, err = integrate.quad(f, 0, np.pi)

integrate.trapezoid и integrate.simpson работают по готовой таблице значений (когда функция доступна лишь в узлах). integrate.quad — адаптивный алгоритм, сам выбирающий узлы и возвращающий оценку ошибки; это рабочая лошадка для гладких функций.

Задача 7.1Порядок численно. Для \(\int_0^1 e^x dx = e-1\) постройте график ошибки составных трапеции и Симпсона в зависимости от \(n\) в двойных логарифмических осях. Подгоните наклон прямой np.polyfit и убедитесь, что он близок к \(-2\) и \(-4\). Что происходит с наклоном Симпсона при \(n \gtrsim 10^4\) и почему (подсказка: ошибки округления)?
Подсказка При очень больших \(n\) ошибка метода становится меньше уровня округления \(\sim u\,|I|\), и график «упирается в пол»: дальнейшее измельчение сетки уже не помогает, а накопление ошибок суммирования даже слегка ухудшает результат. Поэтому увеличивать \(n\) до бесконечности бессмысленно — есть оптимальный шаг.

Сравните \(T_{2,2}\) с \(\pi/4\) и с прямым Симпсоном при \(n=4\) — они должны совпасть.
Задача 7.3Сколько узлов? Для \(\int_0^\pi \sin x\,dx = 2\) определите минимальное число интервалов \(n\), при котором составная трапеция даёт ответ с ошибкой \(< 10^{-6}\), и то же для Симпсона. Сравните с теоретической оценкой остатка \(|R_{\text{trap}}|\le \frac{(b-a)h^2}{12}\max|f''|\) и аналогичной для Симпсона.
Подсказка Здесь \(\max|f''|=\max|f^{(4)}|=1\), \(b-a=\pi\). Из \(\frac{\pi h^2}{12}<10^{-6}\) получаем \(h\) и \(n=\pi/h\) для трапеции; из \(\frac{\pi h^4}{180}<10^{-6}\) — для Симпсона. Симпсон требует на порядки меньше узлов.

Порядок точности и что он даёт

Практикум численно подтвердил теоретические оценки остатков. Сведём их в одну картину. Говорят, что составная квадратура имеет порядок точности \(p\), если её ошибка ведёт себя как \(O(h^p)\) при \(h\to0\).

Итог по порядкам. Для гладкой \(f\): \[ \text{трапеция: } R=O(h^2),\qquad \text{Симпсон: } R=O(h^4). \] Удвоение числа узлов (\(h\to h/2\)) уменьшает ошибку трапеций в \(2^2=4\) раза, ошибку Симпсона — в \(2^4=16\) раз. Поэтому на графике \(\log|R|\) против \(\log n\) точки ложатся на прямые с наклонами \(-2\) и \(-4\) — ровно это видно в интерактиве.

Высокий порядок означает, что для заданной точности Симпсону нужно радикально меньше вычислений \(f\), чем трапеции. Естественно спросить: нельзя ли «выжать» высокий порядок из дешёвой трапеции, не программируя Симпсона напрямую? Можно — экстраполяцией.

Экстраполяция Ромберга

Составная трапеция обладает замечательным свойством: её ошибка раскладывается в ряд только по чётным степеням \(h\) (формула Эйлера–Маклорена):

\[ T(h) = I + c_1 h^2 + c_2 h^4 + c_3 h^6 + \dots \tag{7.2} \]

Это идеальная ситуация для экстраполяции Ричардсона: зная \(T(h)\) и \(T(h/2)\), скомбинируем их так, чтобы убить ведущий член \(c_1 h^2\). Поскольку \(T(h/2)=I+c_1 h^2/4+\dots\), линейная комбинация \((4\,T(h/2)-T(h))/3\) обнуляет \(h^2\):

\[ R_1(h)=\frac{4\,T(h/2)-T(h)}{3}=I+O(h^4). \tag{7.3} \]
Сюрприз. Полученная комбинация \(R_1\) — это в точности составная формула Симпсона! Симпсон оказывается «трапецией, проэкстраполированной по Ричардсону». Продолжив, убиваем \(h^4\): \(R_2=(16\,R_1(h/2)-R_1(h))/15=I+O(h^6)\), и так далее.

Систематически это оформляют в таблицу Ромберга. Пусть \(T_{k,0}=T(h_k)\) — трапеция с \(h_k=(b-a)/2^k\). Тогда

\[ T_{k,j}=\frac{4^{\,j}\,T_{k,j-1}-T_{k-1,j-1}}{4^{\,j}-1},\qquad T_{k,j}=I+O\big(h_k^{\,2(j+1)}\big). \tag{7.4} \]

Столбец \(j=0\) — трапеция (\(O(h^2)\)), столбец \(j=1\) — Симпсон (\(O(h^4)\)), \(j=2\) даёт \(O(h^6)\) и т. д. Диагональ \(T_{k,k}\) сходится исключительно быстро — в практикуме мы получили \(10^{-12}\) уже на шестой строке.

Рис. 2. Таблица Ромберга. Левый столбец — трапеции с удваивающимся числом узлов; каждый шаг вправо комбинирует соседа слева и левого-верхнего, повышая порядок на два.

Почему не брать просто много узлов Ньютона–Котеса

Раз Симпсон (3 узла) лучше трапеции (2 узла), не взять ли формулу Ньютона–Котеса с \(20\) равноотстоящими узлами и получить порядок \(O(h^{20})\)? Нет. Здесь повторяется беда интерполяции из занятия об интерполяционных полиномах.

Предупреждение. Формулы Ньютона–Котеса высокого порядка (много равноотстоящих узлов) неустойчивы: начиная примерно с \(8\)–\(10\) узлов среди весов \(w_i\) появляются отрицательные. Тогда сумма \(\sum w_i f(x_i)\) теряет значимость (вычитание близких больших чисел), а сама квадратура страдает от феномена Рунге — интерполяционный полином дико осциллирует у концов отрезка.

Практический вывод: высокий порядок добывают не наращиванием числа равноотстоящих узлов в одной формуле, а одним из двух надёжных путей:

  • Составные формулы низкого порядка (трапеция, Симпсон) на мелкой сетке — все веса положительны, устойчиво;
  • Экстраполяция Ромберга над составной трапецией — высокий эффективный порядок без отрицательных весов;
  • Квадратуры Гаусса — неравноотстоящие узлы, подобранные оптимально: \(n\) узлов дают точность \(2n-1\) при положительных весах. Это тема следующего занятия.
Анонс занятия 8. Откажемся от требования равноотстоящих узлов и спросим: где разместить \(n\) узлов и какие веса взять, чтобы точность была максимально возможной? Ответ — узлы в корнях ортогональных многочленов (Лежандра) — даёт квадратуры Гаусса с алгебраической точностью \(2n-1\).

Итоги занятия

Что нужно унести с занятия 7:
  1. Квадратурная формула \(\int_a^b f\approx\sum_i w_i f(x_i)\) получается интегрированием интерполяционного полинома. Равноотстоящие узлы ⟹ формулы Ньютона–Котеса.
  2. Базовые формулы: средние прямоугольники и трапеция — точность \(1\); Симпсон (парабола по 3 узлам) — точность \(3\) благодаря симметрии (кубический член «бесплатно»).
  3. Составные формулы: трапеция \(h(\tfrac12 f_0+f_1+\dots+\tfrac12 f_n)\), ошибка \(O(h^2)\); Симпсон \(\tfrac{h}{3}(f_0+4f_1+2f_2+\dots+f_n)\), \(n\) чётное, ошибка \(O(h^4)\).
  4. Порядок \(p\): удвоение \(n\) уменьшает ошибку в \(2^p\) раз (\(4\times\) для трапеции, \(16\times\) для Симпсона). Проверяется наклоном графика ошибки в log-log.
  5. Ромберг = Ричардсон над трапецией: \(T_{k,j}=\dfrac{4^j T_{k,j-1}-T_{k-1,j-1}}{4^j-1}\); столбец \(j=1\) совпадает с Симпсоном, диагональ сходится очень быстро.
  6. Высокие формулы НК неустойчивы (отрицательные веса, Рунге). Надёжно: составные формулы, Ромберг или квадратуры Гаусса (зан. 8).
