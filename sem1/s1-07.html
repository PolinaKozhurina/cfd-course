Метод неопределённых коэффициентов. Геометрическая интерпретация

Метод неопределённых коэффициентов является конструктивным способом построения разностных схем заданного порядка точности: фиксируется шаблон (stencil), вводятся неизвестные коэффициенты, после чего условия аппроксимации (и при необходимости дополнительные свойства) задают систему алгебраических условий на коэффициенты.

Общая схема метода

Метод неопределённых коэффициентов состоит в следующем:
  1. выбирается шаблон разностной схемы и общий вид разностного оператора с неизвестными коэффициентами;
  2. гладкая функция \(u(x,t)\) подставляется в схему и разлагается в ряд Тейлора;
  3. коэффициенты при производных приравниваются коэффициентам исходного уравнения, что даёт систему условий аппроксимации;
  4. оставшиеся степени свободы (если они есть) используются для обеспечения дополнительных свойств: устойчивости, монотонности, диссипативности и т.п.
Замечание. Условия аппроксимации имеют вид линейных алгебраических ограничений на коэффициенты схемы; множество схем заданного порядка точности образует аффинное подпространство в пространстве коэффициентов.

Геометрическая интерпретация

Рассмотрим пространство коэффициентов схемы как \(\mathbb{R}^m\), где \(m\) — число неизвестных параметров. Каждое условие аппроксимации задаёт гиперплоскость. Пересечение гиперплоскостей даёт аффинное множество схем, аппроксимирующих уравнение с данным порядком.

Теорема 1. Для линейной схемы с фиксированным шаблоном условия аппроксимации порядка \(p\) задают систему линейных уравнений на коэффициенты схемы. Множество решений либо пусто, либо является аффинным подпространством размерности \(m - r\), где \(r\) — ранг системы.
Геометрическая интерпретация: плоскость — нулевой порядок, прямая — первый порядок (семейство схем), точка — второй порядок (Лакс–Вендрофф).

Пример: построение трёхточечных схем для уравнения переноса

Рассмотрим уравнение переноса \(u_t + au_x = 0\), \(a = \mathrm{const}\). Пусть выбран явный трёхточечный шаблон:

\[ u_i^{n+1} = \alpha\, u_{i-1}^n + \beta\, u_i^n + \gamma\, u_{i+1}^n, \tag{7.1} \]

где \(\alpha, \beta, \gamma\) — неопределённые коэффициенты. Введём число Куранта \(\nu = a\tau/h\).

Условие сохранения постоянного решения (нулевой порядок): \(\alpha + \beta + \gamma = 1\).

Условие аппроксимации первого порядка:

\[ \alpha + \beta + \gamma = 1, \qquad -\alpha + \gamma = -\nu. \tag{7.2} \]

Условие аппроксимации второго порядка добавляет:

\[ \alpha + \gamma = \nu^2. \tag{7.3} \]

Из системы (7.2)–(7.3) единственным образом определяются все три коэффициента:

\[ \alpha = \frac{\nu(\nu + 1)}{2}, \qquad \beta = 1 - \nu^2, \qquad \gamma = \frac{\nu(\nu - 1)}{2}. \]

Схема Лакса–Вендроффа

Замечание. Полученная схема совпадает со схемой Лакса–Вендроффа: \[ u_i^{n+1} = u_i^n - \frac{\nu}{2}(u_{i+1}^n - u_{i-1}^n) + \frac{\nu^2}{2}(u_{i+1}^n - 2u_i^n + u_{i-1}^n). \]

Если требовать только условия первого порядка (7.2), то выбор параметра (например, суммы \(\alpha + \gamma\)) порождает семейство схем с различной диссипацией. В частности: противопоточная схема первого порядка получается при \(\gamma = 0\) (для \(a > 0\)), что даёт \(\alpha = \nu\), \(\beta = 1 - \nu\); центральные схемы соответствуют симметричным выборам коэффициентов, но требуют дополнительного анализа устойчивости (см. §5).

Упражнение

Упражнение. Для семейства схем первого порядка (7.1)–(7.2) выполнить устойчивостный анализ по фон Нейману и выразить область устойчивости в терминах свободного параметра (например, \(\sigma = \alpha + \gamma\)) и числа Куранта \(\nu\).
