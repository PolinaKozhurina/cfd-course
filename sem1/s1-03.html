Теорема Лакса

Понятия сходимости, аппроксимации и устойчивости связывает фундаментальная теорема Лакса.

Формулировка

Теорема 1 (Лакс). Пусть разностная схема \(L_h u_h = f_h\) аппроксимирует дифференциальную задачу \(Lu = f\) на решении \(u\) с порядком \(p\) и устойчива. Тогда решение \(u_h\) разностной задачи сходится к решению дифференциальной задачи \(Lu = f\), причём имеет место оценка: \[ \|u_h - U_h\| \le C_1 C_2\, h^p = C\, h^p, \tag{3.1} \] где \(U_h\) — проекция точного решения на сетку.

Доказательство

Из определения устойчивости разностной задачи следует, что для двух близких решений \(u_h\) и \(\nu_h\) разностных уравнений

\[ L_h u_h = f_h + \eta_h, \tag{3.2} \] \[ L_h \nu_h = f_h + \eta'_h, \tag{3.3} \]

где \(\eta_h\) и \(\eta'_h\) — малые возмущения, выполняется

\[ \|u_h - \nu_h\| \le C_2\bigl(\|\eta_h\| + \|\eta'_h\|\bigr). \tag{3.4} \]

Положим, что \(u_h\) есть точное решение разностного уравнения \(L_h u_h = f_h\) (то есть \(\eta_h = 0\)). И положим, что \(\nu_h = U_h\) есть проекция точного решения дифференциальной задачи на расчётную сетку. Тогда

\[ \eta'_h = L_h U_h - f_h = \xi_h. \tag{3.5} \]

Перепишем систему в виде

\[ L_h u_h = f_h, \tag{3.6} \] \[ L_h U_h = f_h + \xi_h. \tag{3.7} \]

Отсюда по определению устойчивости следует

\[ \|u_h - U_h\| \le C_2\,\|\xi_h\|. \tag{3.8} \]

Из определения аппроксимации имеем \(\|\xi_h\| \xrightarrow{h \to 0} 0\), откуда

\[ \|u_h - U_h\| \xrightarrow{h \to 0} 0. \tag{3.9} \]

Кроме того, учитывая порядок аппроксимации, получаем оценку

\[ \|u_h - U_h\| \le C_2\,\|\xi_h\| \le C_2\, C_1\, h^p = C\, h^p. \tag{3.10} \]
Логика доказательства: невязка \(\xi_h\) из аппроксимации подставляется в оценку устойчивости.
Ключевое следствие: Для проверки сходимости разностной схемы достаточно проверить два свойства — аппроксимацию и устойчивость. Аппроксимация обычно проверяется сравнительно просто (разложением в ряд Тейлора). Основная трудность состоит в доказательстве устойчивости.
