Метод МакКормака. Явная предикторно-корректорная схема

Блок-схема метода МакКормака: предиктор (разность вперёд) + корректор (разность назад) = второй порядок.

Общая характеристика

Метод МакКормака относится к классу явных предикторно-корректорных разностных схем для гиперболических уравнений. Метод обеспечивает второй порядок аппроксимации по времени и пространству и отличается простотой реализации, поскольку не требует решения задачи Римана.

Шаг предиктора

\[ u_i^* = u_i^n - \frac{\Delta t}{\Delta x}\bigl(f(u_{i+1}^n) - f(u_i^n)\bigr). \tag{27.1} \]

Шаг корректора

\[ u_i^{n+1} = \frac{1}{2}\left(u_i^n + u_i^*\right) - \frac{\Delta t}{2\Delta x}\bigl(f(u_i^*) - f(u_{i-1}^*)\bigr). \tag{27.2} \]

Порядок аппроксимации

Чередование направлений разностей (вперёд на предикторе, назад на корректоре) обеспечивает компенсацию первых погрешностей и итоговый второй порядок: \(O(\Delta x^2, \Delta t^2)\).

Устойчивость

Условие устойчивости метода МакКормака: \(|a|\Delta t / \Delta x \le 1\), где \(a\) — скорость переноса. Для нелинейных задач шаг по времени выбирается из условия Куранта по максимальной характеристической скорости.

Недостатки и область применения

Основным недостатком метода МакКормака является отсутствие монотонности: вблизи разрывов возникают нефизические осцилляции. Метод наиболее эффективен для расчёта гладких течений, акустических волн и задач, в которых разрывы играют второстепенную роль.
