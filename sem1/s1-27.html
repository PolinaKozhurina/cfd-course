Метод МакКормака. Явная предикторно-корректорная схема

Рис. 1. Блок-схема метода МакКормака: предиктор (вперёд) + корректор (назад) = второй порядок.

Общая характеристика метода

Метод МакКормака (R. W. MacCormack, 1969) — явная предикторно-корректорная разностная схема второго порядка аппроксимации по времени и пространству для гиперболических уравнений и систем законов сохранения. Главные достоинства: предельная простота реализации, отсутствие необходимости в решателе задачи Римана и спектральном разложении матриц Якоби. Исторически метод широко применялся в задачах сверхзвуковой газовой динамики, включая расчёты обтекания летательных аппаратов.

Платой за простоту является немонотонность: метод не является TVD и генерирует осцилляции вблизи разрывов решения. Для подавления этих осцилляций применяется искусственная вязкость — простой, но снижающий порядок в окрестности разрывов подход. В современных кодах МакКормак вытеснен TVD-схемами с решателями Римана (§19-24), однако остаётся полезным инструментом там, где разрывы либо отсутствуют, либо играют второстепенную роль.

Постановка задачи

Рассмотрим одномерный скалярный закон сохранения

\[ \frac{\partial u}{\partial t} + \frac{\partial f(u)}{\partial x} = 0 \tag{27.1} \]

на равномерной сетке \(x_i = i\Delta x\), \(t_n = n\Delta t\). Метод МакКормака строится в виде двух последовательных шагов: предиктора, дающего приближение решения на промежуточном слое с использованием односторонних разностей, и корректора, симметризующего схему с помощью разностей противоположного направления.

Шаг предиктора

На первом шаге выполняется явный прогноз значения решения на следующем временном слое с использованием разности вперёд в пространственной производной:

\[ u_i^{*} = u_i^n - \frac{\Delta t}{\Delta x}\bigl(f(u_{i+1}^n) - f(u_i^n)\bigr). \tag{27.2} \]

Полученное значение \(u_i^{*}\) является предварительным приближением решения на \(t^{n+1}\). Сама по себе формула (27.2) — это схема первого порядка (аналог «уголка» с разностью вперёд), но она нужна лишь как промежуточное вспомогательное значение.

Шаг корректора

На втором шаге используется информация, полученная на стадии предиктора, и выполняется симметризация с применением разности назад:

\[ u_i^{n+1} = \frac{1}{2}\left[u_i^n + u_i^{*} - \frac{\Delta t}{\Delta x}\bigl(f(u_i^{*}) - f(u_{i-1}^{*})\bigr)\right]. \tag{27.3} \]

Полусумма «исходного» значения \(u_i^n\) и предсказания \(u_i^{*}\) обеспечивает усреднение по времени, а разность назад по пространственной производной на звёздном слое «гасит» перекос, внесённый шагом (27.2).

Доказательство второго порядка аппроксимации

Разложим \(f(u_{i+1}^n)\) и \(f(u_{i-1}^{*})\) в ряд Тейлора. Обозначая \(f_x = \partial_x f(u(x_i, t^n))\), получаем для предиктора:

\[ f(u_{i+1}^n) - f(u_i^n) = \Delta x\, f_x + \frac{\Delta x^2}{2} f_{xx} + O(\Delta x^3). \] \[ u_i^{*} = u_i^n - \Delta t\, f_x - \frac{\Delta t\,\Delta x}{2} f_{xx} + O(\Delta t\,\Delta x^2). \]

Связь \(u_i^{*}\) с точным решением: \(u(x_i, t^{n+1}) = u_i^n + \Delta t\, u_t + \tfrac{\Delta t^2}{2} u_{tt} + O(\Delta t^3)\). Поскольку \(u_t = -f_x\) и \(u_{tt} = -\partial_t f_x = -(a^2 u_x)_x\), где \(a = f'(u)\):

\[ u_i^{*} = u(x_i, t^{n+1}) - \frac{\Delta t^2}{2} u_{tt} - \frac{\Delta t\,\Delta x}{2} f_{xx} + O(\Delta t^3, \Delta t\,\Delta x^2). \]

Для шага корректора воспользуемся равенством \(f_{xx} = (a u_x)_x\) и тем, что разность назад от \(u^{*}\) аппроксимирует \(f_x\) с точностью первого порядка, но с противоположным знаком главной ошибки. Полусумма (27.3) компенсирует члены первого порядка, и после аккуратного алгебраического свёртывания:

\[ u_i^{n+1} = u(x_i, t^{n+1}) + O(\Delta t^3 + \Delta x^2\,\Delta t). \]

Относительная ошибка за шаг — \(O(\Delta t^2 + \Delta x^2)\) при \(\Delta t \sim \Delta x\), что означает второй порядок аппроксимации. ∎

Альтернативная формулировка. Шаги (27.2)–(27.3) можно переписать эквивалентно как схему Лакса-Вендроффа без явного вычисления якобиана \(a = f'(u)\). Для линейного уравнения \(u_t + a u_x = 0\) метод МакКормака совпадает с Лаксом-Вендроффом в точности. Для нелинейных задач МакКормак даёт аналог Лакса-Вендроффа без необходимости аппроксимировать \(f'\) разностно, что особенно удобно для уравнений газовой динамики с табличными уравнениями состояния.

Устойчивость и условие Куранта

Линейный анализ устойчивости по Фурье для линейного уравнения \(u_t + a u_x = 0\) даёт (тождественно совпадает с Лаксом-Вендроффом):

\[ |\lambda(kh)|^2 = 1 - r^2(1-r^2)\bigl(1 - \cos kh\bigr)^2, \qquad r = \frac{a\,\Delta t}{\Delta x}. \]

Условие \(|\lambda| \le 1\) выполняется при \(|r| \le 1\) — классическое условие Куранта-Фридрихса-Леви. Для систем уравнений, аналогично, шаг по времени ограничен через максимальную характеристическую скорость:

\[ \Delta t \le \mathrm{CFL}\cdot\frac{\Delta x}{\max_i\bigl(|v_i| + a_i\bigr)}, \qquad \mathrm{CFL} \le 1. \]

Применение к системам законов сохранения

Для вектора консервативных переменных \(\mathbf{u} = (\rho, \rho v, E)^{\!\top}\) и потока \(\mathbf{f}(\mathbf{u}) = (\rho v, \rho v^2 + p, v(E+p))^{\!\top}\) метод применяется покомпонентно:

\[ \mathbf{u}_i^{*} = \mathbf{u}_i^n - \frac{\Delta t}{\Delta x}\bigl(\mathbf{f}(\mathbf{u}_{i+1}^n) - \mathbf{f}(\mathbf{u}_i^n)\bigr), \tag{27.4} \]
\[ \mathbf{u}_i^{n+1} = \frac{1}{2}\left[\mathbf{u}_i^n + \mathbf{u}_i^{*} - \frac{\Delta t}{\Delta x}\bigl(\mathbf{f}(\mathbf{u}_i^{*}) - \mathbf{f}(\mathbf{u}_{i-1}^{*})\bigr)\right]. \tag{27.5} \]

Метод остаётся явным и не требует решения никаких дополнительных задач на каждом временном шаге — это ключевое преимущество. Поток \(\mathbf{f}(\mathbf{u})\) вычисляется тривиально по формулам замыкания (УРС): вся схема сводится к трём проходам по массиву за шаг.

Вариант с чередующимися направлениями. Существует симметричная версия МакКормака, где направления предиктора и корректора чередуются от шага к шагу: на нечётных шагах предиктор вперёд / корректор назад, на чётных — наоборот. Это уменьшает анизотропию схемы и улучшает симметрию разрешения разрывов.

Искусственная вязкость

Метод МакКормака не является монотонным. На ступеньке или ударной волне возникают осцилляции — типичное поведение немонотонной схемы второго порядка (см. §8 и §9, теорема Годунова).

Классическое средство подавления осцилляций — искусственная вязкость в форме, предложенной фон Нейманом и Рихтмайером. После шага корректора применяется дополнительная диффузионная коррекция:

\[ \mathbf{u}_i^{n+1} \leftarrow \mathbf{u}_i^{n+1} + \epsilon_i\bigl(\mathbf{u}_{i+1}^{n+1} - 2\mathbf{u}_i^{n+1} + \mathbf{u}_{i-1}^{n+1}\bigr), \tag{27.6} \]

где коэффициент вязкости \(\epsilon_i\) выбирается пропорциональным локальному градиенту давления:

\[ \epsilon_i = C_\text{art}\cdot \frac{|\Delta_2 p_i|}{|p_{i+1}| + 2|p_i| + |p_{i-1}|}, \]

\(\Delta_2 p_i = p_{i+1} - 2p_i + p_{i-1}\), \(C_\text{art} \sim 1 \div 3\). Вязкость включается только на разрывах (где \(\Delta_2 p\) велико относительно \(|p|\)) и остаётся мала на гладких участках — но всё равно снижает порядок схемы в окрестности разрывов до первого.

Цена искусственной вязкости. Вязкость в форме (27.6) стабилизирует решение, но не является TVD-безопасной: при слишком малом \(C_\text{art}\) осцилляции остаются, при слишком большом — сильно размазываются все разрывы. Подбор \(C_\text{art}\) является эвристическим, что и стало одной из причин вытеснения МакКормака современными TVD/WENO-схемами.

Недостатки и область применения

  1. Немонотонность — осцилляции на разрывах без искусственной вязкости.
  2. Падение порядка на разрывах при включённой искусственной вязкости до первого.
  3. Эмпирическая настройка параметра \(C_\text{art}\) — один коэффициент не универсален для разных задач.

Метод МакКормака наиболее эффективен для расчёта гладких течений, акустических волн и задач, в которых разрывы либо отсутствуют, либо играют второстепенную роль (например, ламинарное обтекание крыла при дозвуковых режимах, распространение волн в атмосфере).
