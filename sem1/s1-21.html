Метод HLL (Harten–Lax–van Leer)

Положение метода HLL среди приближённых решателей

Метод HLL (Harten–Lax–van Leer) относится к классу многоволновых приближённых решателей задачи Римана. В отличие от линеаризованных методов (§19) и методов интегрирования вдоль пути (§20), метод HLL не использует полное характеристическое разложение системы, а аппроксимирует решение задачи Римана двумя предельными волнами, ограничивающими область взаимодействия. Такой подход обеспечивает простоту алгоритма и высокую устойчивость схемы при сохранении консервативности.

Двухволновая аппроксимация решения задачи Римана

Рассмотрим задачу Римана для одномерной системы законов сохранения \(\partial_t\mathbf{u} + \partial_x\mathbf{f}(\mathbf{u}) = 0\) с начальными состояниями \(\mathbf{u}_L\) и \(\mathbf{u}_R\). В методе HLL предполагается, что все элементарные волны, возникающие в точном решении, распространяются внутри интервала между двумя скоростями

\[ S_L \le \frac{x}{t} \le S_R, \]

где \(S_L\) и \(S_R\) — оценки минимальной и максимальной скоростей распространения волн в решении задачи Римана.

Решение вне интервала волн

Вне этого интервала решение считается постоянным и совпадает с исходными состояниями \(\mathbf{u}_L\) (при \(x/t < S_L\)) и \(\mathbf{u}_R\) (при \(x/t > S_R\)) соответственно.

Промежуточное состояние HLL

Внутри области \(S_L t < x < S_R t\) решение аппроксимируется одним постоянным промежуточным состоянием \(\mathbf{u}^*\). Это состояние определяется из интегральных соотношений баланса по области взаимодействия волн.

Вывод промежуточного состояния

Используя законы сохранения, запишем баланс консервативных величин в области \([S_L t, S_R t]\):

\[ (S_R - S_L)\mathbf{u}^* = S_R\mathbf{u}_R - S_L\mathbf{u}_L - \bigl(\mathbf{f}(\mathbf{u}_R) - \mathbf{f}(\mathbf{u}_L)\bigr). \]

Откуда получаем выражение

\[ \mathbf{u}^* = \frac{S_R\mathbf{u}_R - S_L\mathbf{u}_L - \mathbf{f}(\mathbf{u}_R) + \mathbf{f}(\mathbf{u}_L)}{S_R - S_L}. \tag{21.1} \]

Таким образом, метод HLL заменяет сложную волновую структуру точного решения задачи Римана на простую двухволновую модель с одним промежуточным состоянием.

Численный поток метода HLL

Численный поток определяется в зависимости от знаков скоростей \(S_L\) и \(S_R\) и имеет следующий вид:

\[ \mathbf{F}_\text{HLL} = \begin{cases} \mathbf{f}(\mathbf{u}_L), & S_L \ge 0, \\[8pt] \dfrac{S_R\mathbf{f}(\mathbf{u}_L) - S_L\mathbf{f}(\mathbf{u}_R) + S_L S_R(\mathbf{u}_R - \mathbf{u}_L)}{S_R - S_L}, & S_L < 0 < S_R, \\[8pt] \mathbf{f}(\mathbf{u}_R), & S_R \le 0. \end{cases} \tag{21.2} \]

Эта формула является центральным результатом метода HLL и полностью определяет численный поток через границу ячейки.

Вывод формулы потока

Поток (21.2) получается из условий Ранкина–Гюгонио на каждой из двух волн. Для левой волны:

\[ \mathbf{F}^* = \mathbf{f}(\mathbf{u}_L) + S_L(\mathbf{u}^* - \mathbf{u}_L). \]

Подстановка (21.1) и упрощение даёт среднюю ветвь формулы (21.2). Крайние ветви соответствуют случаям, когда обе волны бегут в одну сторону (сверхзвуковое течение).

Оценка скоростей волн

Ключевым элементом метода HLL является выбор оценок скоростей \(S_L\) и \(S_R\). На практике используются оценки, основанные на характеристических скоростях матрицы Якоби. Для уравнений Эйлера часто принимают

\[ S_L = \min(v_L - a_L,\; v_R - a_R), \qquad S_R = \max(v_L + a_L,\; v_R + a_R), \]

где \(v\) — скорость потока, \(a\) — скорость звука в соответствующем состоянии. Такой выбор обеспечивает, что область взаимодействия волн гарантированно содержит все волны точного решения.

Альтернативные оценки. Помимо простых оценок по характеристическим скоростям, существуют более точные формулы с учётом средних по Рое (Einfeldt, 1988) или акустических приближений. Улучшение оценок \(S_L, S_R\) повышает точность метода без увеличения вычислительных затрат.

Свойства метода HLL

Метод HLL является консервативным и устойчивым при выполнении условия Куранта. Он не требует вычисления собственных векторов системы и потому отличается простотой и вычислительной эффективностью. Благодаря учёту предельных скоростей волн метод обладает выраженными диссипативными свойствами, что делает его надёжным при расчёте разрывных решений.

Основной недостаток

Основной недостаток метода HLL — неспособность корректно описывать контактные разрывы, поскольку вся волновая структура аппроксимируется одним промежуточным состоянием. Контактный разрыв (разрыв плотности при непрерывном давлении и скорости) полностью размазывается.

Этот недостаток послужил мотивацией для разработки модификации HLLC (§22), которая дополнительно учитывает контактный разрыв.
