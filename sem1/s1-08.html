Консервативность и монотонность разностных схем

Предыдущие параграфы были посвящены локальным свойствам разностных схем — аппроксимации, устойчивости, дисперсии и диссипации. Теперь перейдём к глобальным свойствам решения: сохранению интегральных величин на дискретной сетке и отсутствию нефизических осцилляций на разрывах. Обе характеристики критически важны для задач газовой динамики, где присутствуют ударные волны и контактные разрывы.

Рассматривается одномерный скалярный закон сохранения

\[ \frac{\partial u}{\partial t} + \frac{\partial f(u)}{\partial x} = 0, \tag{8.1} \]

для которого строится разностная аппроксимация на равномерной пространственно-временной сетке с шагами \(\Delta x\) и \(\Delta t\).

Консервативная форма разностной схемы

Определение 1. Разностная схема для уравнения (8.1) называется консервативной, если она может быть записана в виде \[ u_i^{n+1} = u_i^n - \frac{\Delta t}{\Delta x}\bigl(F_{i+1/2}^n - F_{i-1/2}^n\bigr), \tag{8.2} \] где \(F_{i+1/2}^n\) — численный поток, одинаково входящий в уравнения для двух соседних ячеек (с индексом \(i\) слева и \(i+1\) справа).

Консервативная форма (8.2) является дискретным аналогом интегральной формы закона сохранения. Проинтегрируем уравнение (8.1) по ячейке \([x_{i-1/2}, x_{i+1/2}]\) за шаг \(\Delta t\):

\[ \int_{x_{i-1/2}}^{x_{i+1/2}}\!\!\! u(x, t^{n+1})\,dx - \int_{x_{i-1/2}}^{x_{i+1/2}}\!\!\! u(x, t^n)\,dx = -\int_{t^n}^{t^{n+1}}\!\!\bigl[f(u(x_{i+1/2}, t)) - f(u(x_{i-1/2}, t))\bigr]\,dt. \]

Обозначая осреднённые по ячейке значения как \(u_i^n\) и осреднённые по времени потоки как \(F_{i\pm 1/2}^n\), мы немедленно получаем (8.2).

Телескопическое сокращение. Ключевое достоинство формы (8.2) — телескопическая структура. Просуммируем уравнение по всем ячейкам \(i = 1, \ldots, N\): \[ \sum_{i=1}^{N} u_i^{n+1} = \sum_{i=1}^{N} u_i^n - \frac{\Delta t}{\Delta x}\sum_{i=1}^{N}\bigl(F_{i+1/2}^n - F_{i-1/2}^n\bigr) = \sum_{i=1}^{N} u_i^n - \frac{\Delta t}{\Delta x}\bigl(F_{N+1/2}^n - F_{1/2}^n\bigr). \] Все внутренние потоки взаимно сокращаются; остаются только граничные. В отсутствие втекания/вытекания интегральная величина \(\sum u_i^n \cdot \Delta x\) сохраняется в точности на уровне машинной точности.
Требование согласованности потока: на гладком решении численный поток должен удовлетворять \(F_{i+1/2}^n \to f(u_{i+1/2}^n)\) при \(\Delta x, \Delta t \to 0\). Обычно накладывается условие \(F(u, u, \ldots, u) = f(u)\) — поток от константы равен аналитическому.

Теорема Лакса–Вендроффа (о сходимости к слабому решению)

Теорема Лакса–Вендроффа (1960). Если консервативная разностная схема (8.2) с непрерывным согласованным численным потоком сходится в \(L^1_\text{loc}\) к некоторой функции \(u(x, t)\) при \(\Delta x, \Delta t \to 0\) с фиксированным отношением, то предельная функция \(u(x, t)\) является слабым решением закона сохранения (8.1).

Практическое следствие: только консервативные схемы могут правильно ловить скорость ударной волны. Если схема не консервативна, ударная волна в численном решении будет двигаться со скоростью, отличной от определяемой условием Ренкина-Гюгонио. Этот эффект можно обнаружить тривиально — достаточно решить задачу Бюргерса с начальным разрывом и сравнить положение фронта с аналитическим.

Рис. 1. Консервативная форма (8.2): поток \(F_{i+1/2}\) общий для соседних ячеек.

Монотонные схемы

Рассмотрим общий явный оператор разностной схемы:

\[ u_i^{n+1} = H\bigl(u_{i-p}^n, u_{i-p+1}^n, \ldots, u_{i+q}^n\bigr). \tag{8.3} \]
Определение 2 (Годунов, 1959). Разностная схема (8.3) называется монотонной, если оператор \(H\) не убывает по каждому аргументу: \[ \frac{\partial H}{\partial u_k^n} \ge 0 \quad \text{для всех } k = i-p, \ldots, i+q. \tag{8.4} \]

Для линейной схемы \(u_i^{n+1} = \sum_k a_k\, u_{i+k}^n\) условие (8.4) превращается в требование неотрицательности всех коэффициентов: \(a_k \ge 0\) для всех \(k\).

Монотонность по Фридрихсу

Определение 3 (Фридрихс). Разностная схема сохраняет покомпонентный порядок, если из условия \(u_i^n \ge v_i^n\) для всех \(i\) следует \(u_i^{n+1} \ge v_i^{n+1}\) для всех \(i\).

Пусть \(u^n\) и \(v^n\) — две сеточные функции, причём \(u_i^n \ge v_i^n\) для всех \(i\). Применим к ним обе одну и ту же схему. Разность \(u_i^{n+1} - v_i^{n+1}\) в силу монотонности оператора \(H\) (определение 2) тоже будет неотрицательной:

\[ u_i^{n+1} - v_i^{n+1} = H(u^n) - H(v^n) = \sum_k \frac{\partial H}{\partial u_k^n}\Big|_{\xi}\cdot \bigl(u_{i+k}^n - v_{i+k}^n\bigr) \ge 0, \]

где \(\xi\) — некоторое промежуточное значение по теореме о среднем. Таким образом, определения 2 и 3 эквивалентны.

Дискретный принцип максимума

Определение 4. Разностная схема удовлетворяет дискретному принципу максимума, если для всех \(i\) и всех \(n\) \[ \min_{k \in \mathrm{stencil}(i)} u_k^n \ \le\ u_i^{n+1}\ \le\ \max_{k \in \mathrm{stencil}(i)} u_k^n. \tag{8.5} \] Эквивалентная формулировка: \(u_i^{n+1}\) является выпуклой комбинацией значений предыдущего слоя: \[ u_i^{n+1} = \sum_k a_k\, u_{i+k}^n, \quad a_k \ge 0, \quad \sum_k a_k = 1. \tag{8.6} \]
Связь с физикой: принцип максимума запрещает появление новых локальных экстремумов. Если в начальных данных \(0 \le u^0 \le 1\), то и все последующие значения остаются в \([0, 1]\). Для газовой динамики это означает физически корректное поведение: плотность и давление остаются положительными, а разрывы не порождают нефизических «выбросов» и «впадин».

Полная вариация и TVD-свойство

Определение 5. Полной вариацией сеточной функции \(\{u_i^n\}\) называется величина \[ \mathrm{TV}(u^n) = \sum_i |u_{i+1}^n - u_i^n|. \tag{8.7}\] Аналогично для непрерывной функции \(\mathrm{TV}(u) = \int |\partial_x u|\, dx\).

Для гиперболических уравнений точное решение обладает свойством невозрастания полной вариации: \(\mathrm{TV}(u(\cdot, t_2)) \le \mathrm{TV}(u(\cdot, t_1))\) при \(t_2 \ge t_1\). Это свойство естественно потребовать и от численной схемы.

Определение 6 (Хартен, 1983). Разностная схема называется TVD (Total Variation Diminishing), если для любых начальных данных \[ \mathrm{TV}(u^{n+1}) \le \mathrm{TV}(u^n) \quad \text{для всех } n \ge 0. \tag{8.8}\]

Хартен показал, что консервативная схема вида \(u_i^{n+1} = u_i^n + C_{i+1/2}(u_{i+1}^n - u_i^n) - D_{i-1/2}(u_i^n - u_{i-1}^n)\) является TVD, если её коэффициенты удовлетворяют

\[ C_{i+1/2} \ge 0, \quad D_{i-1/2} \ge 0, \quad C_{i+1/2} + D_{i+1/2} \le 1. \tag{8.9} \]
Замечание. Условие TVD ограничивает суммарную вариацию, но формально не запрещает появления новых локальных экстремумов, если они не увеличивают \(\mathrm{TV}\) (например, два симметричных выбросика с равными амплитудами противоположного знака). Поэтому TVD — более слабое свойство, чем монотонность.

Иерархия свойств

Теорема (Хартен–Хайман–Лакс, 1976). Для линейной консервативной схемы справедливы импликации: \[ \text{монотонность (8.4)} \ \Longleftrightarrow\ \text{пол. коэфф. в (8.6)} \ \Longleftrightarrow\ \text{принцип максимума (8.5)} \ \Longrightarrow\ \text{TVD (8.8)}. \] Обратная импликация \(\text{TVD} \Rightarrow \text{монотонность}\) в общем случае неверна.
Рис. 2. Иерархия свойств схем. Монотонные ⊂ TVD ⊂ устойчивые.

Доказательство импликации «монотонность ⇒ TVD»

Пусть схема записана в форме (8.6) с коэффициентами \(a_k \ge 0\), \(\sum a_k = 1\). Вычислим

\[ u_{i+1}^{n+1} - u_i^{n+1} = \sum_k a_k\bigl(u_{i+1+k}^n - u_{i+k}^n\bigr). \]

Применяя неравенство треугольника и пользуясь неотрицательностью \(a_k\):

\[ |u_{i+1}^{n+1} - u_i^{n+1}| \le \sum_k a_k\,|u_{i+1+k}^n - u_{i+k}^n|. \]

Суммируем по \(i\) и меняем порядок сумм:

\[ \mathrm{TV}(u^{n+1}) \le \sum_k a_k \sum_i |u_{i+1+k}^n - u_{i+k}^n| = \sum_k a_k \cdot \mathrm{TV}(u^n) = \mathrm{TV}(u^n). \]

Таким образом, монотонность немедленно влечёт TVD. ∎

Обратное неверно. Схема MUSCL-minmod второго порядка является TVD (это её ключевое свойство), но её коэффициенты явно зависят от решения \(u^n\) (через ограничители) и могут становиться отрицательными для отдельных конфигураций, то есть схема не монотонна в смысле определения 2. Именно это «чуть-чуть нелинейное» поведение и позволяет ей достичь второго порядка, обходя ограничение теоремы Годунова (§9).

Анализ классических схем

1. Схема «уголок» (upwind)

Для уравнения переноса \(u_t + cu_x = 0\) с \(c > 0\):

\[ u_i^{n+1} = (1 - r)\,u_i^n + r\,u_{i-1}^n, \qquad r = \frac{c\,\Delta t}{\Delta x}. \]

Коэффициенты: \(a_0 = 1 - r\), \(a_{-1} = r\). При \(0 \le r \le 1\) оба неотрицательны, сумма равна 1 — схема монотонна, удовлетворяет принципу максимума и является TVD.

2. Схема Лакса-Вендроффа

\[ u_i^{n+1} = u_i^n - \tfrac{r}{2}(u_{i+1}^n - u_{i-1}^n) + \tfrac{r^2}{2}(u_{i+1}^n - 2u_i^n + u_{i-1}^n). \]

Коэффициенты: \(a_{-1} = \tfrac{r(r+1)}{2}\), \(a_0 = 1 - r^2\), \(a_{+1} = \tfrac{r(r-1)}{2}\). При \(0 < r < 1\) коэффициент \(a_{+1} < 0\) — схема не монотонна, принцип максимума нарушен, TVD-свойство не выполнено. Следствие: на разрывах возникают осцилляции.

3. Центральная схема

\[ u_i^{n+1} = u_i^n - \tfrac{r}{2}(u_{i+1}^n - u_{i-1}^n). \]

Коэффициенты \(a_{\pm 1} = \mp r/2\) имеют разные знаки — не монотонна ни при каких \(r\). Более того, она безусловно неустойчива (анализ фон Неймана даёт \(|\lambda|^2 = 1 + r^2\sin^2\theta > 1\)).

Интерактив: монотонная vs немонотонная схемы

Шаг: 0  │  TV(уголок) = —  │  TV(Л-В) = —  │  диапазон Л-В = —
точное решение уголок (монотонная) Лакс-Вендрофф (немонотонная)
Ключевой вывод. Консервативность гарантирует правильную скорость разрывов (теорема Лакса-Вендроффа), монотонность — отсутствие нефизических осцилляций и соблюдение физических ограничений (плотность/давление положительны). Однако теорема Годунова (следующий параграф) утверждает, что линейная монотонная схема не может иметь порядок выше первого. Выход из этого тупика — нелинейные TVD/MUSCL-схемы с ограничителями, рассматриваемые в §23-24.
Упражнение 1. Проверьте непосредственно, что схема Лакса–Фридрихса \[ u_i^{n+1} = \tfrac{1}{2}(u_{i+1}^n + u_{i-1}^n) - \tfrac{r}{2}(u_{i+1}^n - u_{i-1}^n) \] монотонна при \(|r| \le 1\), и вычислите её коэффициент численной диффузии.
Упражнение 2. Покажите в интерактиве выше, что \(\mathrm{TV}\) уголка не возрастает ни при каких \(r \in [0,1]\) и ни для каких начальных данных, а \(\mathrm{TV}\) Лакса-Вендроффа может возрастать (особенно на разрыве).
