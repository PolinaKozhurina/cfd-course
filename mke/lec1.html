Лекция 1 · Метод конечных элементов

Введение. Общее представление о методе конечных элементов

Что такое метод конечных элементов

Метод конечных элементов (МКЭ) относится к классу сеточных методов, поскольку он основан на дискретизации области решения — её разбиении на сетку из элементов. Вычислительные модели на основе метода сеток позволяют найти приближённое решение начально-краевой задачи, заменив исходные дифференциальные уравнения аппроксимирующей системой алгебраических уравнений.

МКЭ часто выделяют в отдельную группу наряду с методом конечных разностей (МКР) и методом конечных объёмов (МКО). Причина — в проекционной или вариационной природе различных вариантов метода и в способе аппроксимации функций внутри элементов. В настоящее время МКЭ широко применяется для решения начально-краевых задач, особенно в механике, физике и инженерии.

Определение. Конечный элемент — это подобласть (отрезок, треугольник, тетраэдр и т.п.), на которую разбивается расчётная область, и на которой искомое решение представляется простой функцией (как правило, полиномом невысокой степени). Совокупность таких элементов с согласованными на границах функциями даёт кусочно-полиномиальную аппроксимацию решения по всей области.

Знакомство с методом мы начнём с примеров построения приближённых решений краевых задач с помощью полиномов: сначала рассмотрим метод коллокаций, затем обобщим его до метода взвешенных невязок и проекционных методов, и лишь после этого придём к идее конечного элемента.

Место МКЭ среди сеточных методов

Метод сеток — наиболее распространённый способ решения начально-краевых задач. Сеточные алгоритмы сводят нахождение приближённого решения к решению системы алгебраических уравнений, аппроксимирующих исходное дифференциальное уравнение. Принципиальное различие конечно-разностных (КР) и конечно-элементных (КЭ) подходов состоит в следующем:

  • В МКР аппроксимируются сами уравнения и граничные условия (производные заменяются разностными отношениями). Результат — конечномерный вектор значений решения в узлах сетки.
  • В МКЭ аппроксимируется искомое решение. Результат — приближённая, но аналитическая по всей области функция, заданная кусочно на элементах.

МКЭ можно рассматривать как разновидность метода взвешенных невязок, в котором весовые функции имеют носители, ограниченные малыми интервалами расчётной области, а решение аппроксимируется полиномами невысокой степени (например, кусочно-линейными). Эта связь и будет лейтмотивом лекции.

Три основных сеточных подхода. МКР дискретизирует уравнения, МКО — интегральные балансы, МКЭ аппроксимирует само решение проекционными и вариационными средствами.

Пример: построение решения методом коллокаций

Рассмотрим краевую задачу для линейного ОДУ 2-го порядка (А. Ф. Филиппов, «Сборник задач по дифференциальным уравнениям»):

\[ \hat L(y) \equiv y'' + y' - 2y = 0,\qquad 0 \le x \le 4,\qquad y(0) = 0,\quad y(4) = 1. \tag{1.1} \]

Это краевая задача для модельного стационарного уравнения (теплопроводности, конвекции-диффузии и т.п.); \(\hat L\) — линейный дифференциальный оператор. Характеристическое уравнение \(\lambda^2 + \lambda - 2 = 0\) даёт корни \(\lambda_1 = 1\), \(\lambda_2 = -2\), поэтому общее решение

\[ y(x) = c_1 e^{x} + c_2 e^{-2x},\qquad c_1, c_2 = \mathrm{const}. \]

Удовлетворяя граничным условиям, получаем точное решение

\[ y(x) = \frac{e^{x} - e^{-2x}}{e^{4} - e^{-8}}. \tag{1.2} \]

Это дважды непрерывно дифференцируемая функция; её значения в отдельных точках: \(\;y(1) = 0{,}0473,\quad y(2) = 0{,}1350,\quad y(3) = 0{,}3679.\)

Рис. 1. Точное решение краевой задачи (1.1)–(1.2). Решение круто нарастает у правого конца отрезка; синими точками отмечены значения \(y(1), y(2), y(3), y(4)\).

Полиномиальный анзац

Построим приближённое решение в аналитическом виде — в виде полинома 3-й степени:

\[ \tilde y(x) = a_0 + a_1 x + a_2 x^2 + a_3 x^3 \approx y(x). \tag{1.3} \]

Коэффициенты \(a_0,\dots,a_3\) — неизвестные, которые надо определить из четырёх условий. Два условия задаёт сама краевая задача:

\[ \tilde y(0) = 0 \;\Rightarrow\; a_0 = 0,\qquad \tilde y(4) = 1 \;\Rightarrow\; a_0 + 4a_1 + 16a_2 + 64a_3 = 1. \]

Ещё два уравнения получим методом коллокаций. Подставим (1.3) в оператор \(\hat L\). Приводя подобные члены, получим выражение для невязки:

\[ \hat L(\tilde y) = (-2a_0 + a_1 + 2a_2) + 2(-a_1 + a_2 + 3a_3)\,x + (-2a_2 + 3a_3)\,x^2 - 2a_3\,x^3. \tag{1.4} \]

Условие \(\hat L(\tilde y) = 0\), вообще говоря, не может выполняться при всех \(x \in (0,4)\) — иначе полином совпадал бы с точным решением, что невозможно (точное решение содержит экспоненты). Поэтому потребуем обращения невязки в ноль лишь в нескольких избранных точках — узлах коллокации. Возьмём для удобства вычислений точки \(x = 0\) и \(x = 1\):

\[ \hat L\bigl(\tilde y(0)\bigr) = 0,\qquad \hat L\bigl(\tilde y(1)\bigr) = 0. \]

Вместе с граничными условиями это даёт систему линейных уравнений:

\[ \begin{cases} a_0 = 0, \\ a_1 + 2a_2 = 0, \\ -a_1 + 2a_2 + 7a_3 = 0, \\ 4a_1 + 16a_2 + 64a_3 = 1. \end{cases} \tag{1.5} \]
Из второго уравнения \(a_1 = -2a_2\). Подставив в третье: \(2a_2 + 2a_2 + 7a_3 = 0\), откуда \(a_2 = -\tfrac{7}{4}a_3\) и \(a_1 = \tfrac{7}{2}a_3\). Четвёртое уравнение даёт \(4\cdot\tfrac{7}{2}a_3 + 16\cdot(-\tfrac{7}{4}a_3) + 64a_3 = 14a_3 - 28a_3 + 64a_3 = 50a_3 = 1\). Следовательно \(a_3 = \tfrac{1}{50}\), \(a_2 = -\tfrac{7}{200}\), \(a_1 = \tfrac{7}{100}\).

Итоговое приближённое решение:

\[ \boxed{\;\tilde y(x) = \frac{7}{100}\,x - \frac{7}{200}\,x^2 + \frac{1}{50}\,x^3.\;} \tag{1.6} \]

Сравнение точного решения и кубического приближения по методу коллокаций. Ошибка нарастает к правому концу отрезка, где решение меняется быстрее всего.
Обсуждение.
  1. Простые «оценочные» решения полезны при отладке программ, но требуют аккуратности. В нашем случае узлы коллокации (\(x=0,1\)) выбраны далеко не лучшим образом — они смещены влево, тогда как решение быстрее всего меняется справа, отсюда и рост погрешности.
  2. Можно взять полином более высокой степени, но тогда система растёт, а её матрица становится плотно заполненной. Стоимость метода Гаусса \(\sim O(n^3)\). Кроме того, при больших \(n\) на отрезке длиной \(>1\) приходится возводить в высокие степени как малые (\(\sim 1/n\)), так и большие числа: в матрице появляются одновременно очень малые и очень большие элементы, что резко ухудшает обусловленность.

Метод взвешенных невязок

Метод коллокаций — частный случай общего метода взвешенных невязок. Исходная посылка: искомую функцию можно представить функциональным рядом. Для ОДУ приближённое решение записывают в виде

\[ \tilde y(x) = f_0(x) + \sum_{j=1}^{n} a_j\, f_j(x). \tag{1.7} \]

Функцию \(f_0\) выбирают так, чтобы она по возможности удовлетворяла граничным (начальным) условиям; функции \(f_j\) — пробные (аппроксимирующие), известные и, как правило, гладкие (полиномы, тригонометрические функции). Приведём уравнение к виду \(F(y) = 0\). Подстановка приближения \(\tilde y\) не обращает его в тождество:

\[ F(\tilde y) = R(x), \]

где \(R(x)\) — невязка. Ключевая идея: подобрать коэффициенты \(a_j\) так, чтобы невязка была мала во всей области в интегральном смысле — чтобы интегралы от невязки, умноженной на весовые функции \(W_m\), обращались в ноль:

\[ \int_\Omega W_m(x)\,R(x)\,dx = 0,\qquad m = 1,2,\dots,n. \tag{1.8} \]

Конечное число весовых функций даёт систему уравнений для \(a_j\). Различный выбор пробных и весовых функций порождает целый класс методов.

Метод подобластей

Область делится на подобласти \(D_m\) (возможно перекрывающиеся). Весовая функция — индикатор подобласти:

\[ W_m(x) = \begin{cases} 1, & x \in D_m,\\ 0, & x \notin D_m.\end{cases} \]

Так обеспечивается выполнение уравнения «в среднем» по каждой подобласти.

Метод коллокаций как взвешенная невязка

В методе коллокаций весовыми функциями служат дельта-функции Дирака:

\[ W_m(x) = \delta(x - x_m). \]

Дельта-функцию можно понимать как предел последовательности гладких функций с сужающимися носителями и единичным интегралом. Интегрирование невязки с весом \(\delta(x - x_m)\) в силу фильтрующего свойства даёт

\[ \int_\Omega \delta(x - x_m)\,R(x)\,dx = R(x_m) = 0, \]

то есть требование обращения невязки в ноль в выбранных узлах \(x_m\) — ровно то, что мы делали в первом примере.

Метод наименьших квадратов

В основе — минимизация интеграла от квадрата невязки:

\[ \Phi(a_1,\dots,a_n) = \int_\Omega R^2(x)\,dx \;\to\; \min. \tag{1.9} \]

Это тоже метод взвешенных невязок: весовой функцией является сама невязка \(R(x)\). Интеграл (1.9) — неотрицательная квадратичная форма от коэффициентов \(a_j\). Условие минимума

\[ \frac{\partial \Phi}{\partial a_m} = 2\int_\Omega R(x)\,\frac{\partial R}{\partial a_m}\,dx = 0,\qquad m = 1,\dots,n, \tag{1.10} \]

даёт систему линейных уравнений относительно \(a_j\). Поскольку \(\Phi\) — неотрицательная квадратичная форма, её стационарная точка является минимумом.

Проекционные методы: геометрическая идея

В вычислительной практике гораздо употребительнее методов коллокаций, подобластей и наименьших квадратов оказались проекционные методы. Их суть проще всего понять на задаче наилучшего приближения вектора.

Пусть \(\mathbf a\) — вектор в евклидовом пространстве \(\mathbb R^N\), а \(\mathbb R^M\) (\(M < N\)) — его подпространство, которому \(\mathbf a\) не принадлежит. Найдём наилучшее приближение \(\mathbf b \in \mathbb R^M\), минимизирующее расстояние:

\[ \|\mathbf a - \mathbf b\| = \min_{\mathbf c \in \mathbb R^M} \|\mathbf a - \mathbf c\|. \]

Геометрически решение очевидно: \(\mathbf b\) есть ортогональная проекция \(\mathbf a\) на \(\mathbb R^M\), а разность \(\mathbf a - \mathbf b\) ортогональна всему подпространству:

\[ (\mathbf a - \mathbf b,\, \mathbf c) = 0\qquad \forall\, \mathbf c \in \mathbb R^M. \tag{1.11} \]
Наилучшее приближение вектора \(\mathbf a\) элементом подпространства \(\mathbb R^M\) — это его ортогональная проекция \(\mathbf b\). Невязка \(\mathbf a - \mathbf b\) ортогональна всем базисным векторам \(\mathbf e_i\). Этот принцип ортогональности невязки лежит в основе метода Галёркина.

Раскладывая \(\mathbf b = \sum_{i=1}^{M} b_i \mathbf e_i\) по базису \(\{\mathbf e_i\}\) и подставляя в (1.11), приходим к системе линейных уравнений

\[ \sum_{k=1}^{M} (\mathbf e_k, \mathbf e_i)\,b_k = (\mathbf a, \mathbf e_i),\qquad i = 1,\dots,M, \tag{1.12} \]

матрица которой — матрица Грама системы линейно независимых векторов: она симметрична и невырожденна. Если базис ортогонален, \((\mathbf e_i, \mathbf e_k) = 0\) при \(i \neq k\), матрица диагональна и система решается тривиально.

Метод Галёркина

Перенесём принцип ортогональности невязки на операторное уравнение. В гильбертовом пространстве \(H\) над \(\mathbb R\) со скалярным произведением \((x,y)\) задано

\[ A x = f, \tag{1.13} \]

где \(A\) — линейный оператор, \(f \in H\). Пусть \(H_N \subset D_A\) — конечномерное подпространство. Приближённым решением по Галёркину называют такой \(x_N \in H_N\), для которого невязка \(A x_N - f\) ортогональна всему \(H_N\):

\[ (A x_N - f,\, \psi) = 0\qquad \forall\, \psi \in H_N. \tag{1.14} \]

Пусть \(\psi_1,\dots,\psi_N\) — базис в \(H_N\). Ищем решение в виде \(x_N = \sum_{k=1}^{N} c_k \psi_k\). Подставляя в (1.14) и полагая \(\psi\) последовательно равным \(\psi_1,\dots,\psi_N\), получаем систему для \(c_k\):

\[ \sum_{k=1}^{N} (A\psi_k,\, \psi_i)\,c_k = (f,\, \psi_i),\qquad i = 1,\dots,N. \tag{1.15} \]

Функции \(\psi_i\) называются координатными (или весовыми) функциями проекционного метода. Их ценность в том, что они позволяют сохранить в приближённой задаче важные свойства исходной — симметрию и знакоопределённость оператора.

Из истории. Борис Григорьевич Галёркин (1871–1945) — русский и советский механик и математик, академик. Его проекционный метод получил широкую известность после работ около 1915 года. Аналогичный приём применял корабельный инженер Иван Бубнов (1913), поэтому метод нередко называют методом Бубнова — Галёркина. Впоследствии он был распространён на задачи теплообмена, электрофизики, гидродинамики.

Вариационный метод Ритца

Метод Ритца, как и метод Галёркина, строит решение в виде линейной комбинации известных функций, но опирается на вариационный принцип. Он применим, когда оператор самосопряжён и положительно определён.

Теорема (вариационный принцип). Пусть в гильбертовом пространстве \(H\) задано уравнение \(A u = f\), причём оператор \(A\) самосопряжён и положительно определён: \[ (Av, w) = (v, Aw)\ \ \forall v,w;\qquad (Av,v) \ge \gamma\,\|v\|^2 > 0\ \ \forall v \neq 0. \] Тогда решение \(u\) уравнения \(Au = f\) доставляет минимум функционалу энергии \[ \Phi(v) = (Av, v) - 2(f, v),\qquad u = \arg\min_{v \in H} \Phi(v). \]
Положим \(v = u + \eta\), где \(\eta\) — произвольное отклонение от решения, и используем \(Au - f = 0\): \[ \Phi(v) = (A(u+\eta), u+\eta) - 2(f, u+\eta). \] Раскрывая и группируя с учётом самосопряжённости \((A\eta, u) = (\eta, Au) = (Au, \eta)\): \[ \Phi(v) = \underbrace{(Au,u) - 2(f,u)}_{\Phi(u)} + 2\,(Au - f,\, \eta) + (A\eta, \eta). \] Средний член равен нулю, а \((A\eta, \eta) \ge 0\) в силу положительной определённости. Поэтому \(\Phi(v) \ge \Phi(u)\), причём равенство только при \(\eta = 0\). Минимум достигается на решении и единствен.

Приближённо ищем минимум на подпространстве \(H_N\): \(x_N = \sum_{k=1}^{N} c_k \psi_k\). Подстановка в \(\Phi\) превращает его в квадратичную форму от \(c_k\):

\[ \Phi(x_N) = \sum_{k=1}^{N}\sum_{m=1}^{N} c_k c_m\,(A\psi_k, \psi_m) - 2\sum_{k=1}^{N} c_k\,(f, \psi_k). \]

Условие минимума \(\partial \Phi/\partial c_i = 0\) с учётом самосопряжённости приводит ровно к той же системе, что и у Галёркина:

\[ \sum_{k=1}^{N} (A\psi_k, \psi_i)\,c_k = (f, \psi_i),\qquad i = 1,\dots,N. \tag{1.16} \]
Из истории. Вальтер Ритц (1878–1909) — швейцарский физик-теоретик и математик. В 1909 году он предложил метод, широко применяемый к вариационным задачам. Совпадение систем (1.15) и (1.16) для самосопряжённого оператора объясняет, почему МКЭ часто называют методом Рэлея — Ритца: минимизация энергии и проекция невязки в этом случае эквивалентны.

Переход к конечным элементам

У классических проекционных методов есть серьёзный практический изъян. Когда в качестве координатных функций берут «глобальные» полиномы или тригонометрические функции высокого порядка, возникающие системы линейных уравнений становятся плохо обусловленными: ошибки округления неприемлемо искажают решение. Это долго затрудняло применение проекционных методов на ЭВМ.

Выход подсказывают сами рассмотренные примеры: вместо полиномов высокой степени использовать полиномы невысокой степени, заданные кусочно — например, кусочно-линейные функции, линейные на каждом интервале разбиения

\[ x_0 < x_1 < x_2 < \dots < x_n. \]

От этого отправного пункта мы и подходим к методу конечных элементов.

Из истории МКЭ. Метод возник при поиске новых путей решения задач строительной механики и теории упругости в 1930-х годах. У истоков идей — Александр Хренников и Рихард Курант (работы 1940-х). Эффективность метода на ЭВМ впервые продемонстрировал Иоаннис Аргирис (1944). В Китае Фэн Кан в 1950-х независимо предложил «метод конечных разностей на основе вариационного принципа». Важнейший толчок дал 1963 год, когда МКЭ был осознан как вариант метода Рэлея — Ритца, сводящего задачу к минимизации потенциальной энергии; после этого метод стал применяться к уравнениям Лапласа и Пуассона. В СССР практическая реализация МКЭ в 1960-х связана с именем Леонарда Оганесяна.

Поясним сеточный вариант метода Галёркина, который одновременно служит примером конечно-элементного подхода. От классического Галёркина его отличают непрерывные финитные пробные функции — равные нулю всюду, кроме малой подобласти, причём каждая равна единице в «своём» узле и нулю во всех остальных узлах. Тогда коэффициенты разложения приобретают прямой физический смысл — это значения искомой функции в узлах:

\[ \tilde u(x) \approx \sum_{j=1}^{N} f_j\,\varphi_j(x), \tag{1.17} \]

где \(f_j\) — приближённые узловые значения, а \(\varphi_j(x)\) — пробные функции, называемые также интерполяционными функциями или функциями формы.

Функции формы: кусочно-линейные и кусочно-квадратичные

В одномерном случае кусочно-линейная пробная функция узла \(j\) имеет вид «шапочки» (hat-функции):

\[ \varphi_j(x) = \begin{cases} 0, & x \le x_{j-1}, \\[4pt] \dfrac{x - x_{j-1}}{x_j - x_{j-1}}, & x_{j-1} \le x \le x_j, \\[8pt] \dfrac{x_{j+1} - x}{x_{j+1} - x_j}, & x_j \le x \le x_{j+1}, \\[8pt] 0, & x \ge x_{j+1}. \end{cases} \tag{1.18} \]

Каждая такая функция равна 1 в узле \(x_j\), линейно спадает до 0 в соседних узлах \(x_{j-1}, x_{j+1}\) и равна нулю вне отрезка \([x_{j-1}, x_{j+1}]\). Между узлами решение (1.17) при этом интерполируется линейно.

Рис. 2. Кусочно-линейные пробные функции («шапочки»). Функция \(\varphi_j\) равна 1 в узле \(x_j\), линейно спадает до 0 в соседних узлах и финитна. Соседние функции перекрываются только на смежных элементах — поэтому матрица системы будет ленточной (разреженной).

Применяют и интерполяцию более высокого порядка. Для кусочно-квадратичной на отрезке \([a,b]\) вводят равномерную сетку с шагом \(h\) и различают чётные и нечётные узлы. В чётном узле \(x_{2i}\) (общем для двух соседних элементов) функция формы записывается через произведение линейных множителей:

\[ \varphi_{2i}(x) = \Bigl(1 + \tfrac{3}{2}\tfrac{x - x_{2i}}{h}\Bigr)\Bigl(1 - \tfrac{3}{2}\tfrac{x - x_{2i}}{h}\Bigr) \quad\text{на } |x - x_{2i}| \le 2h, \]

а в нечётном (внутреннем) узле \(x_{2i+1}\) — «горб», обращающийся в ноль в концах своего элемента:

\[ \varphi_{2i+1}(x) = 1 - \Bigl(\frac{x - x_{2i+1}}{h}\Bigr)^2,\qquad |x - x_{2i+1}| \le h, \]

и нулю вне указанных интервалов.

Рис. 3. Кусочно-квадратичные пробные функции. В нечётных (внутренних) узлах — «горб», обнуляющийся на концах элемента (зелёный); в чётных (стыковых) узлах — функция, охватывающая два соседних элемента (терракота). Квадратичная интерполяция повышает порядок аппроксимации, но заметно усложняет вычисления, особенно в 2D и 3D.
Главная мысль. Переход от глобальных полиномов к финитным функциям формы превращает плотную плохо обусловленную систему в разреженную и хорошо обусловленную: каждая функция \(\varphi_j\) перекрывается лишь с несколькими соседями, поэтому в строке матрицы Галёркина-Ритца почти все элементы нулевые. Именно это сделало МКЭ практичным методом — и это связующее звено между проекционно-вариационной теорией и реальными расчётами.
