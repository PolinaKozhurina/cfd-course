Временно́е интегрирование: Leap-Frog и адаптивный шаг

Зачем нужен симплектичный интегратор

В SPH мы получаем систему ОДУ вида

\[ \frac{d\mathbf r_i}{dt} = \mathbf v_i, \qquad \frac{d\mathbf v_i}{dt} = \mathbf a_i(\mathbf r_1,\ldots,\mathbf r_N;\,\mathbf v_1,\ldots,\mathbf v_N), \tag{3.1} \]

где ускорение \(\mathbf a_i\) зависит от текущих положений (через \(\rho_i, P_i\)) и скоростей (в случае искусственной вязкости, см. лекцию 4). Без вязкости правая часть не зависит от скоростей — это каноническая гамильтонова система со специальной структурой, которую нужно сохранить численно.

Определение (симплектичный интегратор). Численная схема \((r^n, v^n) \to (r^{n+1}, v^{n+1})\) называется симплектичной, если она сохраняет симплектическую форму \(\omega = dr \wedge dv\) на каждом шаге: якобиан преобразования \((r,v)\to(r',v')\) по модулю равен 1 в фазовом пространстве.

Почему это важно? Обычный явный Эйлер — не симплектичен, и при длительном интегрировании гармонического осциллятора даёт секулярный рост полной энергии: амплитуда колебаний монотонно растёт. Для SPH-задач это катастрофа — расчёт «разъезжается» даже без возмущений. Симплектические интеграторы, наоборот, осциллируют энергию вокруг правильного значения, без систематического дрейфа.

Иллюстрация. В гармоническом осцилляторе \(\ddot x = -\omega^2 x\): явный Эйлер даёт \(E^{n+1} = E^n(1 + \omega^2\Delta t^2)\) — экспоненциальный рост. Leap-Frog даёт \(E^{n+1} = E^n(1 + O(\Delta t^4))\) — небольшие осцилляции без тренда.

Схема Leap-Frog («чехарда»)

Простейший симплектичный интегратор 2-го порядка. Скорости вычисляются на полуцелых узлах по времени, координаты — на целых:

\[ \boxed{\; \mathbf v_i^{n+1/2} = \mathbf v_i^{n-1/2} + \Delta t \cdot \mathbf a_i^n, \qquad \mathbf r_i^{n+1} = \mathbf r_i^n + \Delta t \cdot \mathbf v_i^{n+1/2}. \;} \tag{3.2} \]

Схема называется «чехардой» (leap-frog), потому что скорости «перепрыгивают» через координаты: \(v^{1/2} \to r^1 \to v^{3/2} \to r^2\ldots\) Получаем явную одношаговую схему.

Схема «чехарда»: скорости (синие кружки) и координаты (красные) расположены в сдвинутых узлах времени. Синяя дуга через верх — обновление скорости с использованием \(\mathbf a^n\) (ускорение в координатной точке); красная дуга через низ — обновление координаты с помощью \(\mathbf v^{n+1/2}\) (скорость в полуцелой точке). «Перепрыгивания» друг через друга и дают методу название leap-frog.

Свойства Leap-Frog

Теорема 6 (свойства Leap-Frog). Схема (3.2) для системы (3.1) с ускорением, не зависящим от скоростей, обладает следующими свойствами:
  1. Второй порядок точности по \(\Delta t\): локальная ошибка \(O(\Delta t^3)\), глобальная \(O(\Delta t^2)\).
  2. Симплектичность: преобразование фазового пространства сохраняет меру \(dr\wedge dv\).
  3. Обратимость во времени: замена \(\Delta t \to -\Delta t\) приводит ровно к тому же решению, прокрученному назад.
  4. Нет секулярного дрейфа энергии: для гамильтоновых систем полная энергия колеблется вокруг точного значения с амплитудой \(O(\Delta t^2)\) на всех временах.

Доказательство 2-го порядка. Запишем точное разложение \(r(t + \Delta t)\) и \(v(t + \Delta t/2)\) в ряд Тейлора: \[ r(t+\Delta t) = r(t) + \Delta t\,v(t) + \tfrac{1}{2}\Delta t^2\,a(t) + \tfrac{1}{6}\Delta t^3\,\dddot r(t) + O(\Delta t^4). \] В Leap-Frog: \(r^{n+1} = r^n + \Delta t\,v^{n+1/2}\), где \(v^{n+1/2} = v(t^{n+1/2}) = v(t^n) + \tfrac{1}{2}\Delta t\,a(t^n) + O(\Delta t^2)\). Подставляя: \[ r^{n+1} = r^n + \Delta t\,v^n + \tfrac{1}{2}\Delta t^2\,a^n + O(\Delta t^3). \] Сравнение с точным: совпадает до третьего порядка, локальная ошибка \(O(\Delta t^3)\), что при \(N \sim 1/\Delta t\) шагах даёт глобальную ошибку \(O(\Delta t^2)\). \(\square\)

Доказательство симплектичности (2-го свойства). Leap-Frog можно записать как композицию двух канонических преобразований: «kick» (обновление скорости при фиксированной координате — \(v \to v + \Delta t\,a(r)\)) и «drift» (обновление координаты при фиксированной скорости — \(r \to r + \Delta t\,v\)). Каждое из них имеет якобиан \(\det \partial(r',v')/\partial(r,v) = 1\) (треугольная матрица с единицами на диагонали), и произведение двух преобразований с якобианом 1 также имеет якобиан 1 — это и есть сохранение симплектической меры. \(\square\)

Сравнение Эйлер vs Leap-Frog на гармоническом осцилляторе

Классический тест — \(\ddot x = -\omega^2 x\) (частица на пружине). Точное решение: \(x(t) = A\cos(\omega t)\), полная энергия \(E = \tfrac{1}{2}(v^2 + \omega^2 x^2)\) сохраняется.

Leap-Frog имеет ту же вычислительную стоимость за шаг, что и явный Эйлер, но кардинально лучшие статистические свойства на больших временах. Это объясняет его популярность в астрофизике, молекулярной динамике и SPH.

Инициализация полушагом

Схема (3.2) требует начального условия \(\mathbf v^{1/2}\), а физически нам дано \(\mathbf v^0\). Стандартный приём — один полушаг Эйлера:

\[ \mathbf v_i^{1/2} = \mathbf v_i^0 + \tfrac{1}{2}\Delta t^0\cdot\mathbf a_i^0. \tag{3.3} \]

Здесь \(\Delta t^0\) — начальный шаг по времени (вычисляется по CFL на данных \(t=0\)). После этого полушага стандартный цикл (3.2) уже применим со шага \(n = 0\).

Адаптивный шаг по времени: условие Куранта–Фридрихса–Леви

Для гиперболических уравнений (Эйлера) шаг по времени не может быть слишком большим — иначе возмущение проходит более одной расчётной ячейки за шаг, и схема теряет устойчивость. В SPH ячейка — это область влияния ядра размером \(\sim h\), поэтому:

\[ \Delta t \leq \frac{C_0\, h}{\max_i(c_i + |\mathbf v_i|)},\qquad C_0 = 0.3. \tag{3.4} \]

В числителе — длина сглаживания \(h\), в знаменателе — максимальная скорость распространения информации (звук плюс перенос). Коэффициент \(C_0 \in (0, 1]\) — число Куранта. Стандартный выбор \(C_0 = 0.3\) для SPH (в отличие от \(C_0 \sim 0.9\) для явных сеточных схем 1-го порядка) обусловлен тем, что вязкость (лекция 4) вносит дополнительные ограничения на шаг.

Адаптивность. По мере эволюции решения скорости и скорости звука меняются (в ударной волне \(c\) растёт), поэтому \(\Delta t\) нужно пересчитывать на каждом шаге. На практике: после расчёта ускорений на шаге \(n\) пересчитываем \(c_i\), \(v_i\), находим максимум и определяем \(\Delta t^{n+1}\). Это обеспечивает автоматическое сгущение временной сетки в моменты сильных возмущений.

Полный алгоритм одного шага по времени

Алгоритм SPH-шага (без вязкости):
  1. По текущим координатам \(\{\mathbf r_i^n\}\) вычислить плотности \(\{\rho_i^n\}\) по (2.2).
  2. По \(\{\rho_i^n\}\) вычислить давления \(\{P_i^n\}\) и скорости звука \(\{c_i^n\}\) по (2.5).
  3. Вычислить ускорения \(\{\mathbf a_i^n\}\) по (2.4).
  4. Определить шаг \(\Delta t^n\) по (3.4) с использованием \(\max(c_i + |v_i|)\).
  5. Если \(t + \Delta t^n > t_{\text{end}}\), уменьшить: \(\Delta t^n := t_{\text{end}} - t\).
  6. Обновить скорости: \(\mathbf v_i^{n+1/2} = \mathbf v_i^{n-1/2} + \Delta t^n\,\mathbf a_i^n\).
  7. Обновить координаты: \(\mathbf r_i^{n+1} = \mathbf r_i^n + \Delta t^n\,\mathbf v_i^{n+1/2}\).
  8. \(t := t + \Delta t^n\); если \(t < t_{\text{end}}\) — повторить с шага 1.

Особенность: для инициализации (до первого цикла) выполняется полушаг (3.3) с начальным \(\Delta t^0\), после чего у нас есть \(\mathbf v_i^{1/2}\) для начала основного цикла.

Ожидаемый результат на задаче Сода

После \(\sim\)300–500 шагов по времени (адаптивный \(\Delta t\), \(t_{\text{end}} = 0.2\)) мы должны увидеть решение задачи Сода с тремя характерными волнами:

Ожидаемый профиль плотности для теста Сода при \(t = 0.2\): левый плато (\(\rho = 1\)) → волна разрежения (ВР) → промежуточный плато (\(\rho^* = 0.4263\)) → контактный разрыв (КР) → правый промежуточный плато (\(\rho = 0.2656\)) → ударная волна (УВ) → правый плато (\(\rho = 0.125\)).

Но — внимание — без искусственной вязкости в численном решении SPH на ударной волне возникнут нефизические осцилляции давления. Это фундаментальная особенность SPH: ядро «размазывает» ударный фронт на расстояние \(\sim h\), и если физическая ширина фронта (нулевая для идеального газа) меньше этого расстояния, начинается явление типа Гиббса — так же как при приближении разрывной функции конечным числом гармоник Фурье. Устранение этих осцилляций — тема следующей лекции.

Что тестировать в лаб 3. (а) Качественная картина — три волны на своих местах. (б) Сохранение массы: \(M^n = \sum_i m_i = \mathrm{const}\) точно, потому что массы не меняются. (в) Сохранение импульса: \(P^n = \sum_i m_i v_i^n\) сохраняется до машинного эпсилона благодаря симметричной форме (2.4). (г) Осцилляции давления возле ударного фронта (\(x \approx 0.85\)) — ожидаемая «патология» без вязкости.
