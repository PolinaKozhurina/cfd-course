§16 · Семестр 1

Точная задача Римана для одномерных законов сохранения

Постановка задачи Римана

Рассмотрим одномерную гиперболическую систему законов сохранения

\[ \frac{\partial\mathbf{u}}{\partial t} + \frac{\partial\mathbf{f}(\mathbf{u})}{\partial x} = 0, \tag{16.1} \]

и задачу Коши с кусочно-постоянными начальными данными с одним разрывом

\[ \mathbf{u}(x,0) = \begin{cases} \mathbf{u}_L, & x < 0, \\ \mathbf{u}_R, & x > 0, \end{cases} \tag{16.2} \]

где \(\mathbf{u}_L, \mathbf{u}_R\) — постоянные состояния. Такую задачу называют задачей Римана.

Автомодельность решения

Поскольку постановка (16.2) не содержит характерного масштаба по \(x\), решение ищется в автомодельном виде

\[ \mathbf{u}(x,t) = \mathbf{U}(\xi), \qquad \xi = \frac{x}{t}, \quad t > 0. \]

На диаграмме \(x\)–\(t\) это означает, что линии разрывов являются прямыми \(x = St\), а непрерывные волны занимают секторы \(S_1 t \le x \le S_2 t\). В результате плоскость \((x,t)\) распадается на области с постоянными состояниями, разделённые элементарными волнами. Для уравнений Эйлера типичная структура решения включает левую волну, контактный разрыв и правую волну, между которыми расположены два промежуточных состояния \(\mathbf{u}_{*L}\), \(\mathbf{u}_{*R}\).

Диаграмма \(x\)–\(t\) задачи Римана: левая волна (разрежение), контактный разрыв, правая волна (удар). Четыре области постоянных состояний.

Элементарные волны для уравнений Эйлера

Рассмотрим одномерные уравнения Эйлера в переменных \((\rho, v, p)\). Скорость звука обозначим \(a = \sqrt{\gamma p/\rho}\). Тогда три характеристические скорости равны

\[ \lambda_1 = v - a, \qquad \lambda_2 = v, \qquad \lambda_3 = v + a. \]

Им соответствуют три семейства волн: левая акустическая (1), контактная (2) и правая акустическая (3).

Для газовой динамики в них выполняются условия контакта

\[ v_{*L} = v_{*R} = v_*, \qquad p_{*L} = p_{*R} = p_*, \tag{16.3} \]

а плотности \(\rho_{*L}, \rho_{*R}\) вообще говоря различны.

Контактный разрыв

Контактный разрыв распространяется со скоростью

\[ S_C = v_*, \tag{16.4} \]

и на нём сохраняются давление и скорость, см. (16.3). На диаграмме \(x\)–\(t\) контакт — прямая \(x = v_* t\), разделяющая состояния \(\mathbf{u}_{*L}\) и \(\mathbf{u}_{*R}\).

Ударная волна и соотношения Ранкина–Гюгонио

Пусть через разрыв со скоростью \(S\) переходят состояния \((\rho_L, v_L, p_L)\) и \((\rho_R, v_R, p_R)\). Соотношения Ранкина–Гюгонио для законов сохранения массы и импульса дают

\[ S(\rho_R - \rho_L) = \rho_R v_R - \rho_L v_L, \tag{16.5} \]
\[ S(\rho_R v_R - \rho_L v_L) = \rho_R v_R^2 + p_R - (\rho_L v_L^2 + p_L). \tag{16.6} \]

Удобно ввести массовый поток через разрыв \(m = \rho_L(v_L - S) = \rho_R(v_R - S)\). Тогда из (16.6) следует

\[ p_R - p_L = m(v_L - v_R). \tag{16.7} \]

Волна разрежения

Волна разрежения представляет собой непрерывную автомодельную волну, в которой решение зависит только от \(\xi = x/t\). Для левой волны разрежения границы веера определяются характеристическими скоростями:

\[ S_H = v_L - a_L \quad\text{(голова)}, \qquad S_T = v_* - a_* \quad\text{(хвост)}. \tag{16.11} \]

Для правой волны разрежения:

\[ S_H = v_* + a_* \quad\text{(голова)}, \qquad S_T = v_R + a_R \quad\text{(хвост)}. \tag{16.12} \]

Таким образом, область разрежения на диаграмме \(x\)–\(t\) имеет вид клина, внутри которого решение зависит от \(\xi = x/t\).

Структура точного решения

Пусть левая волна является 1-волной, правая — 3-волной, а между ними находится контакт. Тогда диаграмма \(x\)–\(t\) имеет следующий общий вид:

  • область \(x < S_L t\) занята левым состоянием \(\mathbf{u}_L\);
  • область \(S_L t < x < S_C t\) занята левым промежуточным состоянием \(\mathbf{u}_{*L}\);
  • область \(S_C t < x < S_R t\) занята правым промежуточным состоянием \(\mathbf{u}_{*R}\);
  • область \(x > S_R t\) занята правым состоянием \(\mathbf{u}_R\).

Здесь \(S_C = v_*\) — скорость контакта, а \(S_L\) и \(S_R\) зависят от типа волн. Именно поэтому на типовых диаграммах решения задачи Римана наблюдаются четыре комбинации: удар–удар, разрежение–удар, удар–разрежение, разрежение–разрежение.

Для полного построения точного решения необходимо определить параметры промежуточного состояния \(p_*, v_*\) и типы левой и правой волн. Эти шаги составляют содержание точного решателя задачи Римана, который будет рассмотрен в следующем параграфе.

Интерактив. Все четыре типа волновых конфигураций (УВ-УВ, УВ-ВР, ВР-УВ, ВР-ВР) можно посмотреть в анализаторе волновой структуры в §17. Задавайте состояния слева и справа, наблюдайте (x,t)-диаграмму и анимированный профиль во времени.
