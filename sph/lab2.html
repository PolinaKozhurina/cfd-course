Лабораторная 2 · Мини-курс SPH

Реализация правых частей уравнений (без интегрирования по времени)

Цель: реализовать на Maple все три процедуры, составляющие одну итерацию SPH: compute_density(), compute_accel(), вычисление давления/скорости звука. Запустить на начальных данных теста Сода и проверить, что правые части уравнений физически осмыслены.

К занятию освежите: формулу плотности (2.2), симметричную форму ускорения (2.4), уравнение состояния (2.5).

Задание 2.1. Инициализация: тест Сода

ЗАДАНИЕ 2.1

Создать массив частиц с начальными условиями Сода

Написать код, создающий \(N = 400\) частиц, равномерно расположенных на \([0, 1]\). Начальные данные — задача Римана Сода:

Область \(\rho\) \(P\) \(u\)
\(x < 0.5\) (левая)110
\(x \geq 0.5\) (правая)0.1250.10

Частицы расположены равномерно: \(x_j = (j-0.5)\Delta x\), \(\Delta x = 1/N = 0.0025\). Массы частиц:

  • \(m_j = \rho_j\cdot\Delta x\), то есть \(m_j = 0.0025\) при \(x_j < 0.5\) и \(m_j = 0.0003125\) при \(x_j \geq 0.5\).

Начальные скорости \(v_j = 0\) для всех частиц. Параметры: \(\gamma = 1.4\), \(k = P_L/\rho_L^\gamma = 1\), \(h = 2\Delta x = 0.005\).

# --- Задание 2.1: Инициализация теста Сода ---
restart;
with(plots):

# Параметры
N := 400;
dx := 1/N;
h := 2*dx;
gamma_g := 1.4;
k_eos := 1.0;  # = P_L / rho_L^gamma при rho_L=1, P_L=1

# Начальные условия Сода
x := Array(1..N):
rho := Array(1..N):
v := Array(1..N):
P := Array(1..N):
m := Array(1..N):

for j from 1 to N do
  x[j] := (j - 0.5) * dx;
  v[j] := 0;
  if x[j] < 0.5 then
    rho[j] := 1.0;
    P[j]   := 1.0;
  else
    rho[j] := 0.125;
    P[j]   := 0.1;
  end if;
  m[j] := rho[j] * dx;  # масса фиксируется в начальный момент и не меняется!
end do:

# Контрольная проверка
printf("m[левая]  = %.4f (ожид. 0.0025)\n", m[1]);
printf("m[правая] = %.6f (ожид. 0.0003125)\n", m[N]);

# Процедуры для ядра (копии из лаб 1)
sigma1D := 2/3;
W_kernel := proc(q) local qq;
  qq := abs(q);
  if qq >= 2 then return 0
  elif qq >= 1 then return sigma1D*(2-qq)^3/4
  else return sigma1D*(1 - 3*qq^2/2 + 3*qq^3/4)
  end if;
end proc:

dW_kernel := proc(q) local qq, sgn;
  qq := abs(q); sgn := sign(q);
  if qq >= 2 then return 0
  elif qq >= 1 then return -sigma1D*3*(2-qq)^2/4 * sgn
  else return sigma1D*(-3*qq + 9*qq^2/4) * sgn
  end if;
end proc:

Задание 2.2. Процедура compute_density

ЗАДАНИЕ 2.2

Вычисление плотности суммированием по соседям

Написать процедуру compute_density(), которая для каждой частицы \(i\) вычисляет \(\rho_i\) по формуле (2.2) двойным циклом по \(i\) и \(j\).

Построить график \(\rho(x)\) при \(t=0\). Проверить:

  • \(\rho_i \approx 1\) при \(x_i < 0.4\),
  • \(\rho_i \approx 0.125\) при \(x_i > 0.6\),
  • Монотонный переход в окрестности \(x = 0.5\) шириной \(\sim 2h\) (т.е. около 10 частиц при \(h = 2\Delta x\)).
# --- Задание 2.2: compute_density ---
compute_density := proc()
  local i, j, s, qij;
  for i from 1 to N do
    s := 0;
    for j from 1 to N do
      qij := abs(x[i] - x[j]) / h;
      if qij < 2 then
        s := s + m[j] * W_kernel(qij) / h;
      end if;
    end do;
    rho[i] := s;
  end do;
end proc:

compute_density();

# Контрольные значения
printf("rho[x=0.2]  = %.4f (ожид. ~1.0)\n", rho[80]);
printf("rho[x=0.5]  = %.4f (переход)\n", rho[200]);
printf("rho[x=0.8]  = %.4f (ожид. ~0.125)\n", rho[320]);

# График
p_rho0 := plot([seq([x[i], rho[i]], i=1..N)], style=line,
               color=blue, thickness=1.5,
               labels=["x","ρ"], title="Плотность при t=0 (Sod)"):
display(p_rho0);
Что должно быть видно на графике. Слева плато \(\rho \approx 1\), справа плато \(\rho \approx 0.125\), плавный переход на ширине \(\sim 4h = 8\Delta x\) (два-три радиуса носителя ядра). Если переход узкий (как ступенька) — ошибка в вычислении \(q_{ij}\) или \(h\). Если плотность на плато не 1 — ошибка в нормировке ядра (забыли делить на \(h\)).

Задание 2.3. Процедура compute_accel

ЗАДАНИЕ 2.3

Вычисление ускорения (симметричная форма)

Написать процедуру compute_accel(), которая для каждой частицы \(i\) вычисляет ускорение \(a_i\) по формуле (2.4) двойным циклом. Использовать \(P_i\) и \(\rho_i\) из предыдущего задания.

Построить \(a(x)\) при \(t = 0\). Убедиться:

  • \(a_i \approx 0\) вдали от мембраны \(x = 0.5\),
  • \(|a_i|\) максимально при \(x \approx 0.5\),
  • Знак определяется градиентом давления: газ ускоряется из области высокого \(P\) в область низкого \(P\) (вправо, положительное \(a\)).
# --- Задание 2.3: compute_accel ---
a := Array(1..N):

# Сначала пересчитаем давления
for i from 1 to N do
  P[i] := k_eos * rho[i]^gamma_g;
end do:

compute_accel := proc()
  local i, j, sum_a, dxij, qij, dWij;
  for i from 1 to N do
    sum_a := 0;
    for j from 1 to N do
      if i <> j then
        dxij := x[i] - x[j];
        qij := abs(dxij) / h;
        if qij < 2 then
          dWij := dW_kernel(qij) * sign(dxij) / h^2;  # см. формулу (1.11)
          sum_a := sum_a - m[j] * (P[i]/rho[i]^2 + P[j]/rho[j]^2) * dWij;
        end if;
      end if;
    end do;
    a[i] := sum_a;
  end do;
end proc:

compute_accel();

# Контрольные значения
printf("a[x=0.2]  = %.4e (ожид. ~0)\n", a[80]);
printf("a[x=0.5]  = %.4e (максимум по модулю)\n", a[200]);
printf("a[x=0.8]  = %.4e (ожид. ~0)\n", a[320]);

p_a := plot([seq([x[i], a[i]], i=1..N)], style=line, color=red, thickness=1.5,
            labels=["x","a"], title="Ускорение при t=0 (Sod)"):
display(p_a);
Интерпретация результата. Ускорение положительное (вправо) в переходной области, потому что левая часть имеет более высокое давление и «толкает» вправо. Максимум \(|a|\) должен быть в районе сотен — это соответствует интенсивности начального давления скачка \(\Delta P = 0.9\) на ширине \(\sim 2h = 0.01\), то есть \(a \sim \Delta P/\rho\cdot(1/2h) \sim 0.9/1 \cdot 100 = 90\). Если ваш максимум на 3 порядка меньше — забыли поделить на \(h^2\) в формуле градиента.

Задание 2.4. Давление и скорость звука

ЗАДАНИЕ 2.4

Процедура compute_eos

Написать процедуру, вычисляющую для каждой частицы давление \(P_i\) и скорость звука \(c_i\) по формулам (2.5):

\[ P_i = k\,\rho_i^\gamma, \quad c_i = \sqrt{\gamma P_i/\rho_i}. \]

Построить два графика: \(P(x)\) и \(c(x)\) при \(t = 0\).

# --- Задание 2.4: compute_eos ---
c_sound := Array(1..N):

compute_eos := proc()
  local i;
  for i from 1 to N do
    P[i] := k_eos * rho[i]^gamma_g;
    c_sound[i] := sqrt(gamma_g * P[i] / rho[i]);
  end do;
end proc:

compute_eos();

p_P := plot([seq([x[i], P[i]], i=1..N)], style=line, color="DarkGreen",
            thickness=1.5, labels=["x","P"], title="Давление при t=0"):
p_c := plot([seq([x[i], c_sound[i]], i=1..N)], style=line, color="DarkOrange",
            thickness=1.5, labels=["x","c"], title="Скорость звука при t=0"):

display(Array([p_P, p_c]));  # два графика рядом

printf("c_L = %.4f (ожид. sqrt(1.4)=%.4f)\n", c_sound[80], sqrt(1.4));
printf("c_R = %.4f (ожид. sqrt(1.4·0.1/0.125)=%.4f)\n", c_sound[320], sqrt(1.4*0.1/0.125));
Ожидаемые значения. \(c_L = \sqrt{\gamma P_L/\rho_L} = \sqrt{1.4} \approx 1.183\). \(c_R = \sqrt{\gamma P_R/\rho_R} = \sqrt{1.4\cdot 0.1/0.125} = \sqrt{1.12} \approx 1.058\). Эти значения понадобятся в лекции 3 для вычисления CFL-шага.
