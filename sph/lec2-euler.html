SPH-формулировка уравнений газовой динамики

Уравнения Эйлера для идеального газа в лагранжевой форме

Вспомним уравнения Эйлера в лагранжевой форме, где производная \(d/dt\) берётся вдоль траектории жидкой частицы (материальная производная):

\[ \begin{cases} \dfrac{d\rho}{dt} = -\rho\,\mathrm{div}\,\mathbf v,\\[6pt] \dfrac{d\mathbf v}{dt} = -\dfrac{1}{\rho}\nabla P,\\[6pt] \dfrac{de}{dt} = -\dfrac{P}{\rho}\,\mathrm{div}\,\mathbf v. \end{cases} \tag{2.1} \]

Здесь \(\rho\) — плотность, \(\mathbf v\) — скорость, \(P\) — давление, \(e\) — удельная внутренняя энергия. Первое уравнение — сохранение массы в лагранжевой форме; второе — второй закон Ньютона для единичного объёма; третье — баланс энергии. Для идеального газа замыкание — уравнение состояния \(P = P(\rho, e)\).

В SPH мы переходим к частицам \(\{i\}\) с координатами \(\mathbf r_i(t)\) и скоростями \(\mathbf v_i(t) = d\mathbf r_i/dt\). Наша задача — записать дискретный аналог каждого уравнения через суммы по соседям.

Плотность: суммирование по соседям

Применим основную SPH-формулу (1.4) к плотности \(\rho\):

\[ \rho_i = \langle\rho\rangle_i = \sum_j \frac{m_j}{\rho_j}\,\rho_j\,W_{ij} = \sum_j m_j\,W_{ij}. \tag{2.2} \]

Множители \(\rho_j\) сократились! Получаем элегантную формулу: плотность частицы есть сумма масс соседей, взвешенных ядром. Это и есть ключевое свойство SPH — плотность вычисляется без уравнения непрерывности.

Теорема 4 (свойства формулы (2.2)).
  1. \(\rho_i > 0\) всегда (сумма положительных слагаемых — следствие неотрицательности ядра).
  2. Суммирование (2.2) автоматически сохраняет полную массу: \(\int\rho\,dV \approx \sum_i\rho_i\cdot(m_i/\rho_i) = \sum_i m_i = \mathrm{const}\), так как массы частиц не меняются со временем.
  3. При плотном сгущении частиц в одну точку \(\rho_i\to\infty\) правильно (единица массы в нулевом объёме).
Альтернатива. Существует вариант SPH, в котором плотность эволюционирует как отдельная переменная через уравнение непрерывности: \[ \frac{d\rho_i}{dt} = \sum_j m_j (\mathbf v_i - \mathbf v_j)\cdot\nabla_i W_{ij}. \] Это «эволюционная» формулировка, иногда более точная вблизи границ (нет проблем с «неполными шаблонами»). В нашем курсе используем только суммирование (2.2) как более простую.

Уравнение импульса: наивная форма

Применим SPH-градиент (1.6) к \(\nabla P\) и подставим в уравнение импульса из (2.1):

\[ \frac{d\mathbf v_i}{dt} = -\frac{1}{\rho_i}\sum_j \frac{m_j}{\rho_j}\,P_j\,\nabla_i W_{ij}. \tag{2.3} \]

Это наивная дискретизация. Она даёт правильный результат в непрерывном пределе, но имеет два недостатка:

  1. Не симметрична по \(i\) и \(j\) — частица \(j\) воздействует на \(i\) не так, как \(i\) на \(j\).
  2. Не сохраняет полный импульс на дискретной сетке (точная симметрия \(\mathbf F_{ij} = -\mathbf F_{ji}\) нарушена).

Посмотрим геометрически, как именно взаимодействуют две частицы в форме (2.3):

Сравнение двух форм. Слева: наивная (2.3) — силы между парой частиц не равны по модулю, потому что в формуле фигурируют \(\rho_i\) слева от суммы и \(\rho_j\) в знаменателе внутри. Справа: симметричная (2.4) — силы равны по модулю и противоположно направлены, что обеспечивает сохранение импульса.

Вывод симметричной формы (2.4)

Ключ к симметризации — алгебраическое тождество. Запишем обычную производную дроби \(P/\rho\):

\[ \nabla\!\left(\frac{P}{\rho}\right) = \frac{\nabla P}{\rho} - \frac{P}{\rho^2}\nabla\rho. \]

Отсюда перегруппируем слагаемое \(\nabla P/\rho\) (то, что нам нужно для уравнения импульса):

\[ \frac{\nabla P}{\rho} = \nabla\!\left(\frac{P}{\rho}\right) + \frac{P}{\rho^2}\nabla\rho. \tag{2.4a} \]

Вместо того чтобы дискретизировать \(\nabla P/\rho\) «в лоб» (что даёт форму (2.3)), применим SPH-градиент (1.6) к каждому из двух слагаемых справа отдельно:

Первое слагаемое. Подставляем \(f = P/\rho\) в (1.6):

\[ \left[\nabla\!\left(\frac{P}{\rho}\right)\right]_i = \sum_j \frac{m_j}{\rho_j}\,\frac{P_j}{\rho_j}\,\nabla_i W_{ij} = \sum_j \frac{m_j P_j}{\rho_j^2}\,\nabla_i W_{ij}. \]

Второе слагаемое. Подставляем \(f = \rho\) в (1.6):

\[ [\nabla\rho]_i = \sum_j \frac{m_j}{\rho_j}\,\rho_j\,\nabla_i W_{ij} = \sum_j m_j\,\nabla_i W_{ij}. \]

Заметим: \(\rho_j\) в числителе сократилось с \(\rho_j\) в знаменателе — формула градиента плотности получилась очень простой. Умножая на \(P_i/\rho_i^2\), получаем вклад второго слагаемого из (2.4a):

\[ \frac{P_i}{\rho_i^2}\,[\nabla\rho]_i = \sum_j m_j\,\frac{P_i}{\rho_i^2}\,\nabla_i W_{ij}. \]

Складываем оба вклада:

\[ \left[\frac{\nabla P}{\rho}\right]_i = \sum_j m_j\!\left(\frac{P_j}{\rho_j^2} + \frac{P_i}{\rho_i^2}\right)\!\nabla_i W_{ij}. \]

Подставляя в уравнение импульса \(d\mathbf v_i/dt = -\nabla P/\rho\), получаем симметричную форму:

\[ \boxed{\;\frac{d\mathbf v_i}{dt} = -\sum_j m_j\left(\frac{P_i}{\rho_i^2} + \frac{P_j}{\rho_j^2}\right)\nabla_i W_{ij}.\;} \tag{2.4} \]

Теперь в скобках — симметричная по индексам комбинация \(P_i/\rho_i^2 + P_j/\rho_j^2\). Это делает парные силы между частицами парными и противоположно направленными, как показано на рисунке выше.

Почему трюк с тождеством работает? Наивная форма (2.3) фактически дискретизирует производную \(\nabla(P/\rho)\) с выносом \(1/\rho_i\) наружу суммы — а это математически неправильно: \(1/\rho\) — функция от точки, её нельзя вынести из-под оператора \(\nabla\), работающего на этой же точке. Правильная дискретизация через тождество (2.4a) учитывает оба вклада и автоматически обеспечивает симметрию.

Заметим: теперь в скобках симметричная по индексам комбинация \(P_i/\rho_i^2 + P_j/\rho_j^2\), что делает парные силы между частицами парными и противоположно направленными.

Сохранение импульса: третий закон Ньютона

Теорема 5 (сохранение импульса в форме (2.4)). Полный импульс системы \(\mathbf P = \sum_i m_i\mathbf v_i\) сохраняется точно в схеме (2.4) для любого числа частиц и любого шага по времени.

Доказательство. Дифференцируем: \[ \frac{d\mathbf P}{dt} = \sum_i m_i\frac{d\mathbf v_i}{dt} = -\sum_i\sum_j m_i m_j\left(\frac{P_i}{\rho_i^2}+\frac{P_j}{\rho_j^2}\right)\nabla_i W_{ij}. \] Переобозначим во втором слагаемом \(i \leftrightarrow j\) и используем \(\nabla_i W_{ij} = -\nabla_j W_{ji}\) (чётность ядра): \[ \frac{d\mathbf P}{dt} = -\sum_i\sum_j m_i m_j\left(\frac{P_i}{\rho_i^2}+\frac{P_j}{\rho_j^2}\right)\nabla_i W_{ij} = -\frac{1}{2}\sum_i\sum_j m_i m_j\left(\ldots\right)(\nabla_i W_{ij} + \nabla_j W_{ji}) = 0. \] \(\square\)

Это — очень сильное свойство. Оно означает что SPH с формулировкой (2.4) точно сохраняет полный импульс, независимо от интегратора по времени (Эйлер, RK, Leap-Frog). Это ключевой аргумент в пользу симметричной формы.

Физическая интерпретация. Формула (2.4) эквивалентна тому, что между каждой парой частиц \((i,j)\) действует сила \(F_{ij} = -m_i m_j(P_i/\rho_i^2 + P_j/\rho_j^2)\nabla_i W_{ij}\) вдоль линии \(\mathbf r_i - \mathbf r_j\). Эта сила автоматически удовлетворяет третьему закону Ньютона: \(F_{ij} = -F_{ji}\). Полный импульс замкнутой системы сохраняется как следствие.

Интерактив: две частицы и сохранение импульса

Теорема 5 утверждает что симметричная форма (2.4) сохраняет полный импульс точно. Проверим это численно. Рассмотрим систему из двух частиц: левая (красная, \(\rho_L = 1, P_L = 1\)) и правая (синяя, \(\rho_R = 0.3, P_R = 0.15\)), на расстоянии \(\Delta x = 0.8h\). Высокое давление слева должно толкать правую частицу вправо.

Одновременно запустим обе формы: наивная (2.3) и симметричная (2.4). Левая и нижняя панели — фазовый портрет \((x, v)\) каждой частицы; правая — график суммарного импульса системы \(P(t) = m_L v_L + m_R v_R\). В исходной системе \(P(0) = 0\) (обе частицы покоятся).

Уравнение состояния. Политропная модель

Простейшее замыкание — политропная модель (изоэнтропическое течение):

\[ P_i = k\,\rho_i^\gamma, \qquad c_i = \sqrt{\gamma P_i/\rho_i}, \tag{2.5} \]

где \(c_i\) — скорость звука в частице \(i\), константа \(k\) определяется из начальных условий: \(k = P_L/\rho_L^\gamma\). Для идеального одноатомного газа \(\gamma = 5/3\); для двухатомного (воздух) \(\gamma = 1.4\).

Ограничения политропной модели. Формула (2.5) точна для изоэнтропических течений. На ударной волне энтропия растёт, и политропная модель даёт приближённый результат. Полный учёт энергии требует интегрирования третьего уравнения из (2.1): \[ \frac{de_i}{dt} = \frac{P_i}{\rho_i^2}\sum_j m_j(\mathbf v_i - \mathbf v_j)\cdot\nabla_i W_{ij}. \] В этой версии мы работаем с политропной моделью — она проще и достаточна для тестовых задач с умеренной интенсивностью скачков.

Полная система SPH для идеального газа

Собирая всё вместе, получаем систему уравнений, описывающую движение \(N\) частиц:

\[ \boxed{\;\begin{aligned} \rho_i &= \sum_j m_j\,W_{ij}, \\[4pt] P_i &= k\,\rho_i^\gamma, \quad c_i = \sqrt{\gamma P_i/\rho_i}, \\[4pt] \frac{d\mathbf v_i}{dt} &= -\sum_j m_j\!\left(\frac{P_i}{\rho_i^2} + \frac{P_j}{\rho_j^2}\right)\nabla_i W_{ij}, \\[4pt] \frac{d\mathbf r_i}{dt} &= \mathbf v_i. \end{aligned}\;} \tag{2.6} \]

Это — полная математическая модель. Массы \(m_j\) заданы в начальный момент и не меняются (это ключевое свойство лагранжева подхода).

Порядок вычислений на шаге:

  1. По текущим координатам \(\mathbf r_i\) вычислить \(\rho_i\) по формуле суммирования (2.2) — двойной цикл по частицам.
  2. По \(\rho_i\) вычислить давление \(P_i\) и скорость звука \(c_i\) по (2.5).
  3. Вычислить ускорения \(d\mathbf v_i/dt\) по (2.4) — опять двойной цикл.
  4. Продвинуть \(\mathbf v_i\), \(\mathbf r_i\) на шаг \(\Delta t\) каким-либо численным методом (см. лекцию 3).
Вычислительная стоимость. На первом шаге видно что каждый расчёт \(\rho_i\) требует \(O(N)\) операций (проход по всем \(j\)). Для \(N\) частиц это \(O(N^2)\) на шаг, что становится критично при \(N > 10^4\). Поэтому в реальных SPH-кодах используется метод linked-list cells: пространство делится на ячейки размера \(\sim 2h\), для каждой частицы поиск соседей идёт только по ячейкам-соседкам — суммарная стоимость \(O(N)\) на шаг. Мы вернёмся к этому в лекции 5.
