Практикум и задания

Как устроен практикум

До этого момента курс был теоретическим: мы выводили формулы, доказывали теоремы и разбирали алгоритмы. Этот раздел даёт практическую часть — набор задач, по которым можно проверить и закрепить понимание материала всех восьми лекций. Структура практикума повторяет структуру курса: задачи сгруппированы по темам и пронумерованы как Задача T.k, где \(T\) — номер темы (лекции), \(k\) — порядковый номер.

Задачи двух типов:

  • Аналитические — выполняются на бумаге: вывод формул, проверка свойств, ручной счёт небольших матриц и интегралов. К большинству из них дан сворачиваемый ответ — попробуйте решить самостоятельно, прежде чем раскрывать.
  • Вычислительные — требуют написать код. Язык на ваш выбор: Python (NumPy/SciPy/Matplotlib — рекомендуется), Maple (для символьных выкладок и проверки интегралов) или любой другой. Минимально достаточно бумаги, карандаша и калькулятора; для проверки сходимости и двумерных задач понадобится компьютер.
Как работать. Сначала решайте аналитические задачи темы — они формируют интуицию. Затем переходите к вычислительным: каждый собранный вами кусок (локальная матрица, сборка, решатель) пригодится в итоговом проекте. Сверяйтесь с чек-листом — он перечисляет умения, которые должны появиться после каждой темы.

Тема 1. Взвешенные невязки, коллокации, Ритц

Материал лекции 1: метод коллокаций, взвешенные невязки, наименьшие квадраты, Галёркин, вариационный метод Ритца, функции формы.

Задача 1.1. Рассмотрим краевую задачу \(-u'' = 1\) на \((0,1)\), \(u(0)=u(1)=0\). Возьмём однопараметрическое пробное решение \(\tilde u(x) = c\,x(1-x)\), точно удовлетворяющее краевым условиям. Найдите коэффициент \(c\) методом коллокации в точке \(x_0 = 1/2\). Сравните с точным решением \(u(x) = \tfrac12 x(1-x)\).
Показать ответ Невязка \(R(x) = -\tilde u'' - 1 = 2c - 1\) (постоянна, так как \(\tilde u'' = -2c\)). Условие коллокации \(R(x_0)=0\) даёт \(2c-1=0\), то есть \(c = 1/2\). Здесь пробная функция совпадает с точным решением, поэтому \(\tilde u(x) = \tfrac12 x(1-x) = u(x)\) — приближение точное. Это «удачный» случай: базис содержит точное решение.
Задача 1.2. Для той же задачи \(-u''=1\), \(u(0)=u(1)=0\) примените метод наименьших квадратов с тем же базисом \(\tilde u = c\,x(1-x)\): минимизируйте \(J(c)=\int_0^1 R(x)^2\,dx\). Покажите, что результат совпадает с задачей 1.1, и объясните, почему.
Показать ответ \(R = 2c-1\), \(J(c) = \int_0^1 (2c-1)^2 dx = (2c-1)^2\). Минимум: \(dJ/dc = 4(2c-1) = 0 \Rightarrow c=1/2\). Совпадает, потому что невязка не зависит от \(x\): любой разумный критерий обнуления невязки даёт одно и то же значение \(c\).
Задача 1.3. Метод Ритца. Для задачи \(-u''=f\), \(u(0)=u(1)=0\) функционал энергии имеет вид \[ J(v) = \tfrac12\int_0^1 (v')^2\,dx - \int_0^1 f\,v\,dx. \] Возьмите двухпараметрический базис \(v = c_1\sin(\pi x) + c_2\sin(2\pi x)\) и \(f(x)=\sin(\pi x)\). Найдите \(c_1, c_2\), минимизируя \(J\).
Показать ответ Базисные функции ортогональны: \(\int_0^1 \sin(m\pi x)\sin(n\pi x)\,dx = \tfrac12\delta_{mn}\), а \(\int_0^1 (\sin(m\pi x))'(\sin(n\pi x))'dx = \tfrac12 m^2\pi^2\,\delta_{mn}\). Тогда \(J = \tfrac12\big(\tfrac12\pi^2 c_1^2 + \tfrac12\cdot4\pi^2 c_2^2\big) - c_1\cdot\tfrac12\) (правая часть \(\sin\pi x\) проектируется только на первую моду). Из \(\partial J/\partial c_1 = \tfrac12\pi^2 c_1 - \tfrac12 = 0\): \(c_1 = 1/\pi^2\). Из \(\partial J/\partial c_2 = 2\pi^2 c_2 = 0\): \(c_2=0\). Получаем точное решение \(u = \sin(\pi x)/\pi^2\), что проверяется подстановкой в \(-u''=\sin\pi x\).
Задача 1.4. (вычислительная) Реализуйте метод коллокаций для \(-u'' + u = x\) на \((0,1)\), \(u(0)=u(1)=0\), с полиномиальным базисом \(\tilde u(x) = x(1-x)(c_1 + c_2 x)\). Выберите две точки коллокации \(x_1=1/3,\ x_2=2/3\), составьте и решите систему \(2\times2\) относительно \(c_1,c_2\). Постройте график \(\tilde u\) и оцените максимум невязки на сетке из 100 точек.
Задача 1.5. Постройте две кусочно-линейные «шапочки» (hat-функции) \(\varphi_1,\varphi_2\) на равномерной сетке узлов \(0, \tfrac13, \tfrac23, 1\) для внутренних узлов \(x_1=\tfrac13,\ x_2=\tfrac23\). Выпишите их явно по кускам и вычислите \(\int_0^1 \varphi_1'\varphi_2'\,dx\).
Показать ответ Шаг \(h=1/3\). На общем элементе \([\tfrac13,\tfrac23]\) производные \(\varphi_1' = -1/h = -3\), \(\varphi_2' = +1/h = +3\). Интеграл по этому единственному перекрывающемуся элементу: \((-3)(3)\cdot h = -9\cdot\tfrac13 = -3\). Это внедиагональный элемент глобальной матрицы жёсткости \(K_{12} = -1/h = -3\).

Тема 2. Лакс–Мильграм, лемма Сеа, слабые формулировки

Материал лекции 2: сильная и слабая постановки, пространства Соболева, теорема Лакса–Мильграма, лемма Сеа, главные и естественные граничные условия.

Задача 2.1. Выведите слабую (вариационную) формулировку задачи \(-(p\,u')' + q\,u = f\) на \((0,1)\) с \(u(0)=0\), \(u'(1)=g\) (\(p,q>0\)). Укажите билинейную форму \(a(u,v)\), линейную форму \(\ell(v)\) и пространство пробных функций. Куда вошло условие \(u'(1)=g\)?
Показать ответ Умножаем на \(v\) с \(v(0)=0\) и интегрируем по частям: \[ a(u,v)=\int_0^1\!\big(p\,u'v' + q\,u v\big)dx,\qquad \ell(v)=\int_0^1 f v\,dx + p(1)\,g\,v(1). \] Пространство \(V=\{v\in H^1(0,1): v(0)=0\}\). Условие \(u(0)=0\) — главное (встроено в \(V\)); условие \(u'(1)=g\) — естественное, оно вошло в \(\ell(v)\) через граничный член \([p\,u'v]_0^1\).
Задача 2.2. Проверьте, что форма \(a(u,v)=\int_0^1 (p\,u'v' + q\,uv)\,dx\) при \(p\ge p_0>0\), \(q\ge0\) коэрцитивна на \(V=\{v\in H^1: v(0)=0\}\), то есть \(a(v,v)\ge \alpha\|v\|_{H^1}^2\). Используйте неравенство Пуанкаре \(\|v\|_{L^2}\le C_P\|v'\|_{L^2}\).
Показать ответ \(a(v,v)=\int (p (v')^2 + q v^2)\ge p_0\|v'\|_{L^2}^2\). По Пуанкаре \(\|v'\|_{L^2}^2 = \tfrac12\|v'\|^2 + \tfrac12\|v'\|^2 \ge \tfrac12\|v'\|^2 + \tfrac{1}{2C_P^2}\|v\|_{L^2}^2\), поэтому \(a(v,v)\ge p_0\min\!\big(\tfrac12,\tfrac{1}{2C_P^2}\big)\big(\|v'\|^2+\|v\|_{L^2}^2\big)=\alpha\|v\|_{H^1}^2\) с \(\alpha = p_0\min(\tfrac12,\tfrac1{2C_P^2})>0\). Коэрцитивность доказана.
Задача 2.3. Проверьте непрерывность (ограниченность) той же формы: \(|a(u,v)|\le M\|u\|_{H^1}\|v\|_{H^1}\). Выразите \(M\) через \(\max p\) и \(\max q\). После этого сформулируйте, какие три гипотезы теоремы Лакса–Мильграма выполнены и что отсюда следует.
Показать ответ По Коши–Буняковскому \(|a(u,v)|\le \max(p)\|u'\|\,\|v'\| + \max(q)\|u\|\,\|v\| \le M\|u\|_{H^1}\|v\|_{H^1}\) с \(M=\max(\max p,\max q)\). Гипотезы Лакса–Мильграма: (1) \(a\) билинейна и непрерывна (этот пункт), (2) \(a\) коэрцитивна (задача 2.2), (3) \(\ell\) — непрерывный линейный функционал. Следствие: существует единственное слабое решение \(u\in V\), непрерывно зависящее от данных.
Задача 2.4. Лемма Сеа утверждает, что галёркинское решение \(u_h\) квазиоптимально: \(\|u-u_h\|\le \tfrac{M}{\alpha}\inf_{v_h\in V_h}\|u-v_h\|\). Пусть для конкретной задачи \(M=4\), \(\alpha=0.5\), а наилучшая аппроксимация в \(V_h\) даёт ошибку интерполяции \(0.01\). Дайте гарантированную верхнюю оценку ошибки МКЭ. Что произойдёт с этой оценкой при \(\alpha\to0\)?
Показать ответ \(\|u-u_h\|\le (4/0.5)\cdot0.01 = 8\cdot0.01 = 0.08\). При \(\alpha\to0\) (форма теряет коэрцитивность) множитель \(M/\alpha\to\infty\) — оценка вырождается, метод может терять устойчивость. Большое отношение \(M/\alpha\) (число обусловленности задачи) — признак плохо обусловленной системы.
Задача 2.5. (вычислительная) Для задачи 2.1 с \(p=1,\ q=1,\ f=1,\ g=0\) реализуйте галёркинский МКЭ с кусочно-линейными элементами на \(N\) равных отрезках. Постройте численное решение при \(N=8\) и сравните с точным \(u(x)=1-\tfrac{\cosh(1-x)}{\cosh 1}\). Эту реализацию переиспользуете в теме 3.

Тема 3. Матричная техника, ансамблирование, ферма

Материал лекции 3: локальная матрица жёсткости и вектор нагрузки, сборка глобальной системы, учёт граничных условий, расчёт фермы.

Задача 3.1. Выведите локальную матрицу жёсткости линейного 1D-элемента длины \(\ell_e\) для оператора \(-u''\). Используйте функции формы \(N_1=1-\xi,\ N_2=\xi\) (\(\xi=(x-x_1)/\ell_e\in[0,1]\)) и формулу \(K^e_{ij}=\int_0^{\ell_e} N_i' N_j'\,dx\).
Показать ответ \(N_1' = -1/\ell_e,\ N_2' = +1/\ell_e\). Каждый интеграл — постоянная подынтегральная функция, умноженная на длину \(\ell_e\): \[ K^e = \frac{1}{\ell_e}\begin{pmatrix} 1 & -1 \\ -1 & 1 \end{pmatrix}. \] Матрица симметрична, вырождена (сумма строк = 0) — отражает неопределённость до наложения граничных условий.
Задача 3.2. Для реакционного слагаемого \(+q\,u\) (\(q=\)const) выведите локальную матрицу масс \(M^e_{ij}=q\int_0^{\ell_e}N_i N_j\,dx\) того же линейного элемента.
Показать ответ \(\int_0^{\ell_e}N_1^2\,dx = \ell_e\!\int_0^1(1-\xi)^2 d\xi = \ell_e/3\); \(\int N_1 N_2 = \ell_e/6\); \(\int N_2^2 = \ell_e/3\). Поэтому \[ M^e = \frac{q\,\ell_e}{6}\begin{pmatrix} 2 & 1 \\ 1 & 2 \end{pmatrix}. \] Это «согласованная» (consistent) матрица масс; «сосредоточенная» (lumped) получается строчным суммированием: \(\mathrm{diag}(q\ell_e/2,\ q\ell_e/2)\).
Задача 3.3. Цепочка из трёх одинаковых элементов длины \(h=1/3\) соединяет 4 узла. Соберите глобальную матрицу жёсткости \(K\) (\(4\times4\)) из локальных \(K^e=\tfrac1h\begin{pmatrix}1&-1\\-1&1\end{pmatrix}\). Затем наложите \(u_0=u_3=0\) (вычеркнув строки/столбцы 0 и 3) и выпишите итоговую систему \(2\times2\).
Показать ответ Глобально (с \(1/h=3\)): \[ K = 3\begin{pmatrix} 1 & -1 & 0 & 0 \\ -1 & 2 & -1 & 0 \\ 0 & -1 & 2 & -1 \\ 0 & 0 & -1 & 1 \end{pmatrix}. \] После вычёркивания строк/столбцов 0 и 3 (узлы с заданным \(u\)) остаётся внутренний блок \[ K_{\text{red}} = 3\begin{pmatrix} 2 & -1 \\ -1 & 2 \end{pmatrix}. \]
Задача 3.4. (вычислительная) Реализуйте универсальный 1D КЭ-решатель уравнения \(-(p\,u')' + q\,u = f\) на \((a,b)\) с условиями Дирихле, использующий: (1) сборку \(K\) и \(M\) из локальных матриц задач 3.1–3.2 в цикле по элементам, (2) согласованный вектор нагрузки \(F^e_i=\int N_i f\,dx\), (3) наложение ГУ, (4) решение СЛАУ. Проверьте на \(p=q=1,\ f=1\) (задача 2.5).
Задача 3.5. Ферма: стержень длиной \(L\), площадь \(A\), модуль \(E\), ориентирован под углом \(\theta\) к оси \(x\). Локальная матрица жёсткости в глобальных координатах есть \(K^e=\tfrac{EA}{L}\,\mathbf{T}^\top\mathbf{T}\), где для осевого элемента \(\mathbf T = (-c,-s,c,s)\), \(c=\cos\theta,\ s=\sin\theta\). Выпишите \(K^e\) (\(4\times4\)) явно и проверьте, что при \(\theta=0\) он сводится к одномерному стержню.
Показать ответ \[ K^e=\frac{EA}{L}\begin{pmatrix} c^2 & cs & -c^2 & -cs \\ cs & s^2 & -cs & -s^2 \\ -c^2 & -cs & c^2 & cs \\ -cs & -s^2 & cs & s^2 \end{pmatrix}. \] При \(\theta=0\): \(c=1,s=0\), остаются только узлы по \(x\): \(\tfrac{EA}{L}\begin{pmatrix}1&-1\\-1&1\end{pmatrix}\) для степеней свободы \(u_x\) — обычный стержень.

Тема 4. Сетки и структуры данных

Материал лекции 4: структурированные и неструктурированные сетки, отображения, смежность/инцидентность, отношения узел–ячейка.

Задача 4.1. Для структурированной сетки \(n_x\times n_y\) узлов с глобальной нумерацией \(k = j\,n_x + i\) (\(0\le iПоказать ответ Соседи: восток \(k+1\) (если \(i0\)), север \(k+n_x\) (если \(j0\)). Полный набор из 4 соседей — только у внутренних узлов: \((n_x-2)(n_y-2) = 3\cdot2 = 6\) узлов.
Задача 4.2. Дана неструктурированная треугольная сетка с матрицей инцидентности «ячейка–узел» (по строкам — треугольники, столбцы — глобальные номера узлов): \[ T = \begin{pmatrix} 0 & 1 & 3 \\ 1 & 2 & 3 \\ 3 & 2 & 4 \end{pmatrix}. \] Постройте список рёбер без дубликатов и найдите, какие рёбра являются внутренними (общими для двух треугольников).
Показать ответ Рёбра по треугольникам: T0: (0,1),(1,3),(0,3); T1: (1,2),(2,3),(1,3); T2: (2,3),(2,4),(3,4). Уникальные рёбра: (0,1),(0,3),(1,2),(1,3),(2,3),(2,4),(3,4). Общие для двух треугольников: (1,3) — у T0 и T1; (2,3) — у T1 и T2. Они и есть внутренние; остальные — граничные.
Задача 4.3. Оцените объём памяти. Структурированная сетка \(1000\times1000\) хранит только координаты двух краёв и шаг; неструктурированная той же плотности хранит массив координат узлов (2 числа \(\times\,10^6\)) и массив связности треугольников (\(\sim2\cdot10^6\) треугольников \(\times\,3\) индекса). Прикиньте число хранимых чисел в обоих случаях (в double = 8 байт). Во сколько раз неструктурированная сетка «дороже» по геометрии?
Показать ответ Структурированная: топология неявная — порядка десятка чисел на описание сетки (плюс, при необходимости, само поле \(10^6\) значений). Неструктурированная: координаты \(2\cdot10^6\) + связность \(6\cdot10^6\) индексов \(\approx 8\cdot10^6\) чисел только на геометрию/топологию. Разница на геометрию — на порядки; платой за гибкость произвольной геометрии служит явное хранение связности.
Задача 4.4. (вычислительная) Напишите функцию, которая по массиву связности треугольников строит отношение «узел → список инцидентных треугольников» (обратное к «ячейка–узел»). Примените к сетке из задачи 4.2 и выведите для каждого узла список его треугольников.

Тема 5. Лаплас, Пуассон, метод Ритца на треугольниках

Материал лекции 5: функционал энергии для Лапласа/Пуассона, линейные функции формы треугольника, матрица элемента, сборка, граничные условия.

Задача 5.1. Линейный треугольный элемент с вершинами \(P_1=(0,0),\ P_2=(1,0),\ P_3=(0,1)\). Функции формы имеют вид \(N_i = a_i + b_i x + c_i y\). Найдите \(N_i\), коэффициенты \(b_i,c_i\) и площадь \(A\).
Показать ответ \(N_1=1-x-y,\ N_2=x,\ N_3=y\). Отсюда \((b_1,c_1)=(-1,-1)\), \((b_2,c_2)=(1,0)\), \((b_3,c_3)=(0,1)\). Площадь \(A=1/2\). Общая формула: \(b_i=(y_j-y_k)/2A\), \(c_i=(x_k-x_j)/2A\) при циклической нумерации \(i,j,k\).
Задача 5.2. Для того же элемента вычислите локальную матрицу жёсткости Лапласа \(K^e_{ij}=A\,(b_i b_j + c_i c_j)\).
Показать ответ С \(A=1/2\) и коэффициентами из 5.1: \[ K^e = \tfrac12\begin{pmatrix} 2 & -1 & -1 \\ -1 & 1 & 0 \\ -1 & 0 & 1 \end{pmatrix}. \] Сумма каждой строки равна нулю (постоянное поле даёт нулевую энергию) — корректный признак.
Задача 5.3. Учёт источника в уравнении Пуассона \(-\Delta u = f\). Для постоянного \(f\) на треугольнике вектор нагрузки есть \(F^e_i = f\int_{T}N_i\,dA = f\,A/3\). Докажите равенство \(\int_T N_i\,dA = A/3\) через интегральную формулу для барицентрических координат \(\int_T L_1^a L_2^b L_3^c\,dA = \dfrac{a!\,b!\,c!}{(a+b+c+2)!}\,2A\).

Итоговый проект

Цель проекта — собрать из наработанных кусков полноценный двумерный КЭ-решатель уравнения Пуассона на треугольной сетке (линейные элементы P1), как в лекции 5, и провести его верификацию.

Постановка. Решить краевую задачу \[ -\Delta u = f \ \text{в}\ \Omega=[0,1]^2,\qquad u = g \ \text{на}\ \partial\Omega, \] методом конечных элементов на неструктурированной (или структурированной, разбитой на треугольники) сетке с линейными функциями формы. Программа должна:
  1. читать/генерировать сетку: массив координат узлов и массив связности треугольников;
  2. в цикле по элементам вычислять локальную матрицу жёсткости \(K^e_{ij}=A(b_ib_j+c_ic_j)\) и локальный вектор нагрузки \(F^e_i=fA/3\) (для постоянного \(f\)) или через квадратуру;
  3. ансамблировать глобальную разреженную систему \(K\mathbf u = \mathbf F\);
  4. накладывать условия Дирихле \(u=g\) на граничных узлах;
  5. решать СЛАУ и визуализировать поле \(u(x,y)\).

Рекомендуемый тестовый случай (метод производимого решения). Возьмите известное \(u^*(x,y)=\sin(\pi x)\sin(\pi y)\). Тогда \(f=-\Delta u^* = 2\pi^2\sin(\pi x)\sin(\pi y)\), а на границе квадрата \(u^*=0\), то есть \(g=0\). Это даёт точное решение для сравнения.

Критерии приёмки.
  1. Верификация на точном решении. На умеренной сетке (\(\sim20\times20\) узлов, разбитой на треугольники) максимальная узловая ошибка \(\max_i|u_i-u^*(x_i,y_i)|\) мала и убывает при измельчении сетки.
  2. Проверка сходимости. На последовательности вложенных сеток (\(h,\ h/2,\ h/4\)) построить log–log график ошибки от \(h\) в \(L^2\)- или max-норме и показать наклон \(\approx 2\) (порядок \(O(h^2)\) для P1). Привести таблицу: \(h\), ошибка, наблюдаемый порядок \(\log_2(E_h/E_{h/2})\).
  3. Визуализация. Построить изолинии (или поверхность) численного решения и карту поузловой ошибки; убедиться визуально, что решение гладкое и симметричное.
  4. Корректность сборки. Проверить, что глобальная \(K\) симметрична, а сумма строк свободной (без ГУ) матрицы равна нулю (тест на постоянное поле).
Расширения (по желанию). (а) Неоднородные ГУ Дирихле \(g\ne0\); (б) смешанные ГУ (Неймана на части границы) — добавить граничный интеграл в \(\mathbf F\); (в) сравнить решатели: прямой (LU разреженной матрицы) против итерационного (сопряжённые градиенты); (г) перейти к элементам P2 и подтвердить порядок \(O(h^3)\); (д) сложная область (L-образная) и наблюдение падения порядка у входящего угла (сингулярность).

Минимальный отчёт по проекту: исходный код, таблица сходимости, два рисунка (решение и ошибка) и краткий вывод о соответствии наблюдаемого порядка теоретическому \(O(h^2)\).
